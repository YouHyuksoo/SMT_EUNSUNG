# 라우팅관리

## 화면 목적

품목별 **라우팅 그룹**, 그룹 안의 **공정 순서**, 각 공정에 배정할 **현재 BOM 투입자재**를 관리합니다.

## 화면 구성

- 왼쪽: 라우팅 그룹 검색·등록·수정·삭제
- 가운데: 선택 그룹의 공정 순서와 내작/외주 구분
- 오른쪽: 선택 공정의 BOM 자재 후보와 현재 배정 상태

## 사용 순서

1. 라우팅 그룹을 선택하거나 새로 등록합니다. 라우팅 코드, 품목, 라우팅명은 필수입니다.
2. **공정** 버튼으로 공정 순번과 공정 코드를 등록합니다.
3. 공정이 외주이면 실행유형을 `SUBCON`으로 선택하고 공급처를 지정합니다. 내작은 `INTERNAL`입니다.
4. 공정 행을 선택한 뒤 오른쪽에서 BOM 자재를 체크하고 투입수량·투입방식을 설정합니다.
5. **저장**을 눌러 자재 배정을 반영합니다.

## 주요 규칙

- 한 품목에는 활성 라우팅 그룹을 하나만 둘 수 있습니다.
- 한 하위 자재는 같은 라우팅 그룹 안에서 공정 하나에만 배정할 수 있습니다. 다른 공정에 이미 배정된 자재는 선택할 수 없습니다.
- 자재 후보는 라우팅 그룹 품목의 **오늘 기준 유효 BOM**에서 가져옵니다.
- BOM이 바뀌어 현재 BOM에서 사라진 기존 배정도 경고 행으로 표시됩니다. 이 배정은 자동 삭제되지 않으며 휴지통으로 **명시적 삭제**한 뒤 저장해야 합니다.
- 투입수량은 0보다 커야 합니다.
- `BACKFLUSH`는 생산실적 처리 시 자동 차감, `PRE_ISSUE`는 작업 전에 먼저 투입하는 방식입니다.
- 투입자재가 남아 있는 공정은 삭제할 수 없습니다. 자재 배정을 먼저 삭제하십시오.

## 현재 제공하지 않는 기능

이 화면에는 BOM 트리, 양품조건(QC), 자주검사, SG/FG 라벨 설정, 회로사양 입력이 없습니다.
