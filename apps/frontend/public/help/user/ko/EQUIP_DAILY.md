# 일상점검

## 화면 목적
매일 작업 전후에 수행하는 설비 일상점검(DAILY)의 결과를 등록하고 관리합니다. 좌측 설비 목록에서 대상 설비를 선택하고, 우측 패널에서 각 점검항목의 합불(PASS/FAIL)을 판정하여 저장합니다.

## 화면 구성
- **좌측 — 설비 목록(EquipListPanel)**: 금일 점검 대상 설비 목록을 설비유형별로 그룹화하여 표시합니다. 검색 필터와 상태별(미점검/완료OK/완료NG) 필터를 제공합니다.
- **우측 — 점검 입력(InspectEntryPanel)**: 선택한 설비의 점검항목 목록과 판정 입력 폼을 표시합니다.

---

## ① 좌측 설비 목록 컬럼

| 항목 | 역할 / 의미 |
|------|------|
| **설비코드** | 개별 설비의 고유 식별 코드입니다. |
| **설비명** | 설비의 명칭입니다. |
| **점검자** | 해당 설비의 점검을 수행한 작업자 이름입니다. |
| **결과** | 점검 종합 결과입니다(미점검/합격/불합격). |
| **항목 수** | 해당 설비에 등록된 점검항목 개수입니다. |

## ② 점검 입력 패널 구성

| 항목 | 역할 / 의미 |
|------|------|
| **점검일** | 점검 수행일자입니다(기본값: 오늘). |
| **점검자 (필수)** | 점검을 수행한 작업자를 선택합니다. 작업자 마스터에서 조회됩니다. |
| **시작 시각** | 점검 시작 시각입니다. |
| **점검항목 목록** | 선택한 설비에 연결된 점검항목을 표시합니다. |
| **판정 입력** | 각 항목별로 OK(합격) 또는 NG(불합격)를 선택하거나, 측정형은 수치를 입력합니다. |
| **종합 판정** | 모든 항목의 판정 결과를 종합하여 자동 표시됩니다. |

## 점검항목 목록 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **사진** | 점검 참조 이미지(썸네일). |
| **점검항목** | 점검항목 마스터의 항목명입니다. |
| **유형** | 판정형(VISUAL) 또는 측정형(MEASURE) 구분입니다. |
| **기준** | 합불 판단 기준 또는 측정 기준 범위입니다. |
| **측정값 / 입력** | 측정형은 측정값을 입력하고, 판정형은 OK/NG를 선택합니다. |
| **판정** | OK(합격) 또는 NG(불합격) 결과입니다. |

## 사용 순서
1. 좌측 설비 목록에서 점검할 설비를 선택합니다.
2. 우측 패널에서 점검일과 점검자를 확인/선택합니다.
3. 각 점검항목별로 결과를 입력합니다:
   - **판정형(VISUAL)**: 드롭다운 또는 버튼으로 OK/NG 선택
   - **측정형(MEASURE)**: 실제 측정값을 숫자로 입력
4. NG(불합격) 항목이 있으면 불량 내용을 입력합니다.
5. **저장 (PASS)** 또는 **저장 (NG)** 버튼으로 결과를 저장합니다.

## 입력 규칙
- 점검자는 **필수 선택**입니다.
- 모든 점검항목에 대해 판정 결과를 입력해야 저장할 수 있습니다.
- 측정형은 LSL(하한)과 USL(상한) 범위를 벗어나면 자동으로 NG 판정됩니다.
- NG(불합격)인 경우 원인/비고 입력이 필요합니다.
