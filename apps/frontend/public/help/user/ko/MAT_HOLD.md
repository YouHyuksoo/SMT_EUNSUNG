# 자재재고홀드관리

## 화면 목적
자재 LOT(시리얼) 단위로 **재고 홀드(사용 중지)** 또는 **홀드 해제**를 관리합니다. 품질 문제나 기타 사유로 특정 LOT의 사용을 일시 중지해야 할 때 사용합니다. 홀드된 LOT는 출고·생산에 사용할 수 없습니다.

## 화면 구성
- **상단 — 통계 카드 3개**: 전체 LOT / 홀드 중 / 정상
- **필터 영역**: 검색어 + LOT 상태(전체/정상/홀드) 선택
- **DataGrid 목록**: LOT별 상태와 홀드/해제 액션 버튼 표시
- **모달**: 홀드 또는 해제 시 사유 입력

---

## ① DataGrid 목록 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **LOT번호(matUid)** | 홀드 대상 자재 LOT/시리얼 번호입니다. |
| **품목코드(itemCode)** | LOT에 해당하는 품목 코드입니다. |
| **품목명(itemName)** | LOT에 해당하는 품목 이름입니다. |
| **현재수량(qty)** | 해당 LOT의 현재 보유 수량입니다. |
| **공급처(vendor)** | 해당 LOT를 공급한 협력사(공급처)입니다. |
| **상태(status)** | LOT의 현재 상태입니다. **HOLD**(빨강) = 사용 중지, **NORMAL**(초록) = 정상. |

---

## 사용 순서
1. **홀드**: 목록에서 홀드할 LOT의 🔒(잠금) 아이콘을 클릭합니다. 모달에서 사유를 입력하고 확인합니다. 해당 LOT는 `HOLD` 상태가 되며 출고/사용이 차단됩니다.
2. **홀드 해제**: 홀드된 LOT의 🔓(잠금해제) 아이콘을 클릭합니다. 모달에서 사유를 입력하고 확인합니다. LOT가 `NORMAL` 상태로 복원됩니다.
3. 상태 필터로 **홀드 중**인 LOT만 모아서 확인할 수 있습니다.
