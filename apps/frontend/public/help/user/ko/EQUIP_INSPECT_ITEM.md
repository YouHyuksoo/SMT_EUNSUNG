# 설비별 점검항목

## 화면 목적
개별 설비에 [점검항목 마스터]에 등록된 점검항목을 연결(매핑)하여 설비별로 수행할 점검 항목을 구성합니다. 점검유형별(일상/정기/PM/작업자)로 탭이 분리되어 있으며, QR 라벨을 발행하여 현장에서 활용할 수 있습니다.

## 화면 구성
- **좌측 — 설비 목록**: 전체 설비를 설비유형별로 그룹화하여 표시합니다. 검색 필터로 원하는 설비를 찾을 수 있습니다.
- **우측 — 점검항목 패널**: 선택한 설비의 점검항목 목록을 점검유형 탭(DAILY/PERIODIC/PM/WORKER)별로 표시합니다.
- **드로어 — 점검항목 추가**: 우측 상단 버튼으로 열리는 패널에서 점검항목 마스터의 항목을 선택하여 일괄 등록합니다.
- **모달 — QR 라벨 발행**: 점검항목별 QR 코드 라벨을 출력합니다.

---

## ① 점검항목 목록 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **사진(imageUrl)** | 점검항목의 참조 이미지입니다. |
| **항목코드(itemCode)** | 점검항목 마스터의 항목 코드입니다. |
| **항목명(itemName)** | 점검항목의 이름입니다. |
| **판정기준(criteria)** | 점검의 OK/NG 판단 기준 또는 측정 범위입니다. |
| **주기(cycle)** | 점검 수행 주기입니다(DAILY/WEEKLY/MONTHLY 등). |
| **순서(sortSeq)** | 설비 내 점검항목의 표시 순서입니다. |
| **사용(useYn)** | `Y`(초록) = 활성, `N`(빨강) = 비활성. |

## 사용 순서
1. 좌측 설비 목록에서 원하는 설비를 선택합니다.
2. 우측 상단 탭에서 점검유형(DAILY/PERIODIC/PM/WORKER)을 선택합니다.
3. **점검항목 추가** 버튼을 클릭하여 드로어 패널을 엽니다.
4. 추가할 항목을 체크하고 **일괄등록** 버튼으로 설비에 연결합니다.
5. 등록된 항목은 목록에서 확인할 수 있으며, 필요 시 삭제할 수 있습니다.
6. **QR 라벨 발행** 버튼으로 개별 항목의 QR 코드 라벨을 출력하여 현장에 부착합니다.

## 점검유형 탭
| 탭 | 내용 |
|------|------|
| **DAILY** | 매일 작업 전 수행하는 일상 점검 항목 |
| **PERIODIC** | 월/분기/반기/연 단위 정기 점검 항목 |
| **PM** | 예방보전(Preventive Maintenance) 관련 점검 항목 |
| **WORKER** | 작업자가 자체 수행하는 점검 항목 |
