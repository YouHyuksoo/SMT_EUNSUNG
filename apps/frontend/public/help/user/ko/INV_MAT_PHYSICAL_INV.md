# 자재재고실사 관리

## 화면 목적
실제 창고 재고와 시스템 재고를 비교하는 **재고실사(Physical Inventory)** 세션을 관리합니다. 실사 세션을 개시하여 PDA에서 재고를 스캔할 수 있도록 하고, 세션 완료 후 결과를 확인합니다.

## 화면 구성
- **상단 — 기준년월/창고/검색 필터**: 조회 기준을 선택합니다.
- **세션 상태 표시**: 현재 진행 중인 실사 세션이 있으면 상단에 표시됩니다.
- **DataGrid 목록**: 품목별 시스템 수량 / PDA 실사 수량 / 차이를 표시합니다.
- **실사 개시 / 종료 버튼**: 세션을 시작하거나 완료합니다.

---

## ① DataGrid 목록 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **창고(warehouseName)** | 재고가 위치한 창고입니다. |
| **품목코드(itemCode)** | 실사 대상 품목 코드입니다. |
| **품목명(itemName)** | 실사 대상 품목 이름입니다. |
| **자재시리얼(matUid)** | LOT/시리얼 번호입니다. |
| **시스템수량(qty)** | 실사 개시 시점의 시스템 재고 수량입니다. |
| **실사수량(countedQty)** | PDA에서 스캔하여 입력한 실사 수량입니다. 아직 스캔되지 않으면 `-`로 표시됩니다. |
| **차이(diffQty)** | 실사수량 - 시스템수량. **양수(파랑)** = 실제가 더 많음, **음수(빨강)** = 실제가 더 적음, **0(초록)** = 일치. |
| **실사시간(countedAt)** | 마지막으로 PDA 스캔이 이루어진 시각입니다. |
| **최종실사일(lastCountDate)** | 해당 품목이 마지막으로 실사된 일자입니다. |

---

## ② 실사 세션 관리

| 작업 | 설명 |
|------|------|
| **실사 개시** | 기준년월과 창고를 선택하여 실사 세션을 시작합니다. 개시 후 **모든 재고 변동(입고·출고·보정)이 차단**되며, PDA에서 실사 스캔이 가능해집니다. |
| **실사 종료** | PDA 스캔이 완료되면 세션을 종료합니다. 종료 후 재고 변동 차단이 해제됩니다. |
| **실사 반영** | 종료 후 [자재재고실사 반영] 화면에서 실사 수량을 검토하고 재고에 반영합니다. |

> ⚠️ 실사 개시 후에는 자재 입고, 출고, 보정, LOT 분할/병합 등 모든 재고 변동 트랜잭션이 차단됩니다.

## 사용 순서
1. **실사 개시** 버튼으로 기준년월과 창고를 선택하여 세션을 시작합니다.
2. PDA에서 해당 창고의 품목을 스캔하여 실사 수량을 입력합니다.
3. 화면을 새로고침하여 PDA 스캔 현황을 실시간으로 모니터링합니다.
4. 모든 스캔이 완료되면 **실사 종료** 버튼으로 세션을 종료합니다.
5. [자재재고실사 반영] 화면에서 실사 결과를 검토하고 재고에 반영합니다.
