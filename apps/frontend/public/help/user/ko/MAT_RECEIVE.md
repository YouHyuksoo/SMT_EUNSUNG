# 자재입고관리

## 화면 목적
**수입검사(IQC)에 합격한 자재 LOT을 창고에 입고**(적치)하는 화면입니다. 전수 수기 입력 대신 **바코드 스캔 방식**으로, 거래처 바코드와 자체부착 바코드를 매핑해 정확히 입고 확정합니다.

> 흐름: 입하(발주 입고) → 수입검사(IQC) → **합격건만 입고대기** → 스캔 입고 → 재고 반영

## 화면 구성
- **상단 통계**: 입고대기 건수/수량, 금일 입고 건수/수량.
- **필터**: 입하일자(기본 당일), 공급업체, 품목 검색.
- **입고대기 그리드**: IQC 합격 + 미입고(잔여 있는) LOT 목록. 입고 대상 확인용.
- **입고처리(스캔) 모달**: 거래처 바코드 → 자체부착 바코드를 순환 스캔. **스캔된 매핑만 입고 확정**됩니다.

---

## ① 입고대기 컬럼 (입고 가능 LOT)

| 컬럼 | 역할 / 의미 |
|------|------|
| **자재 시리얼(matUid)** | 입고 대상 자재 LOT의 고유 시리얼입니다. 입하 시 채번된 자재 식별값. |
| **품목코드/품목명(part.itemCode / itemName)** | 입고 대상 품목입니다(품목마스터 기준). |
| **품목유형(itemType)** | 원자재/소모품 등 분류입니다. |
| **단위(part.unit)** | 수량 단위입니다. |
| **입하수량(initQty)** | 입하 시점의 LOT 초기 수량입니다. |
| **기입고수량(receivedQty)** | 이미 입고 처리된 수량입니다(부분 입고 누계). |
| **잔여수량(remainingQty)** | 아직 입고되지 않은 수량입니다(= 입고 가능 수량). |
| **입하일(recvDate)** | 자재가 입하된 날짜입니다. |
| **제조일(manufactureDate)** | 자재 제조일자입니다. |
| **유효기한(expireDate / expiryDays)** | 자재의 유효기한과 남은 일수입니다. 기한 임박/경과 관리에 사용. |
| **발주번호(poNo)** | 연결된 발주(PO) 번호입니다. |
| **공급처(vendor)** | 자재를 납품한 거래처입니다. |
| **검사상태(iqcStatus)** | 수입검사 결과입니다. 입고대기에는 **합격** 건만 표시됩니다. |
| **입하창고(arrivalWarehouse)** | 자재가 입하되어 있는 창고입니다. |
| **성적서(certRequired / certUploaded)** | 성적서 필요 여부와 업로드 여부입니다. 필요한데 미업로드면 입고가 차단될 수 있습니다. |
| **입고 차단 사유(receivingBlockedReason)** | 입고가 막힌 경우 그 이유입니다(예: 성적서 미제출). |

## ② 스캔 입고 방법
1. 입고대기에서 대상을 확인하고 **입고처리** 버튼을 누릅니다.
2. **거래처 바코드**를 스캔합니다(공급처가 부착한 바코드).
3. 이어서 **자체부착 바코드(자재 시리얼)**를 스캔합니다.
4. 두 바코드가 매핑된 건만 입고 확정됩니다. 여러 건을 순환 스캔할 수 있습니다.

> 거래처 바코드는 [제조사 바코드 매핑]에 등록된 규칙으로 MES 품번/시리얼과 연결됩니다.

## ③ 입고 이력 컬럼
| 컬럼 | 역할 / 의미 |
|------|------|
| **입고번호/거래번호(receiveNo / transNo)** | 입고 처리 식별 번호입니다. |
| **입고일(transDate)** | 입고 처리 일자입니다. |
| **수량(qty)** | 입고된 수량입니다. |
| **입고창고(toWarehouse)** | 자재가 적치된 창고입니다. |
| **공급처/제조사(vendor / manufacturer)** | 납품 거래처와 제조사입니다. |
| **구분(materialClass)** | 양산(PROD) / 소모품(MRO) 구분입니다. |
| **상태(status)** | 입고 상태입니다. |

## 자주 묻는 질문
- **Q.** 입고대기에 자재가 안 보여요.
  **A.** IQC 합격 + 잔여수량이 있어야 표시됩니다. 검사 미완/불합격이거나 이미 전량 입고된 경우 보이지 않습니다.
- **Q.** 입고가 차단됩니다.
  **A.** 성적서가 필요한 품목인데 미업로드인 경우 등 '입고 차단 사유'를 확인하세요.
- **Q.** 부분만 입고할 수 있나요?
  **A.** 네, 잔여수량 범위에서 부분 입고가 누적됩니다.
