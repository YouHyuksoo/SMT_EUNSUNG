# 제조사 바코드 매핑

## 화면 목적
제조사(거래처)가 자재에 부착한 **바코드를 자사 MES 품번으로 자동 변환**하기 위한 매핑 규칙을 등록·관리합니다. 여기 등록된 규칙은 자재 **입고 스캔** 시 적용되어, 스캔한 제조사 바코드를 자동으로 MES 품목으로 해석합니다.

## 화면 구성
- **왼쪽(목록)**: 매핑 그리드. 상단에서 **검색**(바코드·품번·품명·제조사명)과 **매칭유형 필터**, **새로고침**, **매핑 추가**.
- **오른쪽(슬라이드 패널)**: 행의 ✏️(수정) 또는 "매핑 추가" 클릭 시 열리는 등록/수정 폼. **바코드 정보**·**품목 매핑** 두 영역으로 나뉩니다.

### 매칭 방식 개념
```
스캔한 바코드 ─┬─ EXACT  : 바코드 값이 등록 값과 정확히 같으면 매칭
              ├─ PREFIX : 바코드가 등록 값으로 시작하면 매칭(접두사)
              └─ REGEX  : 바코드가 등록한 정규식 패턴에 맞으면 매칭
       └─▶ 매칭된 규칙의 품번(itemCode)으로 변환
```
> 입고 스캔 시 시스템은 **EXACT → PREFIX → REGEX 순서**로 매칭을 시도하며, 사용여부가 `Y`인 규칙만 사용합니다.

## ① 매핑 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **제조사 바코드(vendorBarcode)** | 제조사가 부착한 바코드를 스캔한 **원본 값**입니다. 이 화면의 식별키(고유 키)이며, 같은 값을 중복 등록할 수 없습니다. 매칭유형이 PREFIX면 접두 문자열, REGEX면 정규식 패턴을 이 칸에 넣습니다. |
| **품번(itemCode)** | 이 바코드가 가리키는 **MES 품목코드**입니다(품목마스터의 itemCode 참조). 변환의 결과가 되는 자사 품번입니다. |
| **품명(itemName)** | 조회 편의를 위해 함께 저장하는 품목 이름입니다. |
| **제조사코드(vendorCode)** | 이 바코드를 발행한 **제조사 코드**입니다(거래처 마스터 참조). 필터·관리용. |
| **제조사명(vendorName)** | 조회 편의를 위해 함께 저장하는 제조사 이름입니다. |
| **매칭유형(matchType)** | 스캔 값과 등록 값을 비교하는 방식입니다. **EXACT(정확 일치)** = 값이 똑같아야 매칭, **PREFIX(접두사)** = 스캔 값이 등록 값으로 시작하면 매칭, **REGEX(정규식)** = 등록한 정규식에 맞으면 매칭. 기본값은 EXACT입니다. |
| **매핑규칙(mappingRule)** | 매칭 규칙에 대한 설명 메모입니다(예: 접두사 패턴, 정규식 식별 규칙 등). 동작에는 영향을 주지 않는 참고용입니다. |
| **사용(useYn)** | `Y`일 때만 입고 스캔 매칭과 조회·선택 대상이 됩니다. `N`이면 규칙이 적용되지 않습니다(목록의 초록/회색 점으로 표시). |
| **비고(remark)** | 매핑 관리 참고사항입니다. |

## 사용 순서
1. 상단 **매핑 추가**(또는 행 ✏️)로 오른쪽 슬라이드 폼을 엽니다.
2. **바코드 정보**: 제조사 바코드 값을 입력하고, **매칭 유형**(정확 일치/접두사/정규식)과 사용여부를 선택합니다.
3. **품목 매핑**: 이 바코드가 가리킬 **품번/품명**과 **제조사 코드/제조사명**을 입력합니다.
4. 필요하면 **매핑 규칙** 설명과 **비고**를 적고 **저장**합니다.
5. 이후 입고 스캔에서 해당 바코드가 등록한 품번으로 자동 변환됩니다.

## 입력 규칙·검증
- **제조사 바코드는 필수**입니다. 비어 있으면 저장 버튼이 비활성화됩니다.
- 같은 제조사 바코드 값을 **중복 등록할 수 없습니다**(이미 등록된 값이면 저장 시 오류).
- 매칭유형은 **EXACT / PREFIX / REGEX** 중 하나만 허용됩니다.
- REGEX로 등록할 때는 **올바른 정규식 패턴**을 넣어야 합니다. 잘못된 패턴은 입고 매칭 시 무시됩니다.
- 사용여부는 `Y` 또는 `N`만 허용됩니다.

## 자주 묻는 질문
- **Q.** 한 제조사가 같은 품목에 매번 끝자리만 다른 바코드를 붙여요.
  **A.** 공통으로 시작하는 부분을 **PREFIX(접두사)**로 등록하면, 그 접두사로 시작하는 모든 바코드가 같은 품번으로 변환됩니다.
- **Q.** 바코드 안에 규칙(자리수·구분자)이 있어 패턴으로만 식별돼요.
  **A.** **REGEX(정규식)**로 패턴을 등록하세요. EXACT·PREFIX로 잡히지 않을 때 마지막으로 시도됩니다.
- **Q.** 등록했는데 입고 스캔에서 변환이 안 돼요.
  **A.** 사용여부가 `Y`인지, 매칭유형과 값이 실제 스캔 값과 맞는지 확인하세요. EXACT는 값이 완전히 같아야 합니다.
- **Q.** 품번을 잘못 연결했어요.
  **A.** 해당 행 ✏️로 수정 패널을 열어 품번/품명을 바로잡고 저장하면 됩니다.
