# 외관검사

## 화면 목적
생산 완료되어 FG 라벨이 발행된 제품의 외관을 육안으로 검사하고 합격(PASS) 또는 불합격(FAIL)을 판정하여 결과를 등록합니다.

## 화면 구성
- **좌측 — 작업지시 목록**: 외관검사 대상 작업지시(완료된 지시) 목록입니다. 작업지시번호, 품목코드로 검색할 수 있습니다.
- **우측 — 외관검사 패널**: 선택한 작업지시의 FG 바코드 스캔, 합불 판정, 검사 이력을 표시합니다.

---

## 사용 순서
1. 좌측 목록에서 검사할 작업지시를 선택합니다.
2. FG 바코드 입력란에 제품 바코드를 스캔합니다.
3. 스캔된 제품 정보를 확인합니다.
4. **합격(PASS)** 또는 **불합격(FAIL)** 버튼을 클릭하여 판정합니다.
   - 불합격 시 불량코드와 상세 사유를 입력해야 합니다.
5. 검사 결과는 우측 하단 이력 그리드에 즉시 반영됩니다.

## 입력 규칙
- **ISSUED** 상태의 FG 라벨만 검사 가능합니다(이미 검사 완료, 포장, 출하된 제품은 검사 불가).
- 스캔한 바코드는 반드시 선택한 작업지시의 제품이어야 합니다.
- 불합격(FAIL) 판정 시 불량코드(`VISUAL_DEFECT` 그룹코드)와 상세 사유를 입력해야 합니다.
- 검사 완료된 제품은 재검사가 불가능합니다.

## 검사 이력 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **검사시각** | 외관검사가 수행된 일시입니다. |
| **판정** | 합격(PASS/초록) 또는 불합격(FAIL/빨강) 결과입니다. |
| **FG 바코드** | 검사 대상 제품의 FG 바코드입니다. |
| **불량코드** | 불합격 시 선택한 불량 코드입니다. |
| **상세 사유** | 불합격 시 입력한 상세 불량 내용입니다. |
