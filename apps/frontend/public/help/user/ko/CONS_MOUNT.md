# 소모품 장착관리

## 화면 목적
생산 설비에 사용하는 **소모품(금형·지그·공구 등)**을 설비에 **장착**하거나 **해제**하고, 필요 시 **수리 상태로 전환**하거나 수리 완료 후 **창고로 복귀**시키는 화면입니다. 개별 소모품의 장착·해제 이력도 확인할 수 있습니다.

> 이 화면은 소모품의 **물리적 위치/상태**를 관리합니다. 입출고나 수명은 [소모품입고], [소모품출고], [수명현황] 화면에서 별도로 관리합니다.

## 화면 구성
- **상단 헤더**: 화면 제목과 새로고침 버튼
- **메인 그리드**: 소모품 목록. 각 행마다 상태에 맞는 액션 버튼이 표시됩니다.
- **액션 모달**: 장착/해제/수리전환/수리완료를 실행하는 팝업
- **이력 모달**: 선택한 소모품의 장착/해제 이력을 표로 조회

## ① 메인 그리드 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **관리** | 행 단위 액션 버튼 모음입니다. 상태에 따라 **장착**, **해제**, **수리전환**, **수리완료**, **이력** 버튼이 나타납니다. |
| **운영상태(operStatus)** | 소모품의 현재 위치/상태입니다. `창고`(WAREHOUSE) / `장착`(MOUNTED) / `수리`(REPAIR) 중 하나입니다. |
| **소모품코드(consumableCode)** | 소모품을 식별하는 고유 코드입니다. |
| **소모품명(consumableName)** | 소모품의 이름입니다. |
| **분류(category)** | 소모품 종류입니다. 금형(MOLD)·지그(JIG)·공구(TOOL) 등 공통코드 `CONSUMABLE_CATEGORY` 값입니다. |
| **장착설비(mountedEquip)** | 현재 소모품이 장착된 설비 코드입니다. 장착 상태가 아니면 `-`로 표시됩니다. |
| **수명상태(status)** | 소모품의 수명 상태입니다. `NORMAL`(정상) / `WARNING`(경고) / `REPLACE`(교체필요) 중 하나입니다. |
| **수명(lifeProgress)** | 현재 사용횟수 대비 기대수명의 비율을 막대 그래프로 표시합니다. 80% 이상 노란색, 100% 이상 빨간색으로 강조됩니다. |
| **보관위치(location)** | 소모품의 기본 보관 장소입니다. |

## ② 상태별 가능한 액션

| 현재 상태 | 가능한 액션 | 설명 |
|------|------|------|
| **창고** (WAREHOUSE) | 장착, 수리전환 | 설비를 선택해 장착하거나 수리 상태로 전환합니다. |
| **장착** (MOUNTED) | 해제, 수리전환 | 현재 장착된 설비에서 해제하거나, 자동 해제 후 수리 상태로 전환합니다. |
| **수리** (REPAIR) | 수리완료 | 수리가 끝난 소모품을 창고 상태로 복귀시킵니다. |

## ③ 액션 모달 입력 항목

| 항목 | 설명 |
|------|------|
| **대상 설비** | 장착 시 필수. 설비 선택 컴포넌트에서 장착할 설비를 선택합니다. |
| **비고** | 선택 사항. 장착/해제/수리 사유 등을 기록합니다. 최대 500자. |

## 사용 순서
1. 상단 검색창이나 **분류/운영상태 필터**로 대상 소모품을 찾습니다.
2. 행의 상태에 맞는 액션 버튼을 클릭합니다.
   - 장착: `장착` 버튼 → 대상 설비 선택 → 저장
   - 해제: `해제` 버튼 → 비고 입력(선택) → 저장
   - 수리: `수리전환` 버튼 → 비고 입력(선택) → 저장
   - 수리완료: `수리완료` 버튼 → 저장
3. 변경 후 목록이 자동으로 새로고침됩니다.
4. 특정 소모품의 과거 이력을 볼 때는 `이력` 버튼을 클릭합니다.

## 입력 규칙 / 검증
- **장착**은 대상 설비를 반드시 선택해야 합니다. 미선택 시 저장 버튼이 비활성화됩니다.
- 이미 장착된 소모품은 다시 장착할 수 없습니다.
- 장착 상태가 아닌 소모품은 해제할 수 없습니다.
- 수리 상태가 아닌 소모품은 수리완료 처리할 수 없습니다.
- 수리 전환 시 장착 중이면 자동으로 해제 기록이 먼저 남은 뒤 수리 상태로 전환됩니다.

## 자주 묻는 질문
- **Q.** 장착 버튼이 안 보여요.
  **A.** 운영상태가 `창고`(WAREHOUSE)인 소모품에만 장착 버튼이 표시됩니다. 필터를 확인하세요.
- **Q.** 수리로 본 후 다시 창고로 돌리려면?
  **A.** `수리완료` 버튼을 누륩니다. 그러면 운영상태가 `창고`(WAREHOUSE)로 복귀됩니다.
- **Q.** 설비에서 해제했는데 수명이 초과됐다고 떠요.
  **A.** 해제 자체는 가능합니다. 수명 상태는 [소모품 마스터]에서 기대수명·경고 임계치를 확인하고 교체 또는 리셋하세요.
- **Q.** 이력에 작업자가 안 보여요.
  **A.** 현재 화면에서 작업자 입력란은 없으며, API 호출 시 로그인 사용자 정보가 기록됩니다. 필요 시 백엔드 로직을 확인하세요.
