# 소모품 마스터

## 화면 목적
생산에 쓰이는 **소모품(금형·지그·공구 등)의 기준정보**를 등록·관리합니다. 여기 등록된 소모품이 입출고·재고·수명(타수) 관리, 키오스크 생산실적의 소모품 투입, 사용처(제품·설비) 매핑의 기준이 됩니다.

> 소모품은 쓸수록 닳거나 수명이 소진되는 자재입니다. 이 화면에서는 **무엇을(코드·이름·분류), 얼마나 오래 쓸 수 있는지(기대수명·경고임계치), 어디에 얼마나 두는지(안전재고·보관위치), 어디에 쓰는지(사용처 매핑)**를 한 곳에서 정의합니다.

## 화면 구성
- **왼쪽(목록)**: 소모품 그리드. 상단에서 **소모품명/코드 검색**, **분류 필터**, **새로고침**, **소모품 등록**.
- **가운데(슬라이드 패널)**: 행의 ✏️(수정) 또는 "소모품 등록" 클릭 시 열리는 등록/수정 폼.
- **오른쪽(사용매핑 패널)**: 목록에서 행을 선택하면 그 소모품의 **제품/설비 사용매핑**을 상시 등록·수정·삭제할 수 있습니다.

## 소모품의 두 가지 쓰임
같은 소모품 마스터지만 현장에서는 두 갈래로 쓰입니다.
- **설비 장착용(금형·지그·블레이드 등)**: 설비에 장착해 타수(사용횟수)가 누적되며, 기대수명에 도달하면 교체합니다. 키오스크의 "소모성 설비부품" 관리 대상입니다.
- **제품 BOM 투입용**: 제품을 만들 때 함께 투입되는 소모성 부품입니다. 제품의 BOM에 포함되어 생산수량만큼 차감됩니다.

> 어느 제품·설비에서 어떤 소모품을 쓰는지는 오른쪽 **사용매핑**으로 정의합니다.

---

## ① 기본정보

| 컬럼 | 역할 / 의미 |
|------|------|
| **소모품 코드(consumableCode)** | 소모품을 식별하는 **고유 코드**입니다. 예: `CM-AP-110`. 입출고·재고·사용매핑이 이 코드로 연결되므로 한 번 등록하면 변경하지 않습니다(수정 모드에서는 잠김). |
| **소모품명(consumableName)** | 현장에서 식별할 이름입니다. 예: `110단자 압착금형`. |
| **분류(category)** | 소모품 종류입니다. 금형(MOLD)·지그(JIG)·공구(TOOL) 중 선택합니다(공통코드 CONSUMABLE_CATEGORY). 목록 필터·통계의 기준입니다. |
| **이미지(imageUrl)** | 소모품 사진입니다. **저장(등록) 후에만** 업로드할 수 있으며, 목록 썸네일로도 표시됩니다. |

## ② 수명 / 관리

| 컬럼 | 역할 / 의미 |
|------|------|
| **기대수명(expectedLife)** | 이 소모품을 교체 없이 쓸 수 있는 **누적 타수(사용횟수)**입니다. 예: `100000`. 누적 타수가 이 값에 도달하면 상태가 **교체필요(REPLACE)**로 바뀝니다. |
| **경고 임계치(warningCount)** | 교체 임박을 미리 알리는 타수입니다. 예: `80000`. 누적 타수가 이 값에 도달하면 상태가 **경고(WARNING)**로 바뀌어, 교체를 준비하도록 합니다. 보통 기대수명보다 작게 둡니다. |
| **안전재고(safetyStock)** | 재고 부족 판단에 쓰는 기준 수량입니다. 보유 재고가 이 값 미만이면 부족으로 봅니다. |

> 누적 타수(현재 사용횟수)는 키오스크 생산·교체 등에서 자동으로 누적·리셋됩니다. 이 화면에서는 그 **기준값(기대수명·경고 임계치)**만 정의합니다.

## ③ 거래처 / 위치

| 컬럼 | 역할 / 의미 |
|------|------|
| **보관위치(location)** | 소모품을 두는 기본 보관 장소입니다. 예: `금형실-A1`. |
| **거래처(vendor)** | 소모품을 공급하는 거래처/제조사입니다. 예: `JST`. |
| **단가(unitPrice)** | 소모품 1개의 단가입니다. 입고 단가·자산 평가 참고용입니다. |

---

## 사용처 매핑 (오른쪽 패널)
선택한 소모품을 **어느 제품(모델)·어느 설비**에서 쓰는지 등록합니다. 키오스크 생산실적에서 작업지시(제품·설비)에 맞는 소모품을 찾는 기준이 됩니다.

| 항목 | 역할 / 의미 |
|------|------|
| **제품/모델(productItemCode)** | 이 소모품을 사용하는 제품/모델입니다. 완제품·반제품 품목만 선택 가능합니다. |
| **설비(equipCode)** | 그 제품을 생산할 때 이 소모품을 쓰는 설비입니다. |
| **단위사용량(usagePerUnit)** | **단위 생산당 소모되는 타수(사용횟수)**입니다. 생산수량 × 이 값만큼 소모품 타수가 누적됩니다. 기본값 1. |
| **사용여부(useYn)** | `Y`일 때만 활성 매핑입니다. `N`이면 매핑은 남기되 사용에서 제외합니다(목록의 `Y` 배지를 눌러 토글). |
| **비고(remark)** | 매핑 관리용 메모입니다. |

> 사용매핑은 **제품 + 설비 + 소모품** 조합으로 1건입니다. 같은 소모품을 여러 제품·설비에 매핑할 수 있습니다.

## 사용 순서
1. 상단 **소모품 등록**(또는 행 ✏️)으로 슬라이드 폼을 엽니다.
2. **기본정보**(코드·이름·분류)를 입력하고 저장합니다.
3. 저장 후 폼을 다시 열어 **이미지**를 업로드합니다(등록 직후엔 업로드 불가).
4. **수명/관리**(기대수명·경고 임계치·안전재고), **거래처/위치**(보관위치·거래처·단가)를 채웁니다.
5. 목록에서 소모품 행을 선택해 오른쪽 **사용매핑**에 제품·설비·단위사용량을 등록합니다.

## 입력 규칙 / 검증
- **소모품 코드**와 **소모품명**은 필수입니다. 비어 있으면 저장 버튼이 비활성화됩니다.
- 소모품 코드는 중복될 수 없습니다(이미 있으면 저장이 거부됩니다).
- 이미지는 **저장 후**에만 등록할 수 있습니다(jpg/png/gif/webp, 5MB 이하).
- 사용매핑은 **제품과 설비를 모두 선택**해야 저장됩니다.

## 자주 묻는 질문
- **Q.** 코드를 잘못 입력했는데 수정 모드에서 바꿀 수 없어요.
  **A.** 소모품 코드는 입출고·재고·매핑이 연결되는 키라 변경할 수 없습니다. 삭제 후 다시 등록하세요.
- **Q.** 등록할 때 이미지 업로드 칸이 흐리게 나와요.
  **A.** 이미지는 저장 후 업로드합니다. 먼저 기본정보를 저장한 뒤 다시 열어 업로드하세요.
- **Q.** 기대수명과 경고 임계치는 어떻게 다른가요?
  **A.** 경고 임계치에 도달하면 **경고(교체 준비)**, 기대수명에 도달하면 **교체필요**로 상태가 바뀝니다. 미리 알림 → 실제 교체 시점 순서입니다.
- **Q.** 목록에 안 보이는 소모품이 있어요.
  **A.** 목록은 사용 중(`useYn=Y`)인 소모품만 보여줍니다. 분류 필터·검색어도 함께 확인하세요.
