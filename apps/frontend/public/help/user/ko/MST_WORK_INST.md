# 작업지도서 관리

## 화면 목적
품목과 공정별로 **작업 지침(작업지도서)**을 등록·관리합니다. 현장 작업자가 수행해야 할 작업 순서, 주의사항, 참조 이미지/파일을 문서 형태로 관리합니다. 품목·공정·리비전 조합으로 문서를 식별하며, 이력 관리를 위해 리비전을 부여할 수 있습니다.

## 화면 구성
- **메인 영역 — DataGrid 목록**: 등록된 작업지도서 목록을 표시합니다. 열 필터·정렬·내보내기 기능을 제공합니다.
- **우측 패널 (미리보기)**: 행 클릭 시 오른쪽에 작업지도서 상세 정보(제목·리비전·첨부파일·본문)를 표시합니다.
- **우측 패널 (등록/수정)**: 추가 또는 수정 버튼 클릭 시 오른쪽에 폼 패널이 열립니다.

---

## ① DataGrid 목록 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **품목코드(itemCode)** | 작업지도서가 적용되는 대상 품목의 코드입니다. `ITEM_MASTERS.ITEM_CODE`를 참조합니다. |
| **품목명(itemName)** | 대상 품목의 이름입니다. |
| **공정코드(processCode)** | 작업지도서가 속한 공정의 코드입니다. 품목코드·리비전과 함께 문서의 식별자(PK)를 구성합니다. |
| **문서제목(title)** | 작업지도서의 제목입니다. 현장에서 문서를 구분하는 기준이 됩니다. |
| **리비전(revision)** | 문서의 개정 버전입니다. 신규 등록 시 기본값은 `A`이며, 개정 시 B, C, ... 순으로 올라갑니다. |
| **최종수정일(updatedAt)** | 문서가 마지막으로 수정된 일시입니다. |
| **사용여부(useYn)** | `Y` = 사용 중(활성), `N` = 미사용(비활성). 초록색/회색 원으로 표시됩니다. |

---

## ② 등록/수정 폼 필드

| 필드 | 역할 / 의미 |
|------|------|
| **품목코드(itemCode)** | 작업지도서를 적용할 품목코드입니다. 등록 후에는 수정할 수 없습니다. |
| **공정코드(processCode)** | 해당 품목 내에서 작업지도서가 속한 공정입니다. 등록 후 수정 불가. |
| **문서제목(title)** | 작업지도서의 제목입니다. 필수 입력값입니다. |
| **리비전(revision)** | 문서 버전입니다. 비워두면 `A`로 저장되며, 등록 후 수정 불가. |
| **첨부파일(imageUrl)** | 작업지도서에 첨부할 이미지/PDF/문서 파일입니다. **드래그&드롭 또는 파일 선택**으로 업로드하거나, **URL을 직접 입력**할 수도 있습니다. |
| **내용(content)** | 작업 순서, 주의사항 등 작업지도서의 본문 내용입니다. |

## 사용 순서
1. **추가** 버튼을 클릭하여 새 작업지도서를 등록합니다(우측 패널).
2. 품목코드·공정코드·문서제목을 입력하고, 필요 시 첨부파일 업로드와 본문 내용을 작성합니다.
3. 목록에서 행을 클릭하면 **미리보기**로 상세 내용을 확인할 수 있습니다.
4. 미리보기에서 **수정** 버튼 또는 그리드의 편집 아이콘을 클릭하여 내용을 변경합니다.

## 입력 규칙 / 검증
- 품목코드·공정코드·문서제목은 **필수 입력**입니다. 하나라도 비어 있으면 저장 버튼이 비활성화됩니다.
- 동일한 `품목코드 + 공정코드 + 리비전` 조합은 중복 등록할 수 없습니다.
- 첨부파일은 이미지, PDF, Office 문서, TXT 파일을 지원하며 **최대 10MB**로 제한됩니다.

## 자주 묻는 질문
- **Q.** 리비전은 어떻게 관리하나요? **A.** 신규 등록 시 기본 `A`입니다. 기존 문서를 수정할 때는 현재 리비전을 유지하거나 새 리비전(예: B)으로 작성할 수 있습니다. 같은 품목·공정에 여러 리비전을 동시에 유지할 수 있습니다.
- **Q.** 이미 등록된 품목코드·공정코드를 바꿀 수 있나요? **A.** 등록 후에는 PK(품목코드·공정코드·리비전)를 변경할 수 없습니다. 새 문서로 등록해야 합니다.
- **Q.** 파일 업로드 없이 URL만 입력해도 되나요? **A.** 네, 외부 이미지/문서의 URL을 직접 입력할 수 있습니다. 업로드와 URL 중 하나를 선택하면 됩니다.
