# 점검이력 조회

## 화면 목적
일상점검(DAILY)과 정기점검(PERIODIC)의 모든 점검 이력을 통합하여 조회합니다. 조회 전용 화면으로, 다양한 필터 조건으로 원하는 이력을 검색할 수 있습니다.

## 화면 구성
- **필터 영역**: 검색어 + 점검유형(일상/정기) + 설비유형 + 결과(합격/불합격/조건부) + 기간 선택
- **DataGrid 목록**: 검색 조건에 맞는 점검 이력을 표 형태로 표시합니다. 컬럼 필터, Excel 내보내기 기능을 제공합니다.

## DataGrid 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **점검일** | 점검 수행일자입니다. |
| **점검유형** | DAILY(일상점검) 또는 PERIODIC(정기점검) 구분입니다. |
| **설비코드** | 점검 대상 설비의 고유 코드입니다. |
| **설비명** | 점검 대상 설비의 명칭입니다. |
| **설비유형** | 설비 유형 구분입니다. |
| **점검자** | 점검을 수행한 작업자 이름입니다. |
| **결과** | 점검 종합 결과(PASS / FAIL / CONDITIONAL)입니다. |
| **비고** | 점검 관련 추가 메모입니다. |

## 사용 순서
1. 상단 필터 영역에서 조건을 설정합니다(점검유형, 설비유형, 결과, 기간).
2. **조회** 버튼을 클릭하여 이력을 검색합니다.
3. 결과 목록에서 각 항목을 확인합니다.
4. 필요 시 Excel 내보내기로 데이터를 다운로드할 수 있습니다.

## 필터 조건

| 필터 | 설명 |
|------|------|
| **검색어** | 설비코드 또는 설비명으로 검색합니다. |
| **점검유형** | 전체 / 일상점검(DAILY) / 정기점검(PERIODIC)을 선택합니다. |
| **설비유형** | 특정 설비 유형으로 필터링합니다. |
| **결과** | 합격(PASS) / 불합격(FAIL) / 조건부(CONDITIONAL)로 필터링합니다. |
| **점검일 기간** | 조회할 점검일의 시작일과 종료일을 선택합니다. |
