# 제품포장관리

## 화면 목적
검사에 합격한 완제품 FG 시리얼을 박스에 담아 출하 가능한 박스 단위로 만드는 화면입니다. 박스를 만들고, 시리얼을 스캔해 담고, 박스를 마감한 뒤 박스 라벨을 출력합니다.

> 이 화면의 핵심은 `OPEN` 박스를 작업 중인 박스로 유지하다가, 포장이 끝나면 `CLOSED` 박스로 마감하는 것입니다.

## 화면 구성
- **상단 버튼**: 새로고침, 포장 대기, 박스 생성.
- **박스 목록**: 생성일, 박스번호/품목코드 검색, 상태 필터로 박스를 조회합니다. 상태를 지정하지 않으면 기간 밖의 `OPEN` 박스도 함께 보여 작업 중인 박스를 놓치지 않게 합니다.
- **선택 행 액션 바**: 선택한 박스에 대해 제품 담기, 박스 마감, 박스 재오픈, 라벨 재발행, 빈 박스 삭제를 실행합니다.
- **우측 박스 구성 패널**: 선택 박스의 품목, 현재 포장수량/박스입수량, 담긴 시리얼 목록, 상태, 팔레트 번호를 확인합니다.
- **모달**: 포장 대기 목록, 박스 생성, 시리얼 추가, 박스 라벨, 삭제 확인 모달이 열립니다.

## ① 박스 목록 컬럼
| 컬럼 | 역할 / 의미 |
|------|------|
| **박스번호(boxNo)** | 박스를 식별하는 번호입니다. 박스 라벨 바코드와 같은 값이며 행 선택, 라벨 출력, 후속 팔레트 적재 기준이 됩니다. |
| **품목코드(itemCode)** | 박스에 담을 완제품 품목입니다. 한 박스에는 같은 품목의 시리얼만 담을 수 있습니다. |
| **품목명(itemName)** | 품목코드의 표시 이름입니다. 마스터에 이름이 없으면 `-`로 보입니다. |
| **포장수량(qty / boxQty)** | 현재 박스에 담긴 시리얼 개수와 품목 마스터의 박스입수량을 함께 보여줍니다. 예: `12 / 20`. |
| **상태(status)** | `OPEN`은 담기/삭제/마감 가능, `CLOSED`는 마감 완료, `SHIPPED`는 출하 완료 상태입니다. |
| **마감일시(closeAt)** | 박스를 마감한 일시입니다. 아직 `OPEN`이면 `-`로 표시됩니다. |

## ② 입력·상세 항목
| 항목 | 역할 / 의미 |
|------|------|
| **생성일(createdFrom / createdTo)** | 박스 생성일 조회 범위입니다. 상태를 전체로 두면 미마감 `OPEN` 박스는 기간 밖이어도 함께 표시됩니다. |
| **검색어(search)** | 박스번호 또는 품목코드로 목록을 좁힙니다. |
| **상태(status)** | 전체, 오픈, 마감, 출하 상태로 목록을 좁힙니다. |
| **포장 품목(itemCode)** | 박스 생성 모달에서 선택하는 완제품 품목입니다. 박스번호는 저장 시 자동 생성됩니다. |
| **시리얼번호(serialInput)** | 제품 담기 모달에서 스캔하거나 입력하는 FG 바코드입니다. Enter 또는 스캐너 CR/LF가 들어오면 바로 추가됩니다. |
| **구성 시리얼(boxItems / serialList)** | 선택 박스에 실제로 담긴 시리얼 목록입니다. `OPEN` 상태에서는 각 시리얼을 제거할 수 있습니다. |
| **팔레트(palletNo)** | 박스가 팔레트에 적재되면 표시됩니다. 팔레트가 있으면 박스 재오픈과 빈 박스 삭제가 제한됩니다. |

## 버튼별 기능
| 버튼 | 언제 사용 | 동작 | 결과 / 주의 |
|------|------|------|------|
| **새로고침** | 목록과 포장 대기 시리얼을 다시 보고 싶을 때 | 박스 목록과 포장 대기 목록을 다시 조회합니다. | 검색 조건은 유지됩니다. |
| **포장 대기** | 아직 박스에 담기지 않은 검사 합격 FG를 확인할 때 | 품목별 포장 대기 시리얼 모달을 엽니다. | 이 모달은 확인용입니다. 실제 담기는 `제품 담기`에서 합니다. |
| **박스 생성** | 새 박스를 만들 때 | 완제품 품목 선택 모달을 열고 저장하면 `OPEN` 박스가 생성됩니다. | 수량은 0에서 시작하고, 시리얼을 담을 때 증가합니다. |
| **제품 담기** | 선택 박스가 `OPEN`일 때 | 시리얼 추가 모달을 열고 스캔한 FG를 박스에 담습니다. | 같은 품목의 검사 합격 FG만 담을 수 있습니다. |
| **시리얼 추가(+)** | 제품 담기 모달에서 시리얼을 입력했을 때 | 입력한 시리얼 1건을 현재 박스에 추가합니다. | Enter나 스캐너 줄바꿈도 같은 동작을 합니다. |
| **방금 추가 취소 / 시리얼 제거** | 잘못 스캔한 시리얼을 빼야 할 때 | 선택한 시리얼을 현재 `OPEN` 박스에서 제거합니다. | 마감된 박스에서는 제거할 수 없습니다. 필요하면 먼저 재오픈합니다. |
| **포장 완료 · 라벨 출력** | 현재 박스 담기가 끝났을 때 | 박스를 마감하고 박스 라벨 모달을 열어 출력합니다. | 마감되면 상태가 `CLOSED`가 되고 OQC 의뢰가 자동 생성됩니다. |
| **박스 마감** | 목록에서 바로 마감할 때 | 선택한 `OPEN` 박스를 `CLOSED`로 바꿉니다. | 마감 후에는 제품 담기와 시리얼 제거가 막힙니다. |
| **박스 재오픈** | 마감 후 팔레트 적재 전 수정이 필요할 때 | `CLOSED` 박스를 다시 `OPEN`으로 되돌립니다. | 팔레트가 배정된 박스는 재오픈할 수 없습니다. |
| **라벨 재발행** | 이미 담긴 박스 라벨을 다시 출력할 때 | 박스 라벨 모달을 열고 인쇄합니다. | 포장수량이 0이면 버튼이 비활성화됩니다. |
| **빈 박스 삭제** | 잘못 만든 빈 박스를 없앨 때 | 제품이 담기지 않은 `OPEN` 박스를 삭제합니다. | 수량, 시리얼, 팔레트, OQC 이력이 있으면 삭제할 수 없습니다. |

## 업무 흐름
1. 상단 **포장 대기**를 눌러 어떤 품목과 시리얼이 대기 중인지 확인합니다.
2. 새 품목을 포장해야 하면 **박스 생성**을 누르고 완제품 품목을 선택해 `OPEN` 박스를 만듭니다.
3. 목록에서 작업할 박스 행을 선택합니다. 우측 **박스 구성** 패널에 품목과 현재 수량이 표시되는지 확인합니다.
4. **제품 담기**를 눌러 시리얼 추가 모달을 엽니다.
5. FG 바코드를 스캔하거나 입력하고 Enter를 누릅니다. 정상 추가되면 현재 박스의 구성 수량과 시리얼 목록이 갱신됩니다.
6. 잘못 담은 시리얼은 **방금 추가 취소** 또는 시리얼 목록의 제거 아이콘으로 즉시 제거합니다.
7. 박스입수량에 도달하면 자동으로 마감과 라벨 출력이 진행됩니다. 입수량보다 적게 담아 마감해야 하면 **포장 완료 · 라벨 출력**을 직접 누릅니다.
8. 마감된 박스는 `CLOSED`가 되며, 이후 **팔레트관리**에서 팔레트에 적재하고 출하검사와 출하확정으로 진행합니다.
9. 마감 후 수정이 필요하면 팔레트 적재 전 **박스 재오픈**을 눌러 `OPEN`으로 되돌린 뒤 시리얼을 수정하고 다시 마감합니다.

## 입력 규칙 / 검증
- 박스 생성은 완제품 품목을 선택해야 저장됩니다.
- 제품 담기는 `OPEN` 박스에서만 가능합니다.
- 시리얼은 검사 합격 상태이고, 아직 다른 박스에 담기지 않았으며, 현재 박스의 품목과 같아야 합니다.
- 품목 마스터에 박스입수량이 있으면 그 수량을 초과해 담을 수 없습니다.
- `CLOSED` 박스는 제품 담기와 시리얼 제거가 막힙니다.
- 팔레트가 배정된 박스는 재오픈할 수 없습니다.
- 빈 박스 삭제는 `OPEN`, 포장수량 0, 시리얼 없음, 팔레트 없음, OQC 이력 없음일 때만 가능합니다.

## 자주 묻는 질문
- **Q. 포장할 시리얼이 보이지 않습니다.**
  **A.** 포장 대기 모달에서 먼저 확인하세요. 검사 합격 전이거나 이미 다른 박스에 담긴 FG는 포장 대기에 나오지 않습니다.
- **Q. 스캔했는데 시리얼 추가가 실패합니다.**
  **A.** 품목 불일치, 검사 미합격, 중복 포장, 박스입수량 초과, 박스 상태가 `OPEN`이 아님 중 하나입니다. 화면 오류 메시지와 우측 박스 품목을 같이 확인하세요.
- **Q. 박스를 마감했는데 다시 수정해야 합니다.**
  **A.** 팔레트에 적재되기 전이면 **박스 재오픈**으로 `OPEN` 상태로 되돌린 뒤 수정하고 다시 마감합니다.
- **Q. 라벨만 다시 출력하고 싶습니다.**
  **A.** 박스 행을 선택하고 **라벨 재발행**을 누릅니다. 라벨 템플릿을 선택한 뒤 인쇄할 수 있습니다.
