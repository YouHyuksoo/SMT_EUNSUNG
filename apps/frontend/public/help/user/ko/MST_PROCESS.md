# 공정마스터

## 화면 목적
생산에 사용되는 **공정(절단·압착·조립·검사·포장 등)을 등록·분류·관리**하는 기준정보 화면입니다. 여기 등록된 공정이 라우팅·생산실적·CAPA·공정조건의 기준이 됩니다. 또한 각 공정에 **실제 설비를 배치(연결)**해 어떤 설비가 어느 공정에서 가동되는지 정의합니다.

## 화면 구성
- **왼쪽(공정 목록)**: 공정 그리드입니다. 행을 클릭하면 그 공정이 선택되고, 오른쪽에 배치 설비가 표시됩니다. 상단의 **공정 추가** 버튼, 행의 ✏️(수정)·🗑(삭제) 아이콘으로 CRUD를 처리합니다.
- **오른쪽(배치된 설비)**: 왼쪽에서 선택한 공정에 배치된 설비 목록입니다. **설비 배치** 버튼으로 설비를 연결하고, 각 행의 🗑 아이콘으로 배치를 해제합니다. 공정을 선택하기 전에는 "좌측에서 공정을 선택해주세요." 안내가 표시됩니다.
- 상단 우측 **새로고침** 버튼으로 목록·설비수를 다시 불러옵니다.

```
공정(PROCESS_MASTERS)  1 ──< N  설비배치(PROCESS_EQUIPMENTS) >── 1  설비(EQUIP_MASTERS)
   └ 왼쪽 그리드                    └ 연결 테이블                       └ 오른쪽 그리드 표시
```

## ① 공정 컬럼 (왼쪽 목록 · 등록/수정 폼)

| 컬럼 | 역할 / 의미 |
|------|------|
| **공정코드(processCode)** | 공정을 식별하는 **고유 코드**입니다. 라우팅·실적·공정조건이 이 코드로 공정을 참조하므로, 등록 후에는 변경하지 않는 것을 원칙으로 합니다(수정 폼에서 비활성). 필수. |
| **공정명(processName)** | 현장에서 식별할 공정 이름입니다(예: 절단, 압착). 필수. |
| **공정유형(processType)** | 공정의 큰 유형 분류입니다(공통코드 PROCESS_TYPE — 절단/압착/조립/검사/포장 등). 생산·실적 처리 방식 구분의 기준이 됩니다. 필수. |
| **공정 대분류(processCategory)** | 공정을 묶는 상위 분류입니다(공통코드 PROCESS_CATEGORY — 조립(ASSY)·검사(INSP)·절단·융착·포장 등). 비워둘 수 있으며, 목록에서 다중 필터로 활용됩니다. |
| **라인구분(lineType)** | 공정이 속한 생산 라인 구분입니다. **LV=저전압 / HV=고전압 / CM=공통**(공통코드 LINE_TYPE). 저전압·고전압 라인을 구분해 운영할 때 사용하며, 비워둘 수 있습니다. |
| **설비(equipCount)** | 그 공정에 **배치된 설비 수**입니다(자동 집계, 입력 항목 아님). 값이 있으면 파란 배지로 강조됩니다. |
| **정렬순서(sortOrder)** | 목록에서의 표시 순서입니다. 작은 값이 위로 오며, 생산 흐름 순서대로 정렬할 때 사용합니다. 기본값 0. |
| **사용여부(useYn)** | `Y`일 때만 조회·선택·사용 대상입니다. 사용 중지는 `N`. |
| **비고(remark)** | 공정 관리 참고사항입니다. |

## ② 배치된 설비 컬럼 (오른쪽 목록)

> 이 영역은 설비마스터에 등록된 설비를 **읽어와 표시**하는 것입니다. 설비 자체의 값은 설비마스터에서 관리하고, 여기서는 공정에 **연결/해제**만 합니다.

| 컬럼 | 역할 / 의미 |
|------|------|
| **설비코드(equipCode)** | 배치된 설비의 고유 코드입니다. |
| **설비명(equipName)** | 설비 이름입니다. |
| **설비유형(equipType)** | 설비 유형입니다(공통코드 EQUIP_TYPE). 값이 없으면 `-`. |
| **모델명(modelName)** | 설비 모델명입니다. 값이 없으면 `-`. |
| **제조사(maker)** | 설비 제조사입니다. 값이 없으면 `-`. |
| **라인코드(lineCode)** | 설비가 속한 라인 코드입니다. 값이 없으면 `-`. |
| **상태(status)** | 설비 가동 상태입니다(공통코드 EQUIP_STATUS — 정상/고장 등). |
| **사용여부(useYn)** | 설비의 사용여부입니다. `Y`만 배치 후보로 표시됩니다. |

## 사용 순서
1. 왼쪽 상단 **공정 추가**(또는 행 ✏️)로 등록/수정 모달을 엽니다.
2. **공정코드·공정명·공정유형**(필수)을 입력하고, 필요 시 대분류·라인구분·정렬순서·비고를 채워 **저장**합니다.
3. 왼쪽 목록에서 공정 행을 클릭해 선택합니다.
4. 오른쪽 **설비 배치** 버튼을 눌러 셀렉트에서 설비를 고르고 **추가**합니다(이미 배치된 설비는 목록에 안 나옴).
5. 배치를 해제하려면 오른쪽 설비 행의 🗑 아이콘을 누릅니다.

## 입력 규칙 / 검증
- **공정코드·공정명·공정유형**이 비어 있으면 저장되지 않습니다(필수).
- **공정코드는 중복 불가**입니다. 이미 존재하는 코드로 저장하면 오류가 납니다.
- **라인구분**은 `LV / HV / CM` 중 하나만 허용됩니다.
- **정렬순서**는 0 이상의 정수입니다.
- 설비 배치 시 **이미 배치된 설비를 다시 추가하면 오류**가 납니다(셀렉트 목록에서 자동 제외됨).

## 자주 묻는 질문
- **Q.** 공정코드를 나중에 바꿀 수 있나요?
  **A.** 수정 폼에서 공정코드는 비활성입니다. 라우팅·실적이 이 코드로 연결되므로 변경하지 않는 것을 원칙으로 합니다.
- **Q.** "설비" 컬럼의 숫자는 뭔가요?
  **A.** 그 공정에 배치된 설비 수를 자동 집계한 값입니다. 직접 입력하는 항목이 아닙니다.
- **Q.** 오른쪽 설비 목록에 원하는 설비가 안 보여요.
  **A.** 이미 그 공정에 배치된 설비는 셀렉트에서 제외됩니다. 또한 설비마스터에서 `사용여부=Y`인 설비만 배치 후보로 나옵니다.
- **Q.** 라인구분(LV/HV/CM)은 꼭 입력해야 하나요?
  **A.** 선택 항목입니다. 저전압·고전압 라인을 구분 운영할 때만 지정하면 됩니다.
