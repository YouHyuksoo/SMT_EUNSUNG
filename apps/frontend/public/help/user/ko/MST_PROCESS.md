# 공정마스터

## 화면 목적
생산에 사용되는 **공정(Workstage)을 등록·분류·관리**하는 기준정보 화면입니다. 여기 등록된 공정이 라우팅·생산실적·공정 CAPA의 기준이 됩니다. 또한 각 공정에 **실제 설비를 배치**해 어떤 설비가 어느 공정에서 가동되는지 정의합니다.

## 화면 구성
- **왼쪽(공정 목록)**: 공정 그리드입니다. 행을 클릭하면 그 공정이 선택되고, 오른쪽에 배치 설비가 표시됩니다. 상단의 **공정 추가** 버튼, 행의 ✏️(수정)·🗑(삭제) 아이콘으로 CRUD를 처리합니다.
- **오른쪽(배치된 설비)**: 왼쪽에서 선택한 공정에 배치된 설비 목록입니다. **설비 배치** 버튼으로 설비를 연결하고, 각 행의 🗑 아이콘으로 배치를 해제합니다. 공정을 선택하기 전에는 "좌측에서 공정을 선택해주세요." 안내가 표시됩니다.
- 공정 추가/수정은 오른쪽에서 밀려나오는 **슬라이드 패널**에서 처리하며, 항목이 많아 **기본 정보 / 생산능력 / 표준시간 / 원가 / 불량·기타** 5개 섹션으로 나뉩니다.
- 상단 우측 **새로고침** 버튼으로 목록·설비수를 다시 불러옵니다.

```
공정(IP_PRODUCT_WORKSTAGE)  1 ──< N  설비(IMCN_MACHINE)
   └ 왼쪽 그리드                        └ WORKSTAGE_CODE로 소속 공정을 가리킴 → 오른쪽 그리드
```

> **설비는 한 번에 하나의 공정에만 속합니다.** 별도의 연결 테이블 없이, 설비 자신이 "나는 어느 공정 소속인가"를 들고 있는 구조입니다.

## ① 공정 필드

### 기본 정보

| 항목 | 역할 / 의미 |
|------|------|
| **공정코드** | 공정을 식별하는 **고유 코드**입니다(예: `W020`). 라우팅·실적이 이 코드로 공정을 참조하므로 등록 후에는 변경할 수 없습니다(수정 시 비활성). 필수, 최대 10자. |
| **공정명** | 현장에서 식별할 공정 이름입니다(예: SP, MOUNT, REFLOW). 필수. |
| **공정유형** | **일반공정(I) / 최종공정(L) / 검사공정(Q)** 중 하나입니다. 최종공정은 완제품 실적 집계의 기준이 됩니다. 필수. |
| **정렬순서** | 공정 흐름 순서입니다. 작은 값이 먼저 표시되고 먼저 진행됩니다. 필수. |
| **시작공정** | 생산 흐름의 **첫 공정**인지 여부입니다. 시작 공정에서 투입 실적이 생성됩니다. |
| **공정그룹** | 공정이 속한 큰 묶음입니다 — SMT / 후공정(PBA) / 패킹 / 수리실 / 창고. 목록에서 다중 필터로 활용됩니다. |
| **공정상태** | 공정의 현재 상태입니다(작업중·정지·수리·설비고장 등). |
| **라인코드** | 공정이 소속된 생산 라인입니다. `*`는 **라인 무관(공용 공정)**을 뜻합니다. |
| **부서 / 교대** | 공정을 담당하는 부서, 기본 교대조입니다. |
| **대표설비** | 실적 집계의 기준이 되는 대표 설비입니다. 오른쪽 배치 설비 목록과는 **별개**로 지정합니다. |
| **코스트센터 / 디스플레이 그룹 / PLC 주소** | 각각 원가 집계 단위, 모니터링 화면 묶음(`*`=미지정), 실적 수집용 PLC 주소입니다. |

### 생산능력

| 항목 | 역할 / 의미 |
|------|------|
| **UPH** | 시간당 생산량(Units Per Hour)입니다. 생산능력 계산의 기준입니다. |
| **표준수량 / 생산능력 / 능력단위** | 표준 생산 수량, 공정의 생산능력과 그 단위(ST=수량, KG=중량)입니다. |
| **가동률(%)** | 공정 가동률입니다. |
| **작업효율(%) / 설비효율(%)** | 각각 작업자 기준, 설비 기준 효율입니다. |
| **작업자수 / 설비대수** | 공정에 투입되는 표준 인원과 설비 대수입니다. |

### 표준시간

| 항목 | 역할 / 의미 |
|------|------|
| **정상시간(ST) / 초과시간(OT)** | 정상 근무시간과 초과 근무시간입니다. |
| **작업자시간 / 설비시간** | 순수 작업시간과 순수 가동시간입니다. |
| **준비·대기·이동 시간** | 셋업, 대기, 다음 공정 이동에 걸리는 시간입니다. |
| **총작업시간** | 위 시간들을 포함한 공정 전체 소요시간입니다. |

### 원가

| 항목 | 역할 / 의미 |
|------|------|
| **임률 / 경비율 / 기계경비율** | 시간당 노무비·경비·기계경비 단가입니다. |
| **조립비 반영** | 원가 계산 시 조립비를 반영할지 여부입니다. |

### 불량·기타

| 항목 | 역할 / 의미 |
|------|------|
| **불량률 통제** | 이 공정에서 불량률을 통제할지 여부입니다. |
| **최대 불량률(%)** | 허용 가능한 최대 불량률입니다. **불량률 통제=불량통제함**일 때만 의미가 있습니다. |
| **불량수량 추출** | 실적 집계 시 불량수량을 별도로 추출할지 여부입니다. |
| **반제품 생성** | 공정 완료 시 하위 반제품을 자동 생성할지 여부입니다. |

> 왼쪽 목록 그리드에는 공정코드·공정명·공정유형·공정그룹·라인코드·시작공정·설비수·UPH·정렬순서만 표시됩니다. 나머지 항목은 수정 패널에서 확인합니다.

## ② 배치된 설비 컬럼 (오른쪽 목록)

> 이 영역은 설비마스터에 등록된 설비를 **읽어와 표시**합니다. 설비 자체의 값은 설비마스터에서 관리하고, 여기서는 공정에 **배치/해제**만 합니다.

| 컬럼 | 역할 / 의미 |
|------|------|
| **설비코드 / 설비명** | 배치된 설비의 고유 코드와 이름입니다. |
| **설비유형** | 설비 유형입니다. 값이 없으면 `-`. |
| **모델명 / 제조사 / 라인코드** | 설비의 모델·제조사·소속 라인입니다. 값이 없으면 `-`. |
| **상태** | 설비 가동 상태입니다(정상/고장 등). |
| **사용여부** | 설비의 사용여부입니다. `Y`인 설비만 배치 후보로 나옵니다. |

## 사용 순서
1. 왼쪽 상단 **공정 추가**(또는 행 ✏️)로 슬라이드 패널을 엽니다.
2. **공정코드·공정명·공정유형·정렬순서**(필수)를 입력하고, 필요 시 나머지 섹션을 채워 **저장**합니다.
3. 왼쪽 목록에서 공정 행을 클릭해 선택합니다.
4. 오른쪽 **설비 배치** 버튼을 눌러 셀렉트에서 설비를 고르고 **추가**합니다.
5. 배치를 해제하려면 오른쪽 설비 행의 🗑 아이콘을 누릅니다. 해당 설비는 **미배치 상태**가 됩니다.

## 입력 규칙 / 검증
- **공정코드·공정명·공정유형·정렬순서**가 비어 있으면 저장되지 않습니다(필수).
- **공정코드는 중복 불가**입니다. 이미 존재하는 코드로 저장하면 오류가 납니다. 최대 10자이며 입력 시 자동으로 대문자로 변환됩니다.
- **정렬순서**는 0 이상의 정수입니다.
- 설비 배치 시 **이미 이 공정에 배치된 설비를 다시 추가하면 오류**가 납니다(셀렉트 목록에서 자동 제외됨).
- **다른 공정에 배치된 설비를 선택하면 그 설비가 이 공정으로 옮겨집니다.** 설비는 한 공정에만 속하기 때문입니다.

## 자주 묻는 질문
- **Q.** 공정코드를 나중에 바꿀 수 있나요?
  **A.** 수정 패널에서 공정코드는 비활성입니다. 라우팅·실적이 이 코드로 연결되므로 변경할 수 없습니다.
- **Q.** "설비" 컬럼의 숫자는 뭔가요?
  **A.** 그 공정에 배치된 설비 수를 자동 집계한 값입니다. 직접 입력하는 항목이 아닙니다.
- **Q.** 셀렉트에서 설비를 골랐는데, 그 설비가 이미 다른 공정에 있었어요.
  **A.** 설비는 한 공정에만 속하므로 **이 공정으로 이동**합니다. 이전 공정의 설비 수가 하나 줄어듭니다.
- **Q.** 배치를 해제하면 설비가 삭제되나요?
  **A.** 아닙니다. 설비는 그대로 남고 **미배치 상태**가 될 뿐입니다. 설비마스터에서는 계속 조회됩니다.
- **Q.** 라인코드가 `*`인 공정이 많은데 정상인가요?
  **A.** `*`는 "라인 무관(공용 공정)"을 뜻하는 현장 관례값입니다. 정상입니다.
- **Q.** 대표설비와 오른쪽 배치 설비 목록은 뭐가 다른가요?
  **A.** 배치 설비 목록은 그 공정에서 가동되는 **모든 설비**이고, 대표설비는 그중 실적 집계의 **기준이 되는 한 대**를 지정하는 항목입니다.
