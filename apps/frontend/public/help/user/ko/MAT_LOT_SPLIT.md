# 자재분할

## 화면 목적
입고완료된 자재 LOT(시리얼) 하나를 **두 조각으로 나누는** 화면입니다. 한 시리얼 안에 들어 있는 수량 중 일부만 다른 작업·창고·용도로 분리해야 할 때 사용합니다. 분할하면 원본 시리얼은 폐기(SPLIT)되고, **분할수량 조각**과 **잔량 조각** 두 개가 각각 새 자재시리얼로 발번됩니다.

> 분할 = 1개 시리얼 → 2개 시리얼. 원본은 그대로 남지 않고 폐기되며, 두 조각 모두 새 시리얼·새 라벨을 받습니다.

## 분할 흐름
```
입고완료 LOT(시리얼 1개, 수량 100)
  → 분할수량 30 입력
  → 원본 시리얼 폐기(SPLIT, 재고 0)
  → 신규 시리얼 A(30) + 신규 시리얼 B(70) 발번
  → 두 조각의 자재 라벨 출력
```

## 화면 구성
- **상단 통계 카드**: 분할 가능한 전체 LOT 수와 총수량을 보여줍니다.
- **메인 그리드(분할 가능 LOT 목록)**: 입고완료 + 재고 1 초과 + 정상(NORMAL) 상태인 LOT만 표시합니다. 상단에 LOT번호·품목명 검색이 있습니다.
- **분할 버튼(행 좌측)**: 해당 LOT의 분할 입력 모달을 엽니다.
- **분할 모달**: 분할수량을 입력하면 두 조각으로 나뉜 미리보기가 표시됩니다.
- **라벨 미리보기 모달**: 분할 성공 후 새 시리얼 2건의 라벨을 출력합니다.

---

## ① 분할 가능 LOT 그리드 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **(액션) 분할** | 해당 LOT의 분할 모달을 여는 버튼입니다. 입고완료·재고 1 초과·정상 상태 LOT만 목록에 있으므로 표시된 행은 모두 분할 가능합니다. |
| **자재시리얼(matUid)** | 자재 LOT의 고유 식별번호(시리얼)입니다. 라벨 바코드로 인쇄되며, 분할의 대상이 되는 원본 시리얼입니다. |
| **품목코드(itemCode)** | 자재의 MES 내부 품목코드입니다. |
| **품목명(itemName)** | 자재 품목 이름입니다. |
| **현재수량(qty)** | 이 시리얼이 현재 보유한 재고 수량입니다. 단위와 함께 표시되며, 분할수량은 이 값보다 작아야 합니다. |
| **공급처(vendor)** | 이 자재를 납품한 거래처(공급업체) 이름입니다. 코드만 있으면 코드가 그대로 표시됩니다. |

> 목록 조건: **입고완료**(입고처리까지 끝난 LOT) + **현재수량 2 이상** + **예약수량 0** + **상태 NORMAL**. 출고 이력이 있거나 보류(HOLD) 상태인 LOT은 목록에 나오지 않습니다.

---

## ② 분할 모달(분할 버튼 클릭 시)

선택한 LOT을 두 조각으로 나누는 입력 화면입니다.

| 입력 항목 | 역할 / 의미 |
|------|------|
| **원본 정보(상단 요약)** | 선택한 자재시리얼 / 품목코드 / 품목명 / 현재수량을 요약 표시합니다(읽기 전용). |
| **분할수량 \*** | 원본에서 떼어낼 수량입니다. **1 이상, 현재수량 미만**이어야 합니다. 나머지 수량은 자동으로 잔량 조각이 됩니다. |
| **분할 미리보기** | 분할수량을 입력하면 `{분할수량} + {잔량}` 형태로 두 조각이 어떻게 나뉘는지 미리 보여줍니다. |
| **비고** | 분할 사유나 메모를 자유롭게 입력합니다(선택, 200자 이내). |

저장하면 원본 시리얼이 폐기되고 새 시리얼 2건이 만들어진 뒤 **라벨 미리보기 모달**이 열립니다.

---

## ③ 라벨 미리보기 모달(분할 성공 후)

분할로 새로 발번된 시리얼 2건의 라벨을 미리 보고 출력합니다.

| 항목 | 역할 / 의미 |
|------|------|
| **라벨 미리보기 영역** | 새로 만들어진 두 조각 시리얼의 라벨이 나열됩니다. 각 라벨에는 새 자재 시리얼, 품목코드, 수량 등이 인쇄됩니다. |
| **제조사 표시** | 두 조각 모두 원본 LOT의 제조사를 그대로 계승해 표시·인쇄됩니다. |
| **라벨 출력(프린트 버튼)** | 로컬 Print Agent를 통해 라벨 프린터로 전송합니다. |

---

## 사용 순서

1. 상단 검색에서 자재시리얼(LOT번호) 또는 품목명으로 분할할 LOT을 찾습니다.
2. 해당 행의 **분할** 버튼을 클릭합니다.
3. 모달에서 **분할수량**을 입력합니다(현재수량보다 작게).
4. 미리보기로 `분할수량 + 잔량`이 의도한 대로 나뉘는지 확인합니다.
5. 필요하면 **비고**를 입력하고 **분할**을 클릭합니다.
6. 라벨 미리보기에서 새 시리얼 2건의 라벨을 확인하고 **라벨 출력**합니다.
7. 출력한 라벨을 각 자재 묶음에 부착합니다(원본 라벨은 폐기).

---

## 입력 규칙 / 검증

- 분할수량은 **1 이상, 현재수량 미만**이어야 합니다. 현재수량과 같거나 크면 분할 버튼이 비활성화됩니다.
- 다음 LOT은 분할할 수 없습니다(목록에 나오지 않거나 저장 시 차단):
  - **입고완료 전** LOT (입고처리를 먼저 진행해야 함)
  - **상태가 NORMAL이 아닌** LOT (HOLD/SPLIT/MERGED/DEPLETED 등)
  - **예약수량이 있는** LOT (예약을 먼저 정리)
  - **출고 이력이 있는** LOT (출고를 먼저 정리)
  - **품목마스터에서 분할 불가(isSplittable = N)로 설정된** 품목
- 분할은 되돌릴 수 없습니다. 원본 시리얼은 폐기되고 새 시리얼 2건으로 대체됩니다.

## 자주 묻는 질문

- **Q.** 분할하면 원본 시리얼은 어떻게 되나요?
  **A.** 원본은 폐기(SPLIT) 처리되어 재고가 0이 되고 더 이상 사용할 수 없습니다. 대신 분할수량 조각과 잔량 조각이 각각 **새 시리얼**로 발번됩니다.

- **Q.** 셋 이상으로 나눌 수 있나요?
  **A.** 한 번의 분할은 두 조각까지입니다. 더 잘게 나누려면, 분할 결과로 나온 새 시리얼을 다시 분할하면 됩니다(재분할 가능).

- **Q.** 분할한 자재의 입고·검사 이력은 추적되나요?
  **A.** 네. 새 조각들은 원본의 입하번호·제조사·최초시리얼(origin) 정보를 계승하므로 거슬러 추적할 수 있습니다.

- **Q.** 목록에 LOT이 안 보여요.
  **A.** 입고완료 전이거나, 현재수량이 1 이하이거나, 예약/출고 이력이 있거나, 상태가 NORMAL이 아닌 경우입니다. [원자재 LOT현황]에서 상태를 확인하세요.

- **Q.** 라벨이 출력되지 않아요.
  **A.** PC에 Print Agent가 실행 중인지 확인하세요.
