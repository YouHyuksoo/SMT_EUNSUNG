# 생산월력관리

## 화면 목적
연도별 근무일정을 편성하는 화면입니다. 회사 전체에 적용되는 **전사 월력**과, 특정 생산라인만 다르게 운영할 때 쓰는 **라인 예외 월력**을 함께 관리합니다. 여기서 확정한 근무일·근무시간이 생산계획의 가용시간 기준이 됩니다. 같은 화면에서 근무시간 계산의 기준이 되는 **교대시간(주간/야간)** 마스터도 관리합니다.

## 화면 구성
상단에 **월력 / 교대시간** 두 개의 탭이 있습니다.

**[월력] 탭**
- **왼쪽(연도·라인·요약)**: 연도 입력 + 라인 선택. 라인을 비워두면(전사) 전사 월력을 편집하고, 특정 라인을 선택하면 그 라인만의 예외 월력을 편집합니다. 아래에 선택한 연도의 **가동일수·비가동일수·반일·특근·총가용시간** 요약이 표시됩니다.
- **오른쪽(월 그리드)**: 월 단위 달력. 날짜를 클릭하면 **일자 편집** 창이 열립니다. 셀 색상은 근무유형(근무=파랑, 휴무=빨강, 반일=노랑, 특근=초록)으로 구분되고, 라인을 선택한 상태에서 그 라인만의 예외 일자에는 **예외** 배지가 붙습니다.
- 헤더 우측 버튼: **연간 생성**, **전사에서 복사**(라인을 선택했을 때만 표시), **확정**, **확정취소**.

**[교대시간] 탭**
- 적용기간(시작일~종료일)별 주간·야간 근무시간 목록. 등록/수정/삭제.

> 라인을 선택하지 않으면 **전사** 월력을 보고 편집합니다. 라인을 선택하면 그 라인의 **예외** 일자만 보고 편집하며, 예외가 없는 날짜는 전사 월력 값이 그대로 표시됩니다.

> 일자가 **확정**되면 그 달력 셀에 자물쇠 아이콘이 붙고 클릭해도 편집 창이 열리지 않습니다. 저장·연간 생성·전사에서 복사도 모두 막힙니다. 다시 편집하려면 먼저 **확정취소**해야 합니다.

## ① 일자 편집 (날짜 클릭 시)

| 필드 | 역할 / 의미 |
|------|------|
| **근무유형(dayType)** | 그날의 가동 구분입니다. 근무(WORK)·휴무(OFF)·반일(HALF)·특근(SPECIAL) 중 선택합니다. |
| **휴무사유(offReason)** | 근무유형이 **휴무(OFF)** 일 때만 표시됩니다. 주말/공휴일/공장휴무/라인정지 중에서 선택합니다. |
| **근무시간(workMinutes)** | 그날의 정규 근무시간(분)입니다. **비워두고 저장하면 교대시간 마스터를 기준으로 자동 계산됩니다**(아래 "근무시간 자동 계산" 참고). 값을 직접 입력하면 그 값이 그대로 저장됩니다(override). |
| **잔업시간(otMinutes)** | 그날의 잔업(연장근로) 시간(분)입니다. |
| **비고(comment)** | 해당 일자 메모입니다. |

> 근무유형을 바꾸면 근무시간 입력칸이 비워집니다. 이전 근무유형 기준으로 넣었던 값이 새 근무유형에는 맞지 않을 수 있기 때문입니다. 그대로 저장하면 새 근무유형 기준으로 자동 계산됩니다.

### 근무시간 자동 계산
근무시간을 비워두고 저장하면 서버가 **교대시간 마스터**를 기준으로 아래처럼 계산해 채웁니다.

| 근무유형 | 근무시간 |
|------|------|
| 휴무(OFF) | 0 |
| 근무(WORK) | 주간 순근무시간 + 야간 순근무시간 |
| 반일(HALF) | 주간 순근무시간 ÷ 2 |
| 특근(SPECIAL) | 주간 순근무시간 |

## ② 교대시간 (교대시간 탭)

| 필드 | 역할 / 의미 |
|------|------|
| **적용시작일(dateset)** | 이 교대시간이 적용되는 시작일입니다. |
| **적용종료일(dateend)** | 적용 종료일입니다. 비워두면 **무기한**(다음 교대시간이 등록되기 전까지 계속 적용)입니다. |
| **주간 시작/종료** | 주간 근무 시작·종료 시각(HH:MM)입니다. |
| **주간 휴식(분)** | 주간 근무 중 휴게시간(분)입니다. |
| **야간 시작/종료** | 야간 근무 시작·종료 시각입니다. **야간은 자정을 넘길 수 있습니다**(예: 20:00 ~ 08:00). |
| **야간 휴식(분)** | 야간 근무 중 휴게시간(분)입니다. |
| **주간 순근무(분) / 야간 순근무(분)** | 시작·종료·휴식을 기준으로 자동 계산되어 목록에 표시됩니다(직접 입력하지 않습니다). |

> 같은 기간이 겹치는 교대시간은 등록/수정할 수 없습니다. 새 교대시간을 등록하려면 기존 항목의 적용종료일을 먼저 채우거나, 그 이후 날짜로 시작일을 잡으세요.

## 사용 순서
1. **교대시간** 탭에서 주간·야간 근무시간을 먼저 등록합니다(적용시작일·시작/종료·휴식). 이 값이 있어야 연간 생성 시 근무시간이 자동으로 채워집니다.
2. **월력** 탭에서 연도를 입력하고 라인을 **전사**(빈 값)로 둔 채 **연간 생성**을 실행합니다. 토요일/일요일 근무 여부를 체크하고, 양력 고정공휴일(1/1, 3/1, 5/5, 6/6, 8/15, 10/3, 10/9, 12/25)은 자동으로 휴무 처리됩니다.
3. 설·추석·대체공휴일 등 음력 기준 휴일은 자동 반영되지 않으므로, 해당 날짜를 달력에서 클릭해 휴무로 직접 수정합니다.
4. 특정 라인만 다르게 운영해야 하면 좌측에서 라인을 선택하고, **전사에서 복사**로 전사 월력을 가져온 뒤 필요한 날짜만 예외로 수정합니다.
5. 편성이 끝나면 **확정**으로 잠급니다. 수정이 필요하면 **확정취소** 후 다시 편집합니다.

## 입력 규칙
- **확정된 일자**는 셀 클릭·일자 저장·연간 생성·전사에서 복사가 모두 막힙니다. 먼저 확정취소하세요.
- **연간 생성**과 **전사에서 복사**는 해당 연도의 기존 일자를 모두 **덮어씁니다**. 손으로 고친 예외가 있으면 실행 전 확인하세요.
- **연간 생성**과 **전사에서 복사**는 대상 범위(연도, 라인)에 확정된 일자가 하나라도 있으면 거부됩니다.
- 휴무사유는 근무유형이 **휴무(OFF)** 일 때만 저장됩니다. 다른 근무유형으로 바꾸면 비워집니다.
- **전사에서 복사**는 라인을 선택했을 때만 실행할 수 있습니다.
- 라인을 선택한 상태에서 예외가 없는 날짜는 전사 월력 값을 그대로 보여줍니다(수정하면 그 라인만의 예외로 새로 만들어집니다).

## 자주 묻는 질문
- **Q.** 라인 예외를 지우려면 어떻게 하나요?
  **A.** 화면에서 예외를 개별 삭제하는 기능은 없습니다. 다시 **전사에서 복사**를 실행하면 그 라인의 해당 연도 예외가 전사 값으로 덮어써집니다.
- **Q.** 근무시간이 0으로 나옵니다.
  **A.** 교대시간 마스터가 등록되어 있지 않거나, 해당 날짜에 적용되는 교대시간이 없기 때문입니다. 교대시간 탭에서 먼저 등록하세요.
- **Q.** 연간 생성을 다시 누르면 손으로 고친 휴무가 사라지나요?
  **A.** 예. 연간 생성은 기존 일자를 전부 덮어씁니다. 보정은 생성 이후에 하세요.
- **Q.** 설날/추석이 근무일로 표시됩니다.
  **A.** 연간 생성은 양력 고정공휴일만 자동으로 휴무 처리합니다. 음력 기준 공휴일과 대체공휴일은 담당자가 달력에서 직접 휴무로 수정해야 합니다.
- **Q.** 확정했는데 수정해야 합니다.
  **A.** **확정취소**로 잠금을 푼 뒤 수정하고, 다시 확정하세요.
- **Q.** 라인 선택 목록이 비어 있습니다.
  **A.** 라인 마스터가 등록되어 있지 않은 것입니다. 라인관리 화면에서 라인을 먼저 등록하세요.
