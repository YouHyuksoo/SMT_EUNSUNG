# 생산월력관리

## 화면 목적
연간 작업일정(근무 캘린더)을 등록하고 **날짜마다 근무/휴무 여부, 교대패턴, 가용시간(분), 잔업시간**을 편성하는 화면입니다. 여기서 확정한 가동일·가용시간이 생산계획·CAPA 산정의 기준이 됩니다. 공장 공통 캘린더뿐 아니라 **공정별 전용 캘린더**도 만들 수 있고, 근무 시간대(주간·야간 등)를 정의하는 **교대패턴**도 같은 화면에서 관리합니다.

## 화면 구성
상단에 **월력관리 / 교대패턴** 두 개의 탭이 있습니다.

**[월력관리] 탭**
- **왼쪽(목록)**: 연도 필터 + 캘린더 목록. 각 항목에 연도 배지·캘린더ID·공정명·상태(DRAFT/CONFIRMED)·삭제 버튼이 표시됩니다. 클릭하면 선택됩니다.
- **오른쪽 상단(정보 패널)**: 선택한 캘린더의 캘린더ID·연도·공정·기본교대수·상태와 **저장 / 확정(또는 확정취소)** 버튼.
- **오른쪽 하단(월별 달력)**: 월 단위 달력. 날짜를 클릭하면 **일자 편집** 모달이 열립니다. 셀 색상은 근무유형(근무=파랑, 휴무=빨강, 반일=노랑, 특근=초록)으로 구분되고, 하단 요약바에 **가동일수·비가동일수·총가용시간**이 표시됩니다.
- 헤더 우측: **연간 생성**, **다른 캘린더에서 복사**, **추가** 버튼(확정 상태에서는 생성/복사 버튼이 숨겨짐).

**[교대패턴] 탭**
- 교대 패턴 목록 테이블(교대코드·교대명·시작/종료시간·휴식·근무분)과 추가/수정/삭제.

> 상태 흐름: **DRAFT(작성중)** 에서 일정을 편성하고 **CONFIRMED(확정)** 로 잠급니다. 확정 상태에서는 캘린더 정보·일자 수정·연간생성·복사·삭제가 모두 차단됩니다(달력 셀도 클릭 불가). 다시 고치려면 **확정취소**로 DRAFT로 되돌립니다.

## ① 캘린더 정보 (헤더)

| 컬럼 | 역할 / 의미 |
|------|------|
| **캘린더ID(calendarId)** | 캘린더를 식별하는 고유 ID입니다(예: `WC-2026-PLANT01`). 생성 시 직접 입력하며, 일자 데이터가 이 ID로 묶입니다. 등록 후에는 변경하지 않습니다. |
| **연도(calendarYear)** | 이 캘린더가 적용되는 4자리 연도입니다(YYYY). 연간 생성 시 이 연도의 1/1~12/31 일자가 만들어집니다. |
| **공정(processCd)** | 이 캘린더를 적용할 공정입니다. 비워두면 **전체(공장기본)** 캘린더가 되어 공정 전용 캘린더가 없는 공정에 공통 적용됩니다. 회사·공장·연도·공정 조합은 중복될 수 없습니다. |
| **기본교대수(defaultShiftCount)** | 근무일에 기본으로 적용할 교대 횟수입니다(1~3). 연간 생성·일자 기본값으로 쓰입니다. |
| **기본교대패턴(defaultShifts)** | 근무일에 기본 적용할 교대패턴 코드 목록입니다(CSV, 예: `DAY,NIGHT`). 연간 생성 시 근무일의 교대와 가용시간 계산 기준이 됩니다. |
| **상태(status)** | `DRAFT`=작성중(수정 가능), `CONFIRMED`=확정(수정 잠금). |
| **비고(remark)** | 캘린더 관리 참고 메모입니다. |

## ② 일자 편집 (날짜 클릭 시)

| 컬럼 | 역할 / 의미 |
|------|------|
| **근무유형(dayType)** | 그날의 가동 구분입니다. 근무(WORK)·휴무(OFF)·반일(HALF)·특근(SPECIAL). 공통코드 `WORK_DAY_TYPE`. 휴무 외 유형(근무·반일·특근)은 가동일수로 집계됩니다. |
| **휴무사유(offReason)** | 근무유형이 **휴무(OFF)** 일 때만 표시·선택합니다. 공통코드 `DAY_OFF_TYPE`(예: 주말 WEEKEND, 공휴일 HOLIDAY 등). 연간 생성 시 주말·공휴일에 자동 부여됩니다. |
| **교대수(shiftCount)** | 그날의 교대 횟수입니다(1~3). 비우면 캘린더 기본교대수를 따릅니다. |
| **교대패턴(shifts)** | 그날 가동할 교대를 사용 중인 교대패턴 목록에서 **다중 선택**합니다(버튼 토글). 선택값은 CSV로 저장됩니다. |
| **잔업시간(otMinutes)** | 그날의 잔업(연장근로) 시간을 **분** 단위로 입력합니다. 가용시간 합계에 더해집니다. |
| **가용시간(workMinutes)** | 그날의 정규 가용 근무시간(분)입니다. 연간 생성 시 기본교대패턴의 근무분 합으로 자동 산출되며, 휴무일은 0입니다. |
| **비고(remark)** | 해당 일자 메모입니다. |

## ③ 교대패턴 (교대패턴 탭)

| 컬럼 | 역할 / 의미 |
|------|------|
| **교대코드(shiftCode)** | 교대를 식별하는 코드입니다(예: `DAY`, `NIGHT`). 캘린더의 기본교대패턴·일자 교대패턴이 이 코드를 참조합니다. 수정 시에는 변경 불가. |
| **교대명(shiftName)** | 화면 표시용 교대 이름입니다(예: 주간, 야간). |
| **시작시간(startTime)** | 교대 시작 시각입니다(HH:MM, 예: `08:00`). |
| **종료시간(endTime)** | 교대 종료 시각입니다(HH:MM, 예: `17:00`). |
| **휴식(분)(breakMinutes)** | 교대 중 휴게 시간(분)입니다(기본 60). |
| **근무(분)(workMinutes)** | 휴게를 제외한 실 근무 시간(분)입니다. 캘린더 연간 생성 시 근무일의 가용시간을 이 값들의 합으로 계산합니다. |

> 교대수와 교대패턴은 별개입니다. **교대수**는 "하루에 몇 교대를 돌리나(1~3)"이고, **교대패턴**은 "어떤 시간대 교대(주간/야간 등)를 적용하나"입니다.

## 사용 순서
1. (최초 1회) **교대패턴** 탭에서 주간·야간 등 교대를 등록합니다(코드·시간·근무분).
2. **월력관리** 탭에서 **추가**를 눌러 캘린더ID·연도·공정·기본교대수·기본교대패턴을 입력해 캘린더를 생성합니다.
3. 목록에서 캘린더를 선택하고 **연간 생성**으로 1년치 일자를 자동 편성합니다. 모달에서 **토요일 근무 / 일요일 근무** 여부를 체크합니다(미체크 시 휴무, 공휴일은 자동 휴무).
4. 달력에서 날짜를 클릭해 예외(특정 휴무·특근·반일·교대 변경·잔업)를 보정합니다.
5. 기존 캘린더가 있으면 **다른 캘린더에서 복사**로 일정을 가져와 시작할 수 있습니다(기존 일자는 덮어씀).
6. 편성이 끝나면 **확정**으로 잠급니다. 수정이 필요하면 **확정취소** 후 다시 편집합니다.

## 입력 규칙·검증
- **확정(CONFIRMED) 캘린더**는 정보 저장·일자 편집·연간 생성·복사·삭제가 모두 차단됩니다. 먼저 확정취소하세요.
- **회사·공장·연도·공정** 조합이 중복되면 생성할 수 없습니다(공장기본 캘린더는 공정이 비어 있는 1건).
- **연간 생성**과 **복사**는 해당 캘린더의 기존 일자를 모두 **삭제 후 재작성(덮어쓰기)** 합니다. 보정해 둔 일자가 사라지므로 실행 전 확인하세요.
- **복사**는 원본 캘린더에 일자 데이터가 있어야 합니다(없으면 오류).
- 교대수·일자 교대수는 **1~3** 범위입니다.
- 휴무사유는 근무유형이 **휴무(OFF)** 일 때만 저장됩니다(다른 유형이면 비워짐).
- 캘린더 **삭제**는 하위 일자 데이터까지 함께 삭제됩니다.

## 자주 묻는 질문
- **Q.** 공정을 비우면 어떻게 되나요?
  **A.** 공장 공통(공장기본) 캘린더가 됩니다. 공정 전용 캘린더가 없는 공정에 공통으로 적용됩니다.
- **Q.** 연간 생성을 다시 누르면 손으로 고친 휴무가 사라지나요?
  **A.** 예. 연간 생성은 기존 일자를 전부 덮어씁니다. 보정은 생성 이후에 하세요.
- **Q.** 가용시간(분)은 어떻게 채워지나요?
  **A.** 연간 생성 시 캘린더의 **기본교대패턴**에 속한 교대들의 **근무(분)** 합으로 근무일에 자동 입력됩니다. 휴무일은 0입니다. 날짜별로 직접 수정할 수도 있습니다.
- **Q.** 확정했는데 수정해야 합니다.
  **A.** **확정취소**로 DRAFT로 되돌린 뒤 수정하고, 다시 확정하세요.
- **Q.** 교대수와 교대패턴 차이는요?
  **A.** 교대수는 하루 교대 횟수(1~3), 교대패턴은 적용할 시간대 교대(주간/야간 등)입니다.
