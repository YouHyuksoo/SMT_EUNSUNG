# 단자검사결과

## 화면 목적
단자 압착·삽입 검사 결과를 조회합니다.
## 화면 구성
- **조회 조건**: 기간, 상태, 품목, 바코드, 담당자 등으로 대상을 좁힙니다.
- **목록 그리드**: 처리 대상과 결과를 행 단위로 표시합니다.
- **상세/처리 영역**: 선택 행의 수량, 판정, 시리얼, 승인, 취소 또는 확정 상태를 확인합니다.

## 사용 순서
1. 조회 조건을 업무 기준에 맞춥니다.
2. 목록에서 품목, 바코드/문서번호, 수량, 상태를 확인합니다.
3. 처리 화면이면 대상 상태가 처리 가능한지 확인한 뒤 실행합니다.
4. 처리 후 재조회해 수량과 상태가 의도대로 바뀌었는지 확인합니다.

## 입력 규칙·검증
- 필수 식별값(품목, 바코드, 문서번호, 수량, 담당자)이 없으면 저장 또는 실행할 수 없습니다.
- 완료/취소/출하/폐기처럼 이미 후속 처리된 건은 중복 처리하지 않습니다.
- 수량은 화면별 기준 단위가 다르므로 품목 집계와 시리얼 상세를 분리해 확인합니다.

## 자주 묻는 질문
- **Q. 화면에 대상이 보이지 않습니다.**
  **A.** 기간, 상태, 품목유형, 회사/공장 스코프를 먼저 확인하세요.
- **Q. 처리 후 수량이 예상과 다릅니다.**
  **A.** 집계 테이블과 시리얼/라벨 상세 테이블을 같이 봐야 합니다.
