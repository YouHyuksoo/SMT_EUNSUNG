# 실적입력(키오스크)

## 화면 목적
현장 설비 옆 **태블릿/PC**에서 작업자가 직접 생산실적을 입력하는 통합 화면입니다. 작업지시 선택부터 자재 투입, 작업지도서 확인, 생산수량/불량 입력, 자주검사까지 한 화면에서 처리합니다. (전체화면(chromeless) 모드로 사용)

## 화면 구성
```
┌──────────────────────────────────────────────────────────┐
│ 상단 헤더 2단: ①설비ID·바코드·설비일일검사  ②작업지시·작업자·생산실적·작업자설비검사 │
├───────────────┬──────────────────────────┬───────────────┤
│ 좌: BOM 자재  │ 중앙: 작업지도서          │ 우: 양품조건  │
│   + 소모성    │   하단 3칸:               │   + 작업이력  │
│   설비부품    │   자주검사 | 불량 | 실적입력 │               │
└───────────────┴──────────────────────────┴───────────────┘
```

## ① 상단 헤더 (선택·점검)
| 항목 | 역할 / 의미 |
|------|------|
| **설비ID/바코드** | 작업할 설비를 선택합니다. 설비 기준으로 작업지시·공정·자재가 연결됩니다. |
| **설비 일일검사** | 설비의 일상점검을 완료해야 작업을 진행할 수 있습니다(준비 점검=인터록). |
| **작업지시** | 이 설비에서 수행할 작업지시(생산 대상·수량)를 선택합니다. |
| **작업자** | 실적을 등록할 작업자를 선택합니다(복수 가능). |
| **작업자 설비검사** | 작업자가 수행하는 설비 점검입니다(인터록 대상). |
| **생산실적(진행률)** | 작업지시 대비 현재까지 등록된 실적/진행률을 표시합니다. |

> **인터록(준비 점검)**: 설비 일일검사·작업자 설비검사 등 필수 점검을 끝내야 실적 저장이 열립니다. 미완 시 진행이 막힙니다.

## ② 좌측 — 자재 투입
| 항목 | 역할 / 의미 |
|------|------|
| **BOM 자재리스트** | 선택한 작업지시(제품)의 BOM 기준 투입 자재 목록입니다. 필요량 대비 투입량을 확인합니다. |
| **자재 스캔** | 투입할 자재 시리얼(matUid)을 스캔해 투입 처리합니다. |
| **소모성 설비부품** | 설비에 장착되는 소모품(제품 BOM과 별개)입니다. 하단 섹션에서 스캔 투입합니다. |

## ③ 중앙 — 작업지도서 + 실적/검사
| 항목 | 역할 / 의미 |
|------|------|
| **작업지도서** | 품목·공정별 작업 지침(이미지/PDF/문서)을 표시합니다. 클릭하면 확대됩니다. |
| **실적입력** | **양품수량(goodQty)**·**불량수량(defectQty)**·(필요 시 생산 시리얼)을 입력해 실적을 저장합니다. |
| **불량** | 불량을 **등급별(치명/중/경)**로 상세 입력합니다. 입력한 불량 합계가 실적의 불량수량으로 함께 저장됩니다. |
| **자주검사** | 작업자 self 검사입니다(아래 흐름 참조). |

## ④ 우측 — 양품조건 · 작업이력
| 항목 | 역할 / 의미 |
|------|------|
| **양품조건** | 양품으로 인정되는 기준 안내입니다. |
| **작업이력** | 이 작업에서 등록된 실적 이력입니다. |

## 자주검사 흐름 (초물·중물·종물)
| 시점 | 트리거 | 의미 |
|------|------|------|
| **초물(FIRST)** | 첫 실적 저장 직후 **자동 오픈** | 생산 초기 품질 확인 |
| **중물(MID)** | 패널 버튼 클릭 또는 **진행률 60%에서 차단** | 진행 중 품질 확인 |
| **종물(LAST)** | 패널 버튼 클릭 | 생산 종료 시 확인 |

## 사용 순서
1. **설비 선택** → 설비 일일검사 완료.
2. **작업지시·작업자 선택** → 작업자 설비검사 완료(인터록 해제).
3. 좌측에서 **자재/소모품 스캔 투입**.
4. 중앙 작업지도서 확인하며 작업.
5. **양품수량·불량수량 입력 후 저장**(불량은 등급별 상세 입력).
6. 시점에 맞춰 **자주검사**(초물/중물/종물) 수행.

## 자주 묻는 질문
- **Q.** 실적 저장 버튼이 안 눌려요.
  **A.** 준비 점검(설비/작업자 검사)이 완료되지 않았거나 작업지시·작업자 미선택일 수 있습니다. 인터록 상태를 확인하세요.
- **Q.** 중물 검사를 안 했는데 더 진행이 안 돼요.
  **A.** 진행률 60%에서 중물 자주검사가 차단됩니다. 자주검사를 수행하면 풀립니다.
- **Q.** 불량수량을 직접 입력했는데 등급별 입력과 중복되나요?
  **A.** 등급별 상세 불량 합계가 있으면 그 값이 우선 적용됩니다(이중 집계되지 않음).
