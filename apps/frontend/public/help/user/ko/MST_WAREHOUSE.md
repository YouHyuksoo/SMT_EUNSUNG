# 창고/로케이션 관리

## 화면 목적
재고가 보관·이동되는 **창고**와 창고 안의 **세부 적재위치(로케이션)**를 등록·관리합니다. 여기 등록된 창고/로케이션이 입고·불출·재고·생산실적·재고이동의 기준이 됩니다.

## 화면 구성
상단 탭으로 2개 영역이 나뉩니다. 우상단 버튼(새로고침/등록)은 선택한 탭에 맞춰 바뀝니다.

- **창고관리 탭**: 창고 마스터 목록(좌측 그리드) + 검색·창고유형 필터. 행 ✏️(수정)/🗑️(삭제), 우상단 **신규 등록**. 클릭 시 등록/수정 모달.
- **로케이션 탭**: 창고 내 세부위치 목록 + 창고 필터. 로케이션 등록/수정/삭제.

두 영역의 관계:
```
창고(WAREHOUSES)
  └─ 로케이션(WAREHOUSE_LOCATIONS)  : 한 창고 안에 여러 세부위치
```

---

## ① 창고관리 탭 — 컬럼 / 폼 필드

| 컬럼 / 필드 | 역할 / 의미 |
|------|------|
| **창고코드(warehouseCode)** | 창고를 식별하는 **고유 코드**입니다. 등록 후에는 변경할 수 없습니다(재고·실적이 이 코드로 연결됨). |
| **창고명(warehouseName)** | 현장에서 식별할 창고 이름입니다. |
| **창고유형(warehouseType)** | 창고의 용도 분류입니다. 입고·재고·생산 처리 방식이 이 값에 따라 달라집니다. 값: 원자재 창고(RAW) · 반제품 창고(WIP) · 완제품 창고(FG) · 공정재공(FLOOR) · 불량 창고(DEFECT) · 폐기 창고(SCRAP) · 외주처(SUBCON). |
| **라인(lineCode)** | **공정재공(FLOOR)** 창고가 소속된 생산라인입니다. 유형이 FLOOR일 때만 입력합니다. |
| **공정(processCode)** | **공정재공(FLOOR)** 창고가 소속된 공정입니다. 유형이 FLOOR일 때만 입력합니다. |
| **기본(isDefault)** | 동일 유형에서 기본으로 사용할 창고인지 여부입니다. 자동 입고/적치 시 유형별 기본창고가 우선 선택됩니다. |
| **사용(useYn)** | `Y`일 때만 조회·선택·사용 대상입니다. 사용 중지는 `N`. |

> 창고유형이 **공정재공(FLOOR)**일 때만 라인·공정 입력칸이 나타납니다. 다른 유형은 라인/공정을 쓰지 않습니다.

## ② 로케이션 탭 — 컬럼 / 폼 필드

창고 1개에 여러 로케이션(bin/rack/shelf 등 세부위치)을 둘 수 있습니다.

| 컬럼 / 필드 | 역할 / 의미 |
|------|------|
| **창고(warehouseCode)** | 이 로케이션이 속한 창고입니다. 등록 후에는 변경할 수 없습니다. |
| **로케이션코드(locationCode)** | 창고 내 세부위치를 식별하는 코드입니다. 등록 후에는 변경할 수 없습니다(창고+로케이션 코드 조합이 유일). |
| **로케이션명(locationName)** | 현장에서 식별할 로케이션 이름입니다. |
| **존(구역)(zone)** | 창고 내 구역(zone) 구분 값입니다. |
| **행(rowNo)** | 랙/선반의 행(row) 번호입니다. |
| **열(colNo)** | 랙/선반의 열(column) 번호입니다. |
| **단(층)(levelNo)** | 랙/선반의 단(level) 번호입니다. |
| **사용(useYn)** | `Y`일 때만 사용 대상입니다(목록에 ● 초록 점으로 표시). |
| **비고(remark)** | 로케이션 관리 참고사항입니다. |

---

## 사용 순서
1. **창고관리 탭**에서 우상단 **신규 등록**으로 창고를 만듭니다(창고코드·창고명·창고유형 필수, FLOOR면 라인·공정).
2. 필요하면 **로케이션 탭**에서 해당 창고에 세부위치(존/행/열/단)를 등록합니다.
3. 유형별로 자동 적치에 쓸 창고는 **기본** 체크로 지정합니다.

## 입력 규칙 / 검증
- 창고코드·로케이션코드는 등록 후 변경 불가(수정 시 코드칸 비활성).
- 같은 창고코드, 같은 (창고+로케이션) 조합은 중복 등록할 수 없습니다.
- **재고가 남아 있는 창고는 삭제할 수 없습니다.**

## 자주 묻는 질문
- **Q.** 라인/공정 칸이 안 보여요.
  **A.** 창고유형을 **공정재공(FLOOR)**으로 선택해야 라인·공정 입력칸이 나타납니다.
- **Q.** 창고를 삭제하려는데 안 됩니다.
  **A.** 해당 창고에 재고가 남아 있으면 삭제가 거부됩니다. 재고를 비운 뒤 다시 시도하거나 사용여부를 `N`으로 바꿔 비활성화하세요.
