# 불량코드관리

## 화면 목적
검사와 생산실적에서 불량을 기록할 때 고르는 **불량코드**를 등록·관리합니다. 불량코드는 곧바로 쓰지 않고, 먼저 **3레벨 분류**(검사단계 → 모델구분 → 불량유형) 아래에 소속시킨 뒤 **불량등급**과 **적용범위**를 붙여 관리합니다.

> 분류 3레벨이란: **1레벨 = 검사단계**(IQC 수입검사 / LQC 공정검사 / OQC 출하검사), **2레벨 = 모델구분**(저전압 LV / 고전압 HV), **3레벨 = 불량유형**(기능 / 외관 / 기타)입니다. 불량코드는 반드시 **3레벨 분류**에 연결됩니다.

## 화면 구성
- **왼쪽 — 등록된 불량 전체**: 모든 불량코드를 표로 보여줍니다. 행을 클릭하면 오른쪽에서 수정합니다. 상단 검색창으로 불량코드/불량명을 찾습니다.
- **오른쪽 위 — 불량코드 추가/수정**: 불량코드 한 건의 모든 항목을 입력합니다. 1·2·3레벨 분류를 차례로 선택해야 저장됩니다.
- **오른쪽 아래 — 분류 빠른 추가**: 새 분류(검사단계/모델구분/불량유형)를 즉석에서 만드는 영역입니다.

## 개념 관계
```
검사단계(1레벨) ──▶ 모델구분(2레벨) ──▶ 불량유형(3레벨) ──연결──▶ 불량코드 ──(등급/적용범위)
   IQC/LQC/OQC        저전압 LV/고전압 HV     기능/외관/기타          예: SOLDER-01
```
- 모델구분(LV/HV)은 불량코드의 **모델구분 적용 목록**에도 자동 반영됩니다. 즉 2레벨로 무엇을 골랐는지에 따라 그 불량코드가 어떤 모델에 쓰이는지가 정해집니다.

---

## ① 불량코드 목록 컬럼 (왼쪽 표)

| 컬럼 | 역할 / 의미 |
|------|------|
| **불량코드(defectCode)** | 불량을 식별하는 고유 코드입니다. 검사·실적 화면에서 이 코드로 불량을 고릅니다. 예: `SOLDER-01`. 등록 후에는 변경할 수 없습니다(연결 깨짐 방지). |
| **불량명(defectName)** | 사람이 읽는 불량 이름입니다. 예: `납땜 불량`. 검사 화면 목록에 함께 표시됩니다. |
| **1레벨** | 이 불량코드가 속한 **검사단계** 분류명입니다(IQC/LQC/OQC). 3레벨 분류를 따라 자동 표시됩니다. |
| **2레벨** | **모델구분** 분류명입니다(저전압/고전압). |
| **3레벨** | **불량유형** 분류명입니다(기능/외관/기타). 불량코드가 실제로 연결된 분류입니다. |
| **등급(defectGrade)** | 불량의 심각도입니다. **치명(CRITICAL)** = 안전·기능에 치명적, **중(MAJOR)** = 기능에 영향 큰 중결함, **경(MINOR)** = 외관 등 경미한 결함. 합·불 판정과 통계 집계 기준이 됩니다. |
| **적용범위(defectScope)** | 이 불량을 어디서 쓰는지입니다. **공통(COMMON)** = 어디서나, **원자재(RAW_MATERIAL)** = 수입검사 등 자재, **제품(PRODUCT)** = 완제품, **공정(PROCESS)** = 생산공정. |
| **사용여부(useYn)** | `사용(Y)`일 때만 검사·실적에서 고를 수 있습니다. `비활성(N)`이면 목록에서 제외됩니다. |

---

## ② 불량코드 추가/수정 필드 (오른쪽 위)

| 필드 | 역할 / 의미 |
|------|------|
| **불량코드(defectCode)** | 신규 등록 시 입력하는 고유 코드입니다. 자동으로 대문자로 변환됩니다. **수정 시에는 잠겨 변경 불가**합니다. |
| **불량명(defectName)** | 불량 이름입니다. 필수 입력입니다. |
| **1레벨(검사단계)** | 검사단계 분류를 고릅니다(IQC/LQC/OQC). 먼저 골라야 2레벨이 활성화됩니다. |
| **2레벨(모델구분)** | 1레벨 하위의 모델구분을 고릅니다(저전압/고전압). 여기서 고른 모델구분이 곧 이 불량코드의 **적용 모델**이 됩니다. |
| **3레벨(불량유형)** | 2레벨 하위의 불량유형을 고릅니다(기능/외관/기타). **불량코드는 이 3레벨 분류에 실제로 연결**되므로 반드시 선택해야 저장됩니다. |
| **등급(defectGrade)** | 치명 / 중 / 경 중 하나를 고릅니다. 기본값은 **중**입니다. |
| **적용범위(defectScope)** | 공통 / 원자재 / 제품 / 공정 중 하나를 고릅니다. 기본값은 **공통**입니다. |
| **사용여부(useYn)** | 사용/비활성을 선택합니다. |
| **설명(description)** | 불량에 대한 보충 설명·기준을 적는 메모입니다(선택). |

> 저장 조건: **불량코드 · 불량명 · 3레벨 분류**가 모두 채워져야 저장됩니다. 1·2·3레벨을 위에서부터 순서대로 골라야 합니다.

---

## ③ 분류 빠른 추가 필드 (오른쪽 아래)

새 검사단계·모델구분·불량유형 분류를 만들 때 사용합니다.

| 필드 | 역할 / 의미 |
|------|------|
| **분류 레벨(levelNo)** | 만들 분류의 레벨입니다. 1레벨(검사단계) / 2레벨(모델구분) / 3레벨(불량유형). |
| **상위 분류(parentCategoryCode)** | 2·3레벨 분류가 소속될 바로 위 분류입니다. **1레벨이면 비워둡니다**(상위 없음). 2레벨이면 1레벨을, 3레벨이면 2레벨을 고릅니다. |
| **분류코드(categoryCode)** | 분류 고유 코드입니다. 대문자로 변환됩니다. 예: `IQC`, `IQC_LV`, `IQC_LV_FUNCTION`. |
| **분류명(categoryName)** | 분류 이름입니다. 예: `IQC`, `저전압`, `기능`. |

> 분류는 레벨이 한 단계씩 정확히 맞아야 합니다. 예) 2레벨의 상위는 반드시 1레벨이어야 하며, 1레벨은 상위를 가질 수 없습니다.

---

## 사용 순서
1. (분류가 없으면) **분류 빠른 추가**로 1레벨(검사단계) → 2레벨(모델구분) → 3레벨(불량유형)을 차례로 만듭니다.
2. **불량코드 추가**에서 1·2·3레벨 분류를 선택하고 불량코드·불량명·등급·적용범위를 입력해 저장합니다.
3. 등록한 불량코드는 [불량관리]·검사·생산실적 키오스크의 불량 입력에서 선택됩니다.

## 입력 규칙 / 검증
- 불량코드·불량명·3레벨 분류는 필수입니다. 하나라도 비면 저장되지 않습니다.
- 불량코드는 등록 후 변경할 수 없습니다(수정 시 코드 입력칸이 잠깁니다).
- 불량코드는 **3레벨(불량유형) 분류에만** 연결됩니다. 1·2레벨에는 직접 연결할 수 없습니다.
- 사용중인(`사용`) 분류에만 불량코드를 연결할 수 있습니다.

## 자주 묻는 질문
- **Q.** 등급(치명/중/경)은 어떻게 정하나요?
  **A.** 안전·핵심 기능에 치명적이면 **치명**, 기능에 영향이 큰 결함이면 **중**, 외관 등 경미하면 **경**입니다. 등급은 검사 합·불 판정과 불량 통계의 기준이 됩니다.
- **Q.** 2레벨(모델구분)은 왜 고르나요?
  **A.** 같은 불량이라도 저전압/고전압 제품에 따라 관리가 다를 수 있어, 모델구분별로 불량코드를 나눠 등록·집계합니다. 고른 모델구분이 그 불량코드의 적용 모델이 됩니다.
- **Q.** 불량코드를 더 안 쓰면 삭제하나요?
  **A.** 삭제 대신 **사용여부를 비활성(N)**으로 두세요. 과거 불량이력과의 연결이 끊기지 않습니다.
