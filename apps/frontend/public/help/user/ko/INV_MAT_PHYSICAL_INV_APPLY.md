# 자재재고실사 반영

## 화면 목적
재고실사 세션 완료 후 PDA에서 스캔한 실사 수량을 검토하고, 필요한 경우 직접 수정하여 시스템 재고에 반영(Apply)합니다. 실사 결과로 발생한 차이를 재고 수량에 실제로 반영하는 최종 단계입니다.

## 화면 구성
- **상단 — 통계 카드 4개**: 전체 / 실사완료 / 일치 / 불일치 건수 요약
- **필터 영역**: 기준년월 + 창고 선택 + 검색어 입력
- **DataGrid 목록**: 품목별 시스템 수량, 실사 수량, 차이 표시. 실사수량 컬럼은 **직접 수정 가능**합니다.
- **실사반영 버튼**: 불일치 항목이 있을 때 활성화됩니다.

---

## ① DataGrid 목록 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **창고(warehouseName)** | 재고가 위치한 창고입니다. |
| **품목코드(itemCode)** | 실사 대상 품목 코드입니다. |
| **품목명(itemName)** | 실사 대상 품목 이름입니다. |
| **자재시리얼(matUid)** | LOT/시리얼 번호입니다. |
| **시스템수량(qty)** | 현재 시스템상의 재고 수량입니다. |
| **실사수량(countedQty)** | PDA 스캔된 실사 수량입니다. **직접 클릭하여 수정**할 수 있습니다. |
| **차이(diffQty)** | 실사수량 - 시스템수량. 양수(파랑) / 음수(빨강) / 0(초록). |
| **실사시간(countedAt)** | 마지막 PDA 스캔 시각입니다. |

---

## 사용 순서
1. 상단 필터에서 기준년월과 창고를 선택하여 실사 결과를 조회합니다.
2. 목록에서 각 품목의 **실사수량**을 PDA 스캔 결과와 비교합니다.
3. 필요 시 실사수량을 **직접 클릭하여 수정**합니다(오류 정정).
4. **실사반영** 버튼을 클릭하여 확인 모달에서 최종 확정합니다.
5. 반영 완료 후 [자재재고현황조회]에서 변경된 재고 수량을 확인합니다.

## 입력 규칙 / 검증
- 실사수량은 **0 이상**의 정수여야 합니다.
- 반영 전에 불일치 항목은 반드시 재확인하세요. 반영 후에는 원복이 어렵습니다.
