# 실적입력(조립)

## 화면 목적
완제품 작업지시에 대해 반제품(SG) 라벨과 설비에 장착된 원자재를 스캔으로 소모하여 완제품(FG) 라벨을 발행·확정하는 조립 실적 입력 화면입니다. 설비를 먼저 선택하면 공정이 자동으로 결정되고, 그 공정에 내려온 완제품 작업지시만 조회됩니다.

## 화면 구성
```
[헤더] 실적입력(조립)                              [초기화]
[상단 고정 바] 설비 선택 버튼  |  작업지시 스캔/검색
┌───────────────┬─────────────────────────┬───────────────┐
│ 좌: 설비 자재  │  중앙: 작업지도서        │ 우: 반제품 SG │
│ 장착(지속)     │  (선택 품목+공정 기준)   │ 스캔(세트·리셋)│
├───────────────┴─────────────────────────┴───────────────┤
│ 하단 액션바: [조립 실행 → FG 라벨 발행] | 발행 라벨 + 실물 스캔 확인 │
└────────────────────────────────────────────────────────┘
```
- 좌측 "설비 자재 장착"은 설비를 바꾸지 않는 한 계속 유지됩니다(작업지시가 바뀌어도 초기화되지 않음).
- 우측 "반제품 SG 스캔"은 작업지시를 바꾸거나 발행·확정할 때마다 초기화됩니다.

## ① 설비 자재 장착 목록 컬럼 (좌측 패널)
| 컬럼 | 역할 / 의미 |
|------|------|
| **품목명 · 품목코드** | 설비에 장착된 원자재의 품목 정보입니다. |
| **LOT(matUid)** | 장착된 자재 LOT/UID입니다. 자재 바코드를 스캔하면 이 값으로 장착됩니다. |
| **잔량** | 이 LOT에서 아직 소모되지 않은 가용 수량입니다. 0이 되면 "보충 스캔" 표시가 뜹니다. |
| **장착 대기(공정재고) 칩** | 자재 출고로 공정재고에는 있지만 아직 설비에 장착 안 된 LOT 목록입니다. 클릭하면 스캔 없이 바로 장착됩니다. |

> 이 화면은 설비 코드만으로 자재 패널을 띄우기 때문에 **작업지시 BOM과 매칭한 체크리스트는 표시되지 않습니다.** BOM 기준 미장착 품목 체크리스트가 필요하면 [서브공정 키팅](/production/subprocess-kitting) 화면을 이용하세요.

## ② 반제품(SG) 스캔 목록 컬럼 (우측 패널)
| 컬럼 | 역할 / 의미 |
|------|------|
| **#** | 스캔한 순서 번호입니다. |
| **SG 바코드** | 스캔한 반제품 묶음/회로 라벨 바코드입니다. 옆의 **묶음/회로** 배지는 라벨 종류(BUNDLE/SG)를 나타냅니다. |
| **품번** | 해당 SG 라벨의 품목코드입니다. |
| **잔량** | SG 라벨에 남은 가닥/수량입니다. |
| **상태** | SG 라벨의 현재 상태(IN_STOCK/MOUNTED 등)입니다. |
| **삭제(휴지통)** | 잘못 스캔한 SG를 목록에서 제거합니다(서버에는 반영 전이라 화면에서만 빠집니다). |

## 버튼별 기능
| 버튼 | 언제 사용 | 동작 | 결과 / 주의 |
|------|------|------|------|
| **설비 선택** (상단 콤보 버튼) | 화면 진입 후 가장 먼저 클릭 | 설비 선택 모달을 엽니다 | 설비를 고르면 공정이 자동 결정되고, 기존에 선택된 작업지시·스캔 목록·발행 상태가 모두 초기화됩니다 |
| **초기화** (헤더 우측) | 처음부터 다시 시작할 때 | 설비·작업지시·자재/SG 스캔 목록·발행 상태를 전부 리셋 | 진행 중이던 입력을 모두 잃으므로 다른 작업으로 전환할 때만 사용하세요 |
| **작업지시번호 스캔/입력** | 설비 선택 후 | 입력값으로 작업지시를 조회해 자동 선택 | 설비 미선택 시 비활성화, `FG`/`SG`로 시작하는 값은 라벨로 판단해 거부됩니다 |
| **검색** (작업지시 옆) | 작업지시번호를 모르거나 목록에서 고르고 싶을 때 | 작업지시 검색 모달을 엽니다(선택 설비 공정 + 완제품 조건으로 필터) | |
| **변경** (작업지시 선택 후) | 다른 작업지시로 바꿀 때 | 작업지시·SG 스캔 목록·발행 상태를 초기화 | 설비·자재 장착 목록은 유지됩니다 |
| **자재 바코드 스캔** (좌측 패널) | 원자재 LOT을 설비에 장착할 때 | 공정재고(장착 대기)에서 해당 LOT을 설비 재고로 이동 | 공정재고가 없으면(출고 미이행) 오류. 이 화면은 작업지시 BOM과 매칭하는 오장착 검증 없이 스캔한 LOT을 그대로 장착합니다 |
| **해제** (장착 자재 행) | 잘못 장착했거나 자재를 교체할 때 | 설비 재고를 공정재고(장착 대기)로 되돌립니다 | 예약 수량이 있으면 해제할 수 없습니다 |
| **SG 바코드 스캔** (우측 패널) | 반제품 라벨을 조립에 투입할 때 | 라벨을 검증 후 스캔 목록에 추가 | 이미 스캔한 라벨, 완제품(FG) 바코드, BOM에 없는 품목(오투입), 잔량 0, 사용 불가 상태는 모두 거부됩니다 |
| **삭제** (스캔 목록 행) | 잘못 스캔한 SG를 뺄 때 | 화면 목록에서만 제거 | |
| **조립 실행 → FG 라벨 발행** (하단, 크게 강조) | 작업지시·설비가 선택되고 SG를 1건 이상 스캔했을 때 활성화 | FG 바코드를 새로 채번해 `ISSUED` 상태로 저장 | 이 단계에서는 아직 SG 소비·자재 차감·재고·실적이 반영되지 않습니다 |
| **실물 FG 라벨 스캔** (발행 후 노출) | 프린터로 출력한 실물 라벨을 제품에 부착한 뒤 | 스캔값이 발행된 바코드와 일치하면 조립을 확정 | 값이 다르면 확정을 거부합니다 |
| **확인** (실물 스캔 옆) | 스캔 대신 버튼으로 확정을 제출할 때 | 입력창의 값으로 확정 요청 | |
| **발행 취소** (되돌리기 아이콘) | 잘못 발행했을 때 | 화면의 발행 상태만 초기화 | ⚠ 서버의 FG 라벨 레코드(`ISSUED`)는 취소되지 않고 그대로 남습니다. 같은 바코드는 화면에서 다시 확정할 수 없으니, 실물 라벨을 폐기하고 다음에 새로 발행하세요 |

## 업무 흐름
1. **설비 선택** — 조립 작업을 수행할 설비를 고릅니다. 설비가 공정을 자동으로 결정합니다.
2. **작업지시 선택** — 스캔/입력 또는 검색으로 완제품 작업지시를 선택합니다(선택 설비 공정 기준으로 조회).
3. **자재 장착 확인** — 좌측 패널에서 필요한 원자재 LOT을 스캔하거나 "장착 대기" 목록에서 선택해 설비에 장착합니다(이 화면은 BOM 매칭 체크리스트 없이 장착된 자재 목록만 보여줍니다. 실제 소요량 검증은 확정 시점에 서버가 수행합니다).
4. **작업지도서 확인** — 중앙 패널에서 선택 품목·공정 기준 작업지도서를 확인합니다.
5. **반제품(SG) 스캔** — 우측 패널에서 BOM의 반제품 구성만큼 SG 라벨을 스캔합니다.
6. **조립 실행 → FG 라벨 발행** 클릭 — FG 바코드가 발행됩니다(아직 재고·실적 미반영).
7. 출력된 실물 FG 라벨을 제품에 부착 후 **스캔하여 확정** — SG 소비, 설비 자재 차감, genealogy 생성, 생산실적 1건 등록, FG 재공재고(WIP) 적재가 한 번에 처리됩니다.
8. 확정 완료 후 SG 스캔 목록과 발행 상태가 초기화되며, 같은 작업지시로 다음 조립을 이어서 진행합니다.

## 입력 규칙 / 검증
- 설비를 먼저 선택해야 작업지시 조회와 자재 장착이 가능합니다(설비 → 공정 자동 결정).
- 완료(`DONE`)·취소(`CANCELED`)·홀딩(`HOLD`) 상태의 작업지시에는 FG 라벨을 발행하거나 확정할 수 없습니다.
- SG 라벨은 상태가 `IN_STOCK` 또는 `MOUNTED`이고 잔량이 1 이상이어야 스캔할 수 있습니다.
- 스캔한 SG의 품목이 선택 작업지시(완제품)의 BOM 반제품 구성에 없으면 "오투입"으로 거부됩니다.
- 자재 스캔 시 해당 LOT이 설비 공정의 공정재고(장착 대기)에 없으면(출고 미이행) 장착할 수 없습니다.
- 실물 FG 라벨 스캔값은 직전에 발행한 바코드와 정확히 일치해야 확정됩니다.

## 자주 묻는 질문
- **Q. 설비를 선택했는데 작업지시가 검색되지 않습니다.**
  **A.** 선택한 설비의 공정에 내려온 **완제품(FINISHED)** 작업지시만 조회됩니다. 공정 배정이 맞는지 작업지시 관리 화면에서 확인하세요.
- **Q. SG 스캔이 "BOM에 없는 반제품입니다(오투입)"으로 거부됩니다.**
  **A.** 선택한 완제품 작업지시의 BOM 반제품 구성에 없는 품목입니다. 올바른 SG 라벨인지, 작업지시를 잘못 선택하지 않았는지 확인하세요.
- **Q. 자재 스캔 시 "장착할 공정재고가 없습니다"라고 나옵니다.**
  **A.** 해당 LOT이 아직 원자재창고에서 공정으로 출고(입고)되지 않았습니다. 자재 출고 처리를 먼저 진행하세요.
- **Q. "발행 취소"를 누르면 발행된 FG 라벨을 다시 쓸 수 있나요?**
  **A.** 아니요. 발행 취소는 화면 상태만 초기화합니다. 서버의 FG 라벨은 `ISSUED` 상태로 그대로 남고 화면에서 다시 확정할 수 없으니, 실물 라벨은 폐기하고 새로 조립을 실행해 라벨을 다시 발행하세요.
- **Q. 실물 FG 라벨을 스캔했는데 확정이 거부됩니다.**
  **A.** 스캔값이 방금 발행된 FG 바코드와 다르거나, 이미 확정 처리된(이력이 있는) 라벨일 수 있습니다.
