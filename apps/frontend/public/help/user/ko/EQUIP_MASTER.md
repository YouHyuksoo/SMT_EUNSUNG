# 설비마스터

## 화면 목적
생산에 사용하는 **모든 설비의 기본정보**와 **설비별 부품·소모품 구성(BOM)**을 등록·관리합니다. 여기 등록된 설비가 생산실적·키오스크·설비통신·점검·금형 등에서 선택·참조되는 기준이 됩니다.

## 화면 구성
화면은 상단 탭으로 두 영역이 나뉩니다.
- **설비 기본정보 탭**: 설비 마스터 그리드(좌) + 등록/수정 슬라이드 패널(우). 상단에서 검색·유형/라인/통신 필터, **새로고침**, **설비 추가**.
- **설비 BOM 관리 탭**: 왼쪽 **설비 목록**에서 설비를 고르면 오른쪽에 그 설비의 **BOM 목록**이 표시됩니다. 상단에서 **BOM 품목 등록**, 선택 설비에 **품목 연결**.

```
설비 기본정보(EQUIP_MASTERS)
   └─ 설비 BOM 연결(EQUIP_BOM_RELS) ──▶ BOM 품목(EQUIP_BOM_ITEMS: 부품/소모품)
```

## ① 설비 그리드 컬럼 (설비 기본정보 탭)

| 컬럼 | 역할 / 의미 |
|------|------|
| **작업(actions)** | 행의 ✏️(수정)·🗑️(삭제) 버튼입니다. 수정은 우측 패널을 열고, 삭제는 확인 모달을 띄웁니다. |
| **설비코드(equipCode)** | 설비를 식별하는 **고유 코드**입니다. 한 번 등록하면 변경하지 않는 것을 원칙으로 합니다(생산실적·BOM·통신이 이 코드로 연결됨). |
| **사진(imageUrl)** | 설비 사진 썸네일입니다. 없으면 이미지 아이콘으로 표시됩니다. |
| **설비명(equipName)** | 현장에서 식별할 설비 이름입니다. |
| **유형(equipType)** | 설비 종류입니다(공통코드 EQUIP_TYPE). 자동압착·단심절단·다심절단·트위스트·솔더·하우징·검사·라벨프린터 등. |
| **공정(processCode)** | 설비가 소속된 공정입니다. 공정명/공정코드를 함께 표시합니다. |
| **통신(commType)** | 설비와 MES 간 통신 방식입니다. None(없음)·MQTT·Serial·TCP/IP 배지로 표시됩니다. |
| **IP/Port(ipAddress)** | 통신 연결 정보입니다. TCP/MQTT는 `IP:포트`, Serial은 라인코드, None은 `-`로 표시됩니다. |
| **제조사(maker)** | 설비 제조사입니다. |
| **모델(modelName)** | 설비 모델명입니다. |
| **사용(useYn)** | 사용 여부입니다. `Y`면 초록 점, `N`이면 회색 점으로 표시됩니다. |

### 설비 등록/수정 폼 필드 (우측 패널)

| 항목 | 역할 / 의미 |
|------|------|
| **설비코드(equipCode)** | 필수. 신규 등록 시에만 입력 가능하며 수정 시 잠깁니다(불변 키). |
| **설비명(equipName)** | 필수. 설비 표시 이름입니다. |
| **유형(equipType)** | 설비 종류 선택입니다(공통코드 EQUIP_TYPE). |
| **통신방식(commType)** | 통신 방식 선택입니다. 선택 값에 따라 아래 통신설정 입력란이 달라집니다. |
| **라인(lineCode)** | 설비가 속한 생산 라인입니다(라인 선택). |
| **IP 주소(ipAddress)** | 통신방식이 **TCP 또는 MQTT**일 때만 표시됩니다. 설비의 IP 주소입니다. |
| **포트(port)** | 통신방식이 TCP/MQTT일 때 입력하는 포트 번호입니다(1~65535). |
| **MQTT Topic(mqttTopic)** | 통신방식이 **MQTT**일 때 표시되는 토픽 입력란입니다. |
| **시리얼 포트(serialPort)** | 통신방식이 **SERIAL**일 때 표시되는 시리얼 포트 입력란입니다. |
| **Baud Rate(baudRate)** | 통신방식이 SERIAL일 때 통신 속도입니다(기본 9600). |
| **제조사(maker)** | 설비 제조사입니다. |
| **모델명(modelName)** | 설비 모델명입니다. |
| **사진(imageUrl)** | 설비 사진입니다. 클릭하여 업로드하며 변경/삭제할 수 있습니다(JPG/PNG/GIF/WebP, 최대 5MB). |

## ② BOM 품목 컬럼 (설비 BOM 관리 탭 — 품목 마스터)

설비에 장착·소모되는 부품·소모품의 **품목 마스터**입니다. 먼저 여기에 품목을 등록한 뒤 설비에 연결합니다.

| 컬럼 | 역할 / 의미 |
|------|------|
| **유형(itemType)** | 품목 구분입니다. **부품(PART)** 또는 **소모품(CONSUMABLE)**. 배지 색으로 구분됩니다. |
| **품목코드(itemCode)** | BOM 품목을 식별하는 고유 코드입니다(수정 시 잠김). |
| **품목명(itemName)** | 부품/소모품 이름입니다. |
| **규격(spec)** | 품목 규격·사양·치수입니다. |
| **제조사(maker)** | 부품/소모품 제조사입니다. |
| **단위(unit)** | 수량 단위입니다(기본 EA). |
| **단가(unitPrice)** | 품목 단가입니다(원화 표시). |
| **교체주기(replacementCycle)** | 소모품 등의 권장 교체 주기 **일수**입니다. |
| **현재재고(stockQty)** | 현재 보유 재고 수량입니다. 안전재고 이하면 빨간 글씨로 강조됩니다. |
| **안전재고(safetyStock)** | 재고 부족 판단 기준 수량입니다. |
| **사용여부(useYn)** | `Y`만 연결·선택 대상입니다. |

## ③ 설비-BOM 연결 컬럼 (설비 BOM 관리 탭 — 연결 목록)

선택한 설비에 실제로 장착·구성된 BOM 품목 목록입니다.

| 컬럼 | 역할 / 의미 |
|------|------|
| **작업(actions)** | ✏️(연결 수정)·🔗해제(연결 삭제) 버튼입니다. |
| **유형(bomItem.itemType)** | 연결된 품목의 부품/소모품 구분입니다. |
| **품목코드(bomItem.itemCode)** | 연결된 BOM 품목의 코드입니다. |
| **품목명(bomItem.itemName)** | 연결된 BOM 품목명입니다. |
| **규격(bomItem.spec)** | 연결된 품목의 규격입니다. |
| **수량(quantity)** | 이 설비에 필요한 수량입니다. |
| **설치일(installDate)** | 부품/소모품을 설치한 날짜입니다. |
| **유효기한(expireDate)** | 교체 권장 기한입니다. 지난 날짜면 빨간 글씨로 표시됩니다. |
| **비고(remark)** | 연결 관련 참고사항입니다. |

## 사용 순서
1. **설비 기본정보 탭**에서 **설비 추가**(또는 행 ✏️)로 폼을 엽니다.
2. 설비코드·설비명·유형을 입력하고, 통신방식을 고른 뒤 필요한 통신설정(IP/포트, MQTT Topic, Serial 등)을 채웁니다.
3. 제조사·모델명·사진을 입력하고 **저장**합니다.
4. **설비 BOM 관리 탭**으로 이동해 **BOM 품목 등록**으로 부품·소모품을 마스터에 등록합니다.
5. 왼쪽 **설비 목록**에서 설비를 선택하고 **품목 연결**로 수량·설치일·유효기한과 함께 BOM을 연결합니다.

## 입력 규칙 / 검증
- **설비코드·설비명은 필수**입니다. 둘 중 하나라도 비면 저장 버튼이 비활성화됩니다.
- 설비코드·품목코드는 **수정 시 변경할 수 없습니다**(고유 키).
- IP 주소는 **유효한 IP 형식**이어야 하며, 포트는 **1~65535** 범위여야 합니다.
- IP 주소·포트는 **통신방식이 TCP 또는 MQTT일 때만** 저장됩니다.
- BOM 품목 등록 시 품목코드·품목명이 필수이고, BOM 연결 시 **BOM 품목 선택**이 필수입니다.
- 사진은 이미지 형식(JPG/PNG/GIF/WebP), 최대 5MB까지만 업로드됩니다.

## 자주 묻는 질문
- **Q.** 통신설정 입력란이 안 보여요.
  **A.** 통신방식을 **TCP/MQTT**로 고르면 IP/포트(MQTT는 Topic)가, **SERIAL**로 고르면 시리얼 포트/Baud Rate가 나타납니다. None이면 통신설정이 없습니다.
- **Q.** 설비코드를 잘못 입력했어요.
  **A.** 코드는 수정할 수 없으므로 삭제 후 다시 등록하거나 운영자에게 문의하세요(다른 데이터와 연결돼 있으면 삭제가 막힐 수 있습니다).
- **Q.** BOM 목록이 비어 있어요.
  **A.** 왼쪽에서 설비를 먼저 선택해야 그 설비의 BOM이 표시됩니다. 연결이 없으면 비어 있습니다.
- **Q.** 부품과 소모품은 무엇이 다른가요?
  **A.** 부품(PART)은 설비에 장착되는 교체 부품, 소모품(CONSUMABLE)은 생산 중 소모되는 자재입니다. 유형 배지 색으로 구분합니다.
