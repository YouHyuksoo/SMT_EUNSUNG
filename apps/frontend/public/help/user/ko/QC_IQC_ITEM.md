# 검사항목마스터

## 화면 목적
수입검사(IQC)에서 사용할 **전역 검사항목**을 한곳에서 등록·관리하는 화면입니다. 여기서 만든 항목은 [품목별 IQC 항목관리](/master/iqc-part-spec)에서 품목마다 골라 쓰는 **공용 항목 풀** 역할을 합니다. 즉, 같은 검사항목(예: 외관, 치수)을 품목마다 새로 만들지 않고 한 번 등록해 재사용합니다.

## 화면 구성
- **좌측 — 검사항목 목록(그리드)**: 등록된 검사항목을 표로 표시. 검색·판정방법 필터 제공.
- **우측 — 추가/수정 패널**: `검사항목추가` 또는 행의 수정(✎)을 누르면 오른쪽에서 슬라이드로 열립니다.

## ① 검사항목 목록(그리드) 컬럼

| 컬럼 | 역할 / 의미 |
|---|---|
| **수정 / 삭제** | ✎ 수정 패널 열기, 🗑 삭제(확인 후). 다른 화면에서 사용 중인 항목은 삭제에 주의. |
| **항목코드(inspItemCode)** | 검사항목의 고유 식별 코드. 품목별 기준에서 이 코드로 항목을 참조. 등록 후 **변경 불가**(키). |
| **검사항목(inspItemName)** | 검사항목 이름. 예: 외관, 치수, 인장강도, 색상. 실제 검사자가 보는 명칭. |
| **판정방법(judgeMethod)** | 검사 결과를 어떻게 판정하는지. **육안(VISUAL)** = 합/불 눈으로 판정, **계측(MEASURE)** = 측정값으로 규격 비교. 배지 색으로 구분. |
| **단위(unit)** | 계측 항목의 측정 단위(mm, g 등). 공통코드 `UNIT_TYPE`에서 선택. 육안 항목은 보통 비움. |

> 상단 도구모음: **검색**(코드·항목명), **판정방법 필터**(육안/계측), **새로고침**, **검사항목추가**.

## ② 검사항목 추가 / 수정 컬럼

`검사항목추가`를 누르면 우측 패널에 아래 입력 항목이 나옵니다.

| 컬럼 | 역할 / 의미 |
|---|---|
| **항목코드** | 필수. 항목의 고유 코드. **수정 시에는 잠겨서 바꿀 수 없습니다**(키). 신규 등록 시에만 입력. |
| **판정방법** | 필수 선택. **육안** 또는 **계측**. 계측을 고르면 단위를 함께 지정하는 것을 권장. |
| **검사항목** | 필수. 검사항목 이름. |
| **단위** | 선택. 공통코드 `UNIT_TYPE` 목록에서 선택(mm, g, ㎏ 등). 육안 항목은 비워도 됩니다. |

## 사용 순서
1. `검사항목추가`를 누른다.
2. **항목코드**·**검사항목**을 입력하고 **판정방법**을 고른다.
3. 계측 항목이면 **단위**를 선택한다.
4. `추가`를 눌러 저장한다.
5. 등록한 항목은 [품목별 IQC 항목관리](/master/iqc-part-spec)에서 품목에 연결해 규격(상·하한, 판정기준)을 설정한다.

## 입력 규칙 / 검증
- **항목코드**와 **검사항목**은 필수입니다. 둘 중 하나라도 비면 저장 버튼이 비활성화됩니다.
- **항목코드는 등록 후 변경 불가**합니다. 잘못 입력했다면 삭제 후 다시 등록하세요.
- 검색은 **항목코드·검사항목명**을 부분일치로 찾습니다.

## 자주 묻는 질문
- **항목코드를 바꾸고 싶어요** — 코드는 키라 수정할 수 없습니다. 삭제 후 새 코드로 등록하세요(이미 품목에 연결됐다면 연결도 다시 설정).
- **상·하한 규격(LSL/USL)이나 판정기준은 어디서 넣나요?** — 이 화면은 **공용 항목**만 정의합니다. 품목마다 다른 규격·판정기준·샘플방식은 [품목별 IQC 항목관리](/master/iqc-part-spec)에서 품목별로 설정합니다.
- **육안과 계측의 차이는?** — 육안은 눈으로 합/불 판정(예: 외관 흠집), 계측은 측정기로 값을 재서 규격과 비교(예: 치수 mm)합니다.
