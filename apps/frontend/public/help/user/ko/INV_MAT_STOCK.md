# 자재재고현황조회

## 화면 목적
창고별·품목별로 보유 중인 자재 재고를 실시간으로 조회합니다. 시리얼(LOT) 단위의 상세 현황과 품목별 그룹 합계를 탭으로 제공하며, 안전재고 대비 상태와 유효기간 만료 여부를 한눈에 확인할 수 있습니다.

## 화면 구성
- **두 개의 탭**: **시리얼별 상세**(개별 LOT 단위) / **품목별 그룹 합계**(품목 기준 집계)
- **상단 필터**: 창고 선택 드롭다운 + 품목코드/품목명 검색 입력
- **DataGrid 목록**: 선택한 탭에 따라 다른 컬럼 구성으로 표시

---

## ① 시리얼별 상세 탭 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **품목코드(itemCode)** | 재고 품목의 코드입니다. |
| **품목명(itemName)** | 재고 품목의 이름입니다. |
| **자재시리얼(matUid)** | 개별 LOT/시리얼 번호입니다. 동일 품목이라도 시리얼이 다르면 별도 행으로 관리됩니다. |
| **창고(warehouseName)** | 재고가 위치한 창고입니다. |
| **수량(qty)** | 현재 창고에 보관 중인 총 재고 수량입니다. |
| **안전재고(safetyStock)** | 해당 품목의 안전재고 기준 수량입니다(품목마스터에 설정). |
| **상태(stockLevel)** | 실제 수량 대비 안전재고 충족 여부입니다. **부족**(빨강) / **주의**(노랑) / **정상**(초록). |
| **제조일자(manufactureDate)** | 해당 LOT의 제조일자입니다. |
| **경과일수(elapsedDays)** | 제조일로부터 경과한 일수입니다. |
| **남은유효기간(remainingDays)** | 유통기한 만료까지 남은 일수입니다. **음수면 만료**를 의미합니다. |
| **유효기간(shelfLifeStatus)** | 유효기간 상태입니다. **만료**(빨강) / **임박**(노랑) / **정상**(초록). |

---

## ② 품목별 그룹 합계 탭 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **품목코드(itemCode)** | 품목 코드입니다. |
| **품목명(itemName)** | 품목 이름입니다. |
| **총 재고수량(totalQty)** | 해당 품목의 전체 창고 총 재고 수량 합계입니다. |
| **안전재고(safetyStock)** | 품목의 안전재고 기준입니다. |
| **상태(stockLevel)** | 총 재고 대비 안전재고 충족 여부. |
| **시리얼 수(serialCount)** | 해당 품목이 보유한 LOT/시리얼 개수입니다. |
| **창고 수(warehouseCount)** | 해당 품목이 분산 보관된 창고 개수입니다. |

---

## 사용 순서
1. 상단 필터에서 조회할 **창고**와 **검색어**(품목코드/품목명)를 입력합니다.
2. **시리얼별 상세** 탭에서 개별 LOT 단위 재고 현황을 확인합니다.
3. **품목별 그룹 합계** 탭에서 품목 기준 집계 현황을 확인합니다.
4. 안전재고 미달(빨강) 또는 유효기간 만료(빨강) 항목은 즉시 조치가 필요할 수 있습니다.
5. 우측 상단 **내보내기**로 현재 조회 결과를 엑셀 파일로 다운로드할 수 있습니다.
