# 품목별 IQC 항목관리

## 화면 목적
자재 품목(원자재) 하나하나에 대해 **수입검사(IQC) 시 무엇을 검사할지**를 정해 두는 화면입니다.
품목을 고른 뒤, 그 품목에 적용할 **검사항목**(예: 외관, 치수, 인장강도)과 각 항목의 **규격**(상·하한, 판정기준), **불량등급**, **샘플 뽑는 방식**을 등록합니다. 여기서 등록한 내용이 실제 수입검사 화면의 검사표로 그대로 펼쳐집니다.

> 검사항목 자체(이름·판정방법·단위)는 [검사항목 풀]에서 먼저 등록합니다. 이 화면에서는 그 항목들을 **품목에 가져다 붙이고 규격을 채우는** 일을 합니다.

## 화면 구성
- **왼쪽 — 자재 품목 목록**: 원자재(RAW_MATERIAL) 품목 리스트. 품목코드·품목명으로 검색하고, 이미 검사항목이 등록된 품목은 오른쪽에 **건수 배지**가 표시됩니다.
- **오른쪽 위 — AQL 기준(요약)**: 선택한 품목에 연결된 AQL 정책으로 산출되는 샘플수·Ac/Re 등을 미리 보여주는 **읽기 전용 카드**입니다(품목마스터에서 연결한 정책 기준).
- **오른쪽 아래 — 검사항목**: 그 품목의 검사항목을 표로 추가·수정·삭제하고 [저장]하는 영역입니다. 상단 툴바에서 **기본 시료수**·**파괴검사 여부**도 함께 설정합니다.

## ① 자재 품목 목록 (왼쪽)

| 컬럼 | 역할 / 의미 |
|------|------|
| **품목코드 / 품목명** | 검사기준을 설정할 원자재 품목입니다. 클릭하면 오른쪽에 해당 품목의 검사항목이 표시됩니다. |
| **건수 배지** | 그 품목에 등록된 **검사항목 수**입니다. 0이면 배지가 없고(미설정), 숫자가 있으면 이미 기준이 등록된 품목입니다. |
| **검색** | 품목코드 또는 품목명으로 목록을 거릅니다. |

## ② AQL 기준 요약 카드 (오른쪽 위, 읽기 전용)
이 카드는 입력란이 아니라 **참고용 요약**입니다. 값은 [품목마스터]에서 품목에 연결한 **AQL 정책**과 [AQL 기준관리]의 설정에서 자동 계산됩니다.

| 항목 | 역할 / 의미 |
|------|------|
| **LOT 수량 미리보기** | 입고 LOT 수량을 가정해 넣으면, 그 수량 기준으로 아래 샘플수·Ac/Re가 어떻게 잡히는지 미리 보여줍니다. |
| **AQL 정책** | 품목마스터에서 이 품목에 연결한 AQL 정책 코드입니다. 비어 있으면 AQL 자동판정이 적용되지 않습니다. |
| **기본시료수** | 이 품목 검사기준 헤더에 설정된 기본 시료수입니다(아래 툴바의 값과 동일). |
| **검사수준** | AQL 정책의 검사수준(보통 II)입니다. LOT 크기와 만나 샘플 크기를 정합니다. |
| **샘플수량** | 위 LOT 수량 기준으로 산출된 표본 개수입니다. |
| **Major Ac/Re** | 중결함의 합격(Ac)·불합격(Re) 판정개수입니다. |
| **Minor Ac/Re** | 경결함의 합격·불합격 판정개수입니다. |
| **검사항목 기준** | 이 품목에 등록된 검사항목 건수입니다. |
| **파괴/고정** | 검사항목 중 **파괴검사·전수·고정 샘플** 등 별도 표본수를 쓰는 항목 수입니다. |

## ③ 검사항목 헤더 (오른쪽 아래 툴바)

| 항목 | 역할 / 의미 |
|------|------|
| **기본 시료수** | 이 품목 검사 시 기본으로 뽑는 표본 개수입니다(개별 항목에서 고정 샘플수를 지정하지 않은 경우의 기본값). |
| **파괴검사 / 비파괴** | 이 품목 검사가 표본을 파괴하는지 여부입니다. `파괴`로 두면 검사 후 시료를 사용할 수 없는 검사임을 표시합니다. |
| **템플릿 불러오기/관리** | 자주 쓰는 검사항목 묶음을 템플릿으로 저장해 두고, 새 품목에 한 번에 적용합니다. |
| **항목 추가** | 검사항목 한 줄을 추가하고 바로 편집 모드로 들어갑니다. |
| **저장** | 변경한 헤더·검사항목 전체를 저장합니다. 변경이 없으면 비활성화됩니다. |

## ④ 검사항목 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **순서(seq)** | 검사표에 표시되는 항목 순서입니다. 행 추가·삭제 시 자동으로 다시 매겨집니다. |
| **검사항목(inspItemCode)** | [검사항목 풀]에 등록된 항목 중 하나를 선택합니다. 선택하면 종류·단위가 자동으로 따라옵니다. |
| **종류(judgeMethod)** | 항목의 판정방법입니다. **측정형(MEASURE)**=숫자를 재서 상·하한과 비교, **판정형(VISUAL)**=양호/불량 등 눈으로 확인. 측정형일 때만 LSL/USL을 입력합니다. |
| **검사유형(inspectionType)** | 표본을 정하는 방식입니다. `AQL`=AQL 표로 자동 산출, `DESTRUCTIVE`=파괴검사, `FULL`=전수검사. AQL이 아니면 **샘플수**를 직접 입력합니다. |
| **샘플수(sampleQty)** | 파괴/전수/고정 유형에서 검사할 **고정 표본 개수**입니다. AQL 유형이면 `자동`으로 표시되고 입력하지 않습니다. |
| **불량등급(defectGrade)** | 이 항목 불량의 심각도입니다. `CRITICAL`(치명)·`MAJOR`(중)·`MINOR`(경). AQL 판정에서 등급별로 Ac/Re가 달리 적용됩니다. |
| **검사수준(inspectionLevel)** | 이 항목에 적용할 ISO 2859-1 검사수준(II, S4 등)입니다. 표본 크기 결정에 사용됩니다. |
| **AQL** | 이 항목에 적용할 합격품질한계 값(0.65/1.0/2.5 등)입니다. **작을수록 엄격**합니다. |
| **하한(LSL)** | 측정형 항목의 **허용 최소값**입니다. 측정값이 이보다 작으면 불량입니다. |
| **상한(USL)** | 측정형 항목의 **허용 최대값**입니다. 측정값이 이보다 크면 불량입니다. |
| **판정기준(judgeCriteria)** | 판정형 항목 등에서 합·불을 가르는 기준 문구입니다(예: "스크래치·이물 없을 것"). |
| **단위(unit)** | 측정값의 단위입니다. 검사항목 풀에서 따라오며 표시 전용입니다. |

> 예시: `치수` 항목을 측정형으로 두고 LSL=9.8, USL=10.2 → 측정값이 9.8~10.2(mm) 범위를 벗어나면 불량으로 판정합니다.

## 사용 순서
1. 왼쪽에서 검사기준을 설정할 **품목을 선택**합니다.
2. 툴바에서 **기본 시료수**와 **파괴검사 여부**를 설정합니다.
3. **[항목 추가]**(또는 **[템플릿 불러오기]**)로 검사항목을 넣고, 각 행에서 종류·검사유형·불량등급·규격을 채웁니다.
4. 측정형 항목은 **하한(LSL)·상한(USL)**을, 판정형 항목은 **판정기준**을 입력합니다.
5. 각 행은 **[수정]→[확인]**으로 편집하고, 마지막에 **[저장]**합니다.

## 입력 규칙 / 검증
- **검사항목(inspItemCode)이 비어 있는 행은 저장되지 않습니다**(빈 행은 자동 제외).
- 측정형이 아닌 항목에는 LSL/USL을 입력할 수 없습니다(판정기준으로 관리).
- 검사유형이 `AQL`이면 샘플수는 자동 산출되어 직접 입력하지 않습니다.
- 한 번에 한 행만 편집할 수 있으며, 편집 중에는 다른 행의 수정·삭제가 잠깁니다.

## 자주 묻는 질문
- **Q.** 검사항목 선택 목록에 원하는 항목이 안 보여요.
  **A.** [검사항목 풀]에서 먼저 등록(사용여부 `Y`)해야 이 화면에서 선택할 수 있습니다.
- **Q.** AQL 기준 요약 카드의 값을 여기서 바꿀 수 있나요?
  **A.** 아니요. 그 값은 [품목마스터]의 AQL 정책 연결과 [AQL 기준관리] 설정에서 정해지는 읽기 전용 참고 정보입니다.
- **Q.** 매번 같은 항목을 넣기 번거롭습니다.
  **A.** [템플릿 불러오기/관리]로 항목 묶음을 저장해 두고 새 품목에 한 번에 적용하세요.
