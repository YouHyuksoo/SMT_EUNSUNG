# 팔레트적재관리

## 화면 목적

확정된 출하지시(CONFIRMED)를 기준으로 **팔레트**를 생성하고, 포장 완료 박스를 스캔·선택해 팔레트에 적재합니다.
팔레트 구성이 끝나면 마감(CLOSE)하고 라벨을 출력해 출하를 준비합니다.

- 상태 흐름: `OPEN(박스 적재 중) → CLOSED(마감) → LOADED(상차) → SHIPPED(출하완료)`
- CLOSED 상태에서는 재오픈해 박스를 수정할 수 있습니다.

## 화면 구성

```
┌──────────────────────────────────────────────┬─────────────────────┐
│  ① 팔레트 목록 그리드                          │  ② 포함 박스 패널    │
│  (생성일 기간 · 검색 · 상태 · 바코드 스캔 필터)  │  (행 클릭 후 표시)   │
└──────────────────────────────────────────────┴─────────────────────┘
```

> **기간 밖 미완료 알림**: 조회 기간 밖이지만 아직 출하가 끝나지 않은 팔레트(OPEN/CLOSED/LOADED)는 자동으로 목록에 포함되고, 왼쪽 주황색 테두리로 표시됩니다.

## ① 팔레트 목록 그리드 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **(액션 버튼)** | 박스추가(+)·마감(자물쇠)·재오픈·라벨출력·빈팔레트삭제 아이콘. 각 버튼의 활성 조건은 아래 버튼별 기능 참조. |
| **출하지시번호** | 이 팔레트가 연결된 출하지시 번호. 없으면 `-`. |
| **팔레트번호** | 팔레트 고유 식별 번호. 자동 채번됩니다. |
| **박스수** | 현재 이 팔레트에 적재된 박스 개수. |
| **총수량** | 적재된 박스 내 완제품 수량 합계. |
| **상태** | 팔레트의 현재 상태 배지(OPEN/CLOSED/LOADED/SHIPPED). |
| **출하번호** | 출하(선적) 처리 시 부여되는 출하 식별 번호. 없으면 `-`. |
| **생성일시** | 팔레트가 생성된 일시. 기간 필터 기준. |

### 그리드 도구 모음

| 도구 | 역할 |
|------|------|
| **생성일 기간** | 기본값은 당일. 기간을 변경해 과거 팔레트를 조회합니다. |
| **검색** | 팔레트번호로 특정 팔레트를 찾습니다. |
| **상태** | 특정 상태(OPEN/CLOSED 등)만 골라 봅니다. |
| **바코드 스캔** | 팔레트 바코드를 스캔하면 해당 팔레트를 목록에서 자동 검색합니다. |

## ② 포함 박스 패널

팔레트 행을 클릭하면 우측에 해당 팔레트에 적재된 박스 목록이 표시됩니다.

| 항목 | 역할 / 의미 |
|------|------|
| **박스번호** | 적재된 박스의 고유 번호. |
| **품목코드** | 박스에 담긴 완제품 품목 코드. |
| **수량** | 박스 내 수량. |
| **제거(X)** | OPEN 상태 팔레트에서만 표시. 클릭 시 확인 팝업 후 해당 박스를 팔레트에서 분리합니다. |

## 버튼별 기능

### 헤더 버튼

| 버튼 | 활성 조건 | 동작 |
|------|------|------|
| **새로 고침** | 항상 | 팔레트 목록과 출하지시 목록을 동시에 재조회합니다. |
| **팔레트 생성** | 항상 | 팔레트 생성 모달을 엽니다. |

### 그리드 행 액션 버튼

| 버튼(아이콘) | 활성 조건 | 동작 |
|------|------|------|
| **박스추가(+)** | OPEN + 출하지시 연결됨 | 박스 할당 모달을 열어 박스를 스캔·선택합니다. |
| **마감(자물쇠)** | OPEN + 출하지시 연결됨 | 팔레트를 OPEN → CLOSED로 마감합니다. |
| **재오픈(자물쇠해제)** | CLOSED | 팔레트를 CLOSED → OPEN으로 되돌려 박스를 수정합니다. |
| **라벨출력(프린터)** | 항상 | 팔레트 라벨 미리보기 모달을 열고 바코드 라벨을 인쇄합니다. |
| **빈팔레트삭제(휴지통)** | OPEN + 박스 0개 + 출하번호 없음 | 확인 팝업 후 빈 팔레트를 삭제합니다. |

## 팔레트 생성 모달

확정(CONFIRMED) 출하지시를 선택하거나 스캔해 팔레트를 생성합니다.

- 좌측: 팔레트 생성 대기중인 출하지시 목록. 팔레트가 아직 생성되지 않은 CONFIRMED 지시만 표시됩니다.
- 우측: 출하지시번호 스캔 입력(Enter로 확인) + 선택된 출하지시 정보 표시.
- 팔레트 번호는 자동으로 채번됩니다.

> 하나의 출하지시에는 팔레트 1개만 생성할 수 있습니다.

## 박스 할당 모달

팔레트에 적재할 박스를 스캔하거나 목록에서 선택합니다.

| 항목 | 역할 |
|------|------|
| **박스번호 스캔** | 바코드 스캐너로 박스를 스캔합니다. Enter 키로 확인. 적재 가능한 박스면 선택 목록에 즉시 추가됩니다. |
| **박스 목록** | 적재 가능한 박스(포장 완료 + 미할당 + OQC 합격) 전체 목록. 클릭으로 선택/해제합니다. |
| **할당 버튼** | 선택된 박스를 팔레트에 일괄 할당합니다. |

> OQC(외관검사) 기능이 활성화된 경우, OQC 합격(PASS) 박스만 적재 목록에 표시됩니다.

## 라벨 출력 모달

팔레트 라벨을 미리보기하고 바코드 라벨을 출력합니다.

- **라벨 템플릿 선택**: 기본 라벨 또는 등록된 라벨 디자인 중 선택합니다.
- **라벨 내용**: 팔레트번호(Code128 바코드), 박스수, 총수량, 상태, 대표품목, 생성일시, 구성박스 목록.
- **라벨 출력 버튼**: 클릭 시 100mm × 120mm 용지에 라벨을 인쇄합니다.

## 업무 흐름

```
출하지시관리(/shipping/order)에서 출하지시 CONFIRMED 확정
    ↓
[팔레트 생성] 버튼 → 출하지시 선택(스캔 또는 목록) → [생성]
    ↓
팔레트(OPEN) 생성됨 → 행 클릭으로 선택
    ↓
액션 [+박스추가] → 박스 스캔 또는 목록 선택 → [할당]
    ↓  (반복 가능)
박스 구성 완료 → 액션 [마감] → CLOSED 전환
    ↓
액션 [라벨출력] → 바코드 라벨 인쇄
    ↓
출하처리(/shipping/pallet-ship)에서 LOADED → SHIPPED
```

## 입력 규칙 / 검증

- 팔레트 생성은 **CONFIRMED 상태의 출하지시**에만 가능합니다.
- 출하지시 1개당 팔레트는 **1개만** 생성할 수 있습니다.
- 박스 할당은 **OPEN 상태 팔레트**에서만 가능합니다.
- 적재 가능 박스 조건: **포장 완료(CLOSED) + 다른 팔레트 미할당 + OQC 합격(활성화 시)**.
- 마감은 **출하지시가 연결된 OPEN 팔레트**에서만 가능합니다.
- 재오픈은 **CLOSED 상태**에서만 가능합니다.
- 빈 팔레트 삭제는 **OPEN + 박스 0개 + 출하 미처리** 조건에서만 가능합니다.

## 자주 묻는 질문

- **Q. 팔레트 생성 모달에 선택할 출하지시가 없습니다.**
  **A.** 출하지시관리 화면에서 해당 출하지시가 CONFIRMED 상태인지 확인하세요. 이미 팔레트가 생성된 지시는 목록에 표시되지 않습니다.

- **Q. 박스 할당 모달에 적재 가능한 박스가 없습니다.**
  **A.** 포장관리 화면에서 해당 박스가 포장 완료(CLOSED) 처리됐는지, 다른 팔레트에 이미 할당되지 않았는지 확인하세요. OQC 기능이 켜진 경우 OQC 합격 박스만 표시됩니다.

- **Q. 박스를 스캔했는데 "적재 가능한 박스를 찾을 수 없습니다."가 나옵니다.**
  **A.** 해당 박스가 포장 완료 상태가 아니거나, 이미 다른 팔레트에 할당됐거나, OQC 검사를 통과하지 않은 경우입니다. 포장관리·OQC 화면에서 해당 박스 상태를 확인하세요.

- **Q. 마감(자물쇠) 버튼이 비활성 상태입니다.**
  **A.** 팔레트에 출하지시가 연결되어 있어야 하며 OPEN 상태여야 합니다. 팔레트 생성 시 출하지시를 선택했는지 확인하세요.

- **Q. 생성일 기간을 당일로 설정했는데 오래된 팔레트가 보입니다.**
  **A.** 기간 밖이지만 미완료(OPEN/CLOSED/LOADED) 상태인 팔레트는 자동으로 포함됩니다. 상태 필터를 지정하면 이 자동 포함이 해제됩니다.

- **Q. 라벨 인쇄가 되지 않습니다.**
  **A.** 브라우저 팝업 차단 설정을 확인하고 허용으로 변경하세요. 라벨 용지 크기는 100mm × 120mm입니다.
