# 작업자관리

## 화면 목적
생산·검사·포장 현장에서 일하는 **작업자(인원)의 기본정보**를 등록·관리합니다. 여기 등록된 작업자가 생산실적 등록, 키오스크/PDA 로그인·작업자 선택, QR 스캔 식별의 기준이 됩니다.

## 화면 구성
- **왼쪽(목록)**: 작업자 그리드. 사진(아바타)·작업자코드·이름·영문명·부서·직급·전화번호·사용여부를 보여줍니다. 상단에서 **코드/이름 검색**, **사용여부 필터**, **새로고침**, **작업자 추가**를 할 수 있습니다.
- **오른쪽(슬라이드 패널)**: 행의 ✏️(수정) 또는 "작업자 추가" 클릭 시 오른쪽에서 열리는 등록/수정 폼입니다. 상단 **사진 업로드** 영역과 **기본정보 / 근무정보 / 비고** 섹션으로 구성됩니다.
- 각 입력 필드 라벨 옆 도움말(?) 아이콘에 항목 설명이 붙어 있습니다.

---

## ① 목록 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **작업(actions)** | 행별 ✏️ 수정 / 🗑️ 삭제 버튼입니다. 수정은 오른쪽 폼을 열고, 삭제는 확인 후 작업자를 제거합니다. |
| **사진(photoUrl)** | 작업자 사진(아바타)입니다. 사진이 없으면 이름·부서 기반 기본 아바타로 표시됩니다. |
| **작업자코드/사번(workerCode)** | 작업자를 식별하는 **고유 코드(사번)**입니다. 한 번 등록하면 변경하지 않는 것을 원칙으로 합니다(생산실적·키오스크가 이 코드로 연결됨). |
| **작업자명(workerName)** | 현장에서 식별할 작업자 이름입니다. |
| **영문명(engName)** | 작업자의 영문 이름입니다(없으면 `-`). |
| **부서(dept)** | 작업자가 소속된 부서입니다(없으면 `-`). |
| **직급(position)** | 작업자의 직급/직위입니다(공통코드 JOB_POSITION, 없으면 `-`). |
| **전화번호(phone)** | 작업자 연락처입니다(없으면 `-`). |
| **사용(useYn)** | 사용여부 배지입니다. `Y`는 "사용", `N`은 "미사용"으로 표시됩니다. |

## ② 입력 폼 — 기본정보

| 컬럼 | 역할 / 의미 |
|------|------|
| **사진(photoUrl)** | 폼 상단에서 업로드하는 작업자 사진입니다. 이미지 파일만 가능하며 목록 아바타로 표시됩니다. |
| **작업자코드/사번(workerCode)** | **필수**. 작업자 식별 고유 코드(사번)입니다. 수정 시에는 잠겨(변경 불가) 표시됩니다. |
| **작업자명(workerName)** | **필수**. 작업자 이름입니다. |
| **영문명(engName)** | 작업자의 영문 이름입니다. |
| **부서(dept)** | 작업자가 소속된 부서입니다. 부서 선택 목록에서 고릅니다. |
| **직급(position)** | 작업자의 직급/직위입니다. 공통코드(JOB_POSITION) 셀렉트에서 고릅니다. |
| **전화번호(phone)** | 작업자 연락처(전화번호)입니다. |
| **이메일(email)** | 작업자 이메일 주소입니다. |
| **QR코드(qrCode)** | 키오스크/PDA에서 작업자를 식별하는 QR 스캔용 코드입니다. 비워두면 사번(작업자코드)으로도 스캔 식별이 가능합니다. |

## ③ 입력 폼 — 근무정보

| 컬럼 | 역할 / 의미 |
|------|------|
| **입사일(hireDate)** | 작업자의 입사일입니다(날짜 선택). |
| **퇴사일(quitDate)** | 작업자의 퇴사일입니다. 재직 중이면 비워둡니다. |
| **사용(useYn)** | 조회·선택·사용 가능 여부입니다. `Y`만 활성, 사용 중지는 `N`. |

## ④ 입력 폼 — 비고

| 컬럼 | 역할 / 의미 |
|------|------|
| **비고(remark)** | 작업자 관리 참고사항(메모)입니다. |

## 사용 순서
1. 상단 **작업자 추가**(또는 행 ✏️)로 오른쪽 슬라이드 폼을 엽니다.
2. 필요 시 상단에서 **사진을 업로드**합니다.
3. **기본정보**에서 작업자코드(사번)·작업자명을 입력합니다(필수). 부서·직급·연락처·QR코드 등을 채웁니다.
4. **근무정보**에서 입사일·퇴사일·사용여부를 설정합니다.
5. 헤더의 **저장**을 누릅니다. 닫으려면 **취소**를 누릅니다.
6. 검색창에 코드/이름을 입력하거나 사용여부 필터로 목록을 좁힐 수 있습니다.

## 입력 규칙 · 검증
- **작업자코드(사번)**와 **작업자명**은 필수입니다. 둘 중 하나라도 비어 있으면 저장 버튼이 비활성화되고 "필수 항목을 입력하세요." 안내가 뜹니다.
- **작업자코드는 중복될 수 없습니다.** 이미 존재하는 코드로 추가하면 "이미 존재하는 작업자 코드입니다" 오류가 표시됩니다.
- 작업자코드는 수정 화면에서 **변경할 수 없습니다**(잠금). 코드를 바꾸려면 새로 등록해야 합니다.
- 사진은 **이미지 파일만** 업로드할 수 있습니다.
- 입사일/퇴사일은 `YYYY-MM-DD` 날짜 형식입니다.

## 자주 묻는 질문
- **Q.** 작업자코드와 QR코드는 무엇이 다른가요?
  **A.** 작업자코드(사번)는 작업자를 구분하는 기본 식별자이고, QR코드는 키오스크/PDA에서 스캔으로 식별하기 위한 별도 코드입니다. QR코드를 비워두면 사번으로도 스캔 식별이 됩니다.
- **Q.** 퇴사한 작업자는 어떻게 처리하나요?
  **A.** 퇴사일을 입력하고 사용여부를 `N`(미사용)으로 바꾸면 선택 목록에서 제외됩니다. 이력 보존을 위해 삭제보다 미사용 처리를 권장합니다.
- **Q.** 작업자가 키오스크/선택 목록에 안 보여요.
  **A.** 사용여부가 `N`이면 보이지 않습니다. `Y`로 활성화하세요.
- **Q.** 작업자코드를 잘못 입력했어요.
  **A.** 코드는 수정할 수 없으므로 해당 작업자를 삭제하고 올바른 코드로 다시 등록하세요(생산실적 연결이 있으면 미사용 처리를 검토).

## 관련 화면
- 생산실적·키오스크 화면에서 이 작업자 정보를 작업자 선택·QR 스캔으로 참조합니다.
