# 팔레트출하

## 화면 목적
적재가 완료되어 출하 가능 상태(CLOSED)인 팔레트를 바코드 스캔 또는 입력으로 조회하여, 최종 출하 확정(SHIPPED)을 처리합니다. 팔레트별로 포함된 박스 목록을 확인할 수 있습니다.

## 화면 구성
- **좌측 — 출하지시 목록**: 출하확정(CONFIRMED) 상태의 출하지시 목록을 표시합니다.
- **중앙 — 팔레트 목록**: 선택한 출하지시의 팔레트 목록을 DataGrid로 표시합니다.
- **우측 — 박스 상세**: 선택한 팔레트에 포함된 박스 목록과 상태를 표시합니다.

---

## 사용 순서
1. 좌측 출하지시 목록에서 처리할 출하지시를 선택합니다.
2. 중앙 팔레트 목록에서 출하할 팔레트를 확인합니다.
3. **팔레트 스캔** 입력란에 팔레트 바코드를 스캔하거나 수동 입력합니다.
4. 입력한 팔레트가 유효하면 목록에 추가되고, **출하확정** 버튼이 활성화됩니다.
5. 출하확정 버튼을 클릭하여 최종 출하 처리합니다.

## 중앙 — 팔레트 DataGrid 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **팔레트번호(PALLET_NO)** | 팔레트의 고유 식별 번호입니다. |
| **박스 수(BOX_COUNT)** | 팔레트에 적재된 박스 개수입니다. |
| **총 수량(TOTAL_QTY)** | 팔레트의 총 제품 수량입니다. |
| **상태(STATUS)** | 팔레트의 현재 상태입니다(OPEN / CLOSED / LOADED / SHIPPED). |
| **적재완료시각(CLOSE_TIME)** | 팔레트 적재가 완료된 시각입니다. |
| **출하시각(SHIPPED_TIME)** | 출하 확정 시각입니다. |

## 우측 — 박스 상세 컬럼

| 컬럼 | 역할 / 의미 |
|------|------|
| **박스번호(BOX_NO)** | 박스의 고유 식별 번호입니다. |
| **품목코드** | 박스에 포함된 제품의 품목 코드입니다. |
| **품목명** | 제품명입니다. |
| **수량(QTY)** | 박스당 제품 수량입니다. |
| **상태(STATUS)** | 박스 상태입니다. |

## 입력 규칙
- 팔레트 번호는 바코드 스캐너로 스캔하거나 직접 입력할 수 있습니다.
- **CLOSED**(적재완료) 상태의 팔레트만 출하 가능합니다.
- 출하확정 시 팔레트와 포함된 모든 박스의 상태가 **SHIPPED**로 변경되며, 재고가 차감됩니다.

## 상태 흐름
```
OPEN(팔레트생성) → CLOSED(적재완료,출하가능) → SHIPPED(출하확정)
```
