# 정기점검

## 화면 목적
주기(월/분기/반기/연) 단위로 수행하는 설비 정기점검(PERIODIC)의 결과를 등록하고 관리합니다. 일상점검과 동일한 좌/우 분할 레이아웃으로, PERIODIC 유형의 점검 데이터를 처리합니다.

## 화면 구성
- **좌측 — 설비 목록**: 정기점검 대상 설비를 설비유형별로 그룹화하여 표시합니다.
- **우측 — 점검 입력 패널**: 선택한 설비의 PERIODIC 점검항목 목록과 판정 입력 폼을 표시합니다.

## 점검 입력 패널 구성

| 항목 | 역할 / 의미 |
|------|------|
| **점검일** | 정기점검 수행일자입니다. |
| **점검자 (필수)** | 점검 수행 작업자를 작업자 마스터에서 선택합니다. |
| **시작 시각** | 점검 시작 시각입니다. |
| **점검항목 목록** | 해당 설비에 연결된 PERIODIC 점검항목입니다. |
| **판정 입력** | 각 항목의 OK(합격) / NG(불합격) 판정 또는 측정값을 입력합니다. |
| **종합 판정** | 전체 항목의 판정 결과를 종합하여 자동 표시됩니다. |

## 사용 순서
1. 좌측 설비 목록에서 정기점검할 설비를 선택합니다.
2. 우측 패널에서 점검일과 점검자를 확인/선택합니다.
3. 각 점검항목별로 결과를 입력합니다(판정형 OK/NG, 측정형 수치 입력).
4. 불합격(NG) 항목이 있으면 불량 내용을 입력합니다.
5. **저장 (PASS)** 또는 **저장 (NG)** 버튼으로 저장합니다.
