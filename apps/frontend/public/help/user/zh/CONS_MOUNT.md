# 消耗品安装管理

## 屏幕用途

本屏幕用于将生产设备的**消耗品（模具、夹具、工具等）****安装**到设备上或**解除安装**，必要时**切换为维修状态**，或在维修完成后**返回仓库**。也可以查看单个消耗品的安装/解除历史。

> 本屏幕管理消耗品的**物理位置/状态**。入库、出库或寿命请在[消耗品入库]、[消耗品出库]、[寿命现状]屏幕中分别管理。

## 屏幕构成

- **顶部标题栏**：屏幕标题和刷新按钮
- **主网格**：消耗品列表。每行根据状态显示相应的操作按钮。
- **操作弹窗**：执行安装/解除/维修转换/维修完成的弹出窗口
- **历史弹窗**：以表格形式查看所选消耗品的安装/解除历史

## ① 主网格列

| 列 | 作用 / 含义 |
|------|------|
| **管理** | 行单位操作按钮集合。根据状态显示**安装**、**解除**、**维修转换**、**维修完成**、**历史**按钮。 |
| **运营状态(operStatus)** | 消耗品的当前位置/状态。为`仓库`(WAREHOUSE) / `安装`(MOUNTED) / `维修`(REPAIR) 之一。 |
| **消耗品代码(consumableCode)** | 用于识别消耗品的唯一代码。 |
| **消耗品名称(consumableName)** | 消耗品的名称。 |
| **分类(category)** | 消耗品种类。如模具(MOLD)、夹具(JIG)、工具(TOOL)等，对应公共代码 `CONSUMABLE_CATEGORY` 的值。 |
| **安装设备(mountedEquip)** | 当前消耗品所安装的设备代码。未安装状态下显示为 `-`。 |
| **寿命状态(status)** | 消耗品的寿命状态。为 `NORMAL`(正常) / `WARNING`(警告) / `REPLACE`(需更换) 之一。 |
| **寿命(lifeProgress)** | 以条形图显示当前使用次数相对于预期寿命的比例。80% 以上为黄色，100% 以上为红色高亮。 |
| **保管位置(location)** | 消耗品的基本保管场所。 |

## ② 各状态下可执行的操作

| 当前状态 | 可执行操作 | 说明 |
|------|------|------|
| **仓库** (WAREHOUSE) | 安装、维修转换 | 选择设备进行安装，或切换为维修状态。 |
| **安装** (MOUNTED) | 解除、维修转换 | 从当前安装的设备上解除，或自动解除后切换为维修状态。 |
| **维修** (REPAIR) | 维修完成 | 将维修完成的消耗品恢复为仓库状态。 |

## ③ 操作弹窗输入项

| 项目 | 说明 |
|------|------|
| **目标设备** | 安装时必填。通过设备选择组件选择要安装的设备。 |
| **备注** | 可选。记录安装/解除/维修原因等。最多 500 字。 |

## 使用顺序

1. 通过顶部搜索框或**分类/运营状态筛选器**查找目标消耗品。
2. 点击与行状态对应的操作按钮。
   - 安装：`安装` 按钮 → 选择目标设备 → 保存
   - 解除：`解除` 按钮 → 输入备注（可选） → 保存
   - 维修：`维修转换` 按钮 → 输入备注（可选） → 保存
   - 维修完成：`维修完成` 按钮 → 保存
3. 变更后列表会自动刷新。
4. 查看特定消耗品的历史时，点击 `历史` 按钮。

## 输入规则 / 校验

- **安装**必须选择目标设备。未选择时保存按钮处于禁用状态。
- 已安装的消耗品无法再次安装。
- 未处于安装状态的消耗品无法解除。
- 未处于维修状态的消耗品无法执行维修完成处理。
- 维修转换时若正在安装，会先自动记录解除历史，然后切换为维修状态。

## 常见问题

- **Q.** 看不到安装按钮。
  **A.** 仅对运营状态为`仓库`(WAREHOUSE)的消耗品显示安装按钮。请检查筛选器。
- **Q.** 维修后如何返回仓库？
  **A.** 点击 `维修完成` 按钮。此后运营状态将恢复为`仓库`(WAREHOUSE)。
- **Q.** 从设备上解除后提示寿命已超期。
  **A.** 解除本身可以执行。寿命状态请在[消耗品主数据]中确认预期寿命、警告阈值后进行更换或重置。
- **Q.** 历史中看不到作业人员。
  **A.** 当前屏幕没有作业人员输入栏，API 调用时会记录登录用户信息。如有需要请确认后端逻辑。
