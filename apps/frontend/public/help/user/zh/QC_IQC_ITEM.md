# 检验项目主数据

## 画面目的
集中注册·管理进货检验(IQC)中使用的**全局检验项目**的画面。此处创建的项目作为**公用项目池**，在[物料IQC项目管理](/master/iqc-part-spec)中按物料选择使用。即同一检验项目（例如：外观、尺寸）无需为每个物料新建，一次注册即可重复使用。

## 画面构成
- **左侧 — 检验项目列表（网格）**：以表格显示已登记的检验项目。提供搜索·判定方法过滤。
- **右侧 — 添加/修改面板**：点击`添加检验项目`或行的修改(✎)时从右侧滑动打开。

## ① 检验项目列表（网格）列

| 列 | 作用 / 含义 |
|---|---|
| **修改 / 删除** | ✎ 打开修改面板，🗑 删除（需确认）。在其他画面使用中的项目请注意删除。 |
| **项目代码(inspItemCode)** | 检验项目的唯一标识代码。在物料基准中以此代码引用项目。注册后**不可更改**（键）。 |
| **检验项目(inspItemName)** | 检验项目名称。例如：外观、尺寸、拉伸强度、颜色。实际检验人员看到的名称。 |
| **判定方法(judgeMethod)** | 如何判定检验结果。**目视(VISUAL)**=肉眼判断合格/不合格，**计量(MEASURE)**=通过测量值与规格比较。以标签颜色区分。 |
| **单位(unit)** | 计量项目的测量单位（mm、g等）。从公共代码`UNIT_TYPE`中选择。目视项目通常留空。 |

> 上方工具栏：**搜索**（代码·项目名称）、**判定方法过滤**（目视/计量）、**刷新**、**添加检验项目**。

## ② 检验项目添加 / 修改列

点击`添加检验项目`时右侧面板显示以下输入项目。

| 列 | 作用 / 含义 |
|---|---|
| **项目代码** | 必填。项目的唯一代码。**修改时锁定不可更改**（键）。仅新登记时输入。 |
| **判定方法** | 必选。**目视**或**计量**。选择计量时建议同时指定单位。 |
| **检验项目** | 必填。检验项目名称。 |
| **单位** | 可选。从公共代码`UNIT_TYPE`列表中选择（mm、g、kg等）。目视项目可留空。 |

## 使用顺序
1. 点击`添加检验项目`。
2. 输入**项目代码**·**检验项目**并选择**判定方法**。
3. 如为计量项目选择**单位**。
4. 点击`添加`保存。
5. 已登记的项目在[物料IQC项目管理](/master/iqc-part-spec)中关联到物料，设置规格（上下限、判定基准）。

## 输入规则 / 校验
- **项目代码**和**检验项目**为必填项。任一为空时保存按钮禁用。
- **项目代码注册后不可更改**。输入错误时请删除后重新登记。
- 搜索按**项目代码·检验项目名称**进行部分匹配查找。

## 常见问题
- **想更改项目代码** — 代码为键，不可修改。请删除后用新代码重新登记（如已关联物料，需重新设置关联）。
- **上下限规格(LSL/USL)或判定基准在哪里设置？** — 本画面仅定义**公用项目**。各物料不同的规格·判定基准·抽样方式在[物料IQC项目管理](/master/iqc-part-spec)中按物料设置。
- **目视与计量的区别？** — 目视是用眼睛判断合格/不合格（例如：外观瑕疵），计量是用测量仪器读取数值与规格比较（例如：尺寸mm）。
