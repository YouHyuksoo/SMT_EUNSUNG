# 物料主数据

## 画面目的
注册和管理MES中处理的**所有物料（原材料·半成品·成品·消耗品）的基本信息**。此处登记的物料是BOM·生产·物料收发·进货检验·库存的基础。

## 画面构成
- **左侧（列表）**：物料网格。上方提供物料类型·使用与否过滤、品番/物料名称搜索、**ERP同步**、**添加物料**。
- **右侧（滑动面板）**：点击行的✏️（修改）或"添加物料"时打开的注册/修改表单。
- IQC对象(`iqcYn=Y`)的物料可单独设置**IQC检验基准**。

---

## ① 标识信息

| 列 | 作用 / 含义 |
|------|------|
| **物料代码(itemCode)** | MES内部标识物料的**唯一代码**。原则上注册后不予更改（BOM·库存·实绩通过此代码关联）。 |
| **品番(itemNo)** | 图纸·ERP·客户文档中使用的品番。业务中最常使用的编号。 |
| **物料名称(itemName)** | 现场识别的物料名称。 |
| **客户品番(custPartNo)** | 客户使用的品番（用于出货·标签匹配）。 |
| **Rev(rev)** | 物料图纸/规格的版本。用于区分规格变更历史。 |
| **标记文字(markingText)** | 传递给标签·打标设备的显示文字。 |

## ② 分类

| 列 | 作用 / 含义 |
|------|------|
| **物料类型(itemType)** | MES流程中的大类。原材料(RAW_MATERIAL)·半成品(SEMI_PRODUCT)·成品(FINISHED)·消耗品(CONSUMABLE)。物料收发·生产处理方式据此值变化。 |
| **物料组(productType)** | 产品系列/物料组代码（公共代码PRODUCT_TYPE）。用于回路规格等参考。 |
| **车型/型号(modelName)** | 区分车辆型号或车型的管理特性。 |

## ③ 规格

| 列 | 作用 / 含义 |
|------|------|
| **规格(spec)** | 物料规格·规范·尺寸等辅助说明。 |
| **颜色(color)** | 线色等物料颜色信息。 |
| **单位(unit)** | 数量解释的基本单位（EA等，公共代码UNIT_TYPE）。库存·发放数量以此单位为准。 |

## ④ 进货检验(IQC)

| 列 | 作用 / 含义 |
|------|------|
| **IQC与否(iqcYn)** | 入库时是否为进货检验对象。`Y`时入库LOT适用IQC。 |
| **检验方式(inspectMethod)** | IQC检验/免检等应用方式（公共代码IQC_INSPECT_METHOD）。 |
| **默认样品数(sampleQty)** | IQC检验时使用的默认样品数量（与AQL计算的样本数无关）。 |
| **AQL政策(iqcAqlPolicyCode)** | 适用于该物料的**AQL政策**。入库LOT检验时据此政策自动计算样本数·合格(Ac)·不合格(Re)。政策在[AQL基准管理]中登记，留空则不应用AQL自动判定。 |

## ⑤ 数量 · 包装

| 列 | 作用 / 含义 |
|------|------|
| **箱数量(boxQty)** | 每箱装入的标准数量（包装·箱标签基准）。 |
| **最小包装数量(minPackQty)** | 物料发放时可作为最小单位处理的数量。 |
| **LOT构成单位(lotUnitQty)** | 将生产工序品以批次单位处理的标准数量。 |
| **托盘构成单位(packUnit)** | 托盘或上级包装单位构成基准。 |
| **安全库存(safetyStock)** | 判断库存不足的基准数量。低于此值视为不足。 |

## ⑥ 有效期

| 列 | 作用 / 含义 |
|------|------|
| **有效期(expiryDate)** | 基于入库或制造的有效期**天数**。 |
| **有效期延长(expiryExtDays)** | 品质判断后可延长有效期的**最大天数**。 |

## ⑦ 其他

| 列 | 作用 / 含义 |
|------|------|
| **装载位置(storageLocation)** | 按物料固定的默认装载位置。 |
| **照片(imageUrl)** | 物料照片。在表单中上传，也显示在列表缩略图中。 |
| **使用与否(useYn)** | 仅`Y`时为查询·选择·使用对象。停用为`N`。 |
| **备注(remark)** | 物料管理参考事项。 |

## ERP同步
上方**ERP同步**按钮从ERP获取物料主数据并在MES中**新增/更新**。执行前会弹出确认模态框。一次可能同步数万条记录，请仅在有意操作时执行。

## 使用顺序
1. 通过上方**添加物料**（或行✏️）打开滑动表单。
2. 输入标识信息（物料代码·品番·物料名称）和分类（物料类型）。
3. 填写必要项目（规格·IQC·包装·有效期等）后保存。
4. IQC对象时关联**IQC检验基准**和物料的**AQL政策**。

## 常见问题
- **问：** 物料代码和品番有何不同？
  **答：** 物料代码是MES内部标识符（建议不变），品番是图纸/ERP/客户文档中使用的编号。
- **问：** AQL政策留空会怎样？
  **答：** 入库检验中不应用AQL自动判定（免检或手动）。
- **问：** ERP同步误添加了物料。
  **答：** 通过ERP-IF导入的物料可由运营人员整理（参照运营指南）。
