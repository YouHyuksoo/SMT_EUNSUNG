# 产品生产计划

## 画面目的
以计划形式登记和管理成品·半成品**何时（哪个月）生产多少**的画面。
计划按**月度(YYYY-MM)**单位创建，一条记录表示"本月生产该物料多少个"。计划**确认**后，可基于该计划**发行工单**以指示实际生产。

> 生产计划按月而非按日。即使同月·同一物料也可创建多条计划，每条计划分配计划编号(PLAN_NO)。

## 画面构成
- **上方 — 工具按钮**：刷新 / 模板下载 / Excel上传 / ERP接口 / 自动编排 / 添加计划
- **上方过滤（网格上方）**：开始日期·结束日期（计划月范围）、搜索词、物料类型、状态
- **中间 — 计划列表（网格）**：逐行显示已登记的生产计划。DRAFT行左侧显示修改·删除图标。
- **右侧 — 登记/修改面板**：点击'添加计划'或修改图标时从右侧滑动打开。

## 状态流程
```
DRAFT(草稿) ──确认──▶ CONFIRMED(确认) ──关闭──▶ CLOSED(关闭)
                 ◀─取消确认─┘
仅DRAFT可修改·删除 / 仅CONFIRMED可发行工单
```

## ① 计划列表列

| 列 | 作用 / 含义 |
|------|------|
| **（修改/删除）** | 仅**草稿(DRAFT)**行显示的左侧图标。铅笔=打开修改面板，垃圾桶=删除。确认后状态不显示（不可修改·删除）。 |
| **计划编号(planNo)** | 标识计划的唯一编号。格式为`PP-YYYYMM-NNN`（例如：`PP-202606-001`）。登记时自动生成，不可更改。 |
| **计划月(planMonth)** | 该计划适用的**生产目标月份**（`YYYY-MM`）。不含日期，在发行工单时指定具体计划日期。 |
| **操作(actions)** | 根据状态切换按钮。DRAFT显示**确认**，CONFIRMED显示**发行工单**·**取消确认**按钮。剩余数量为0时发行按钮禁用。 |
| **物料类型(itemType)** | `FINISHED`（成品）或`SEMI_PRODUCT`（半成品）。区分生产什么类型的计划。 |
| **物料代码(partCode)** | 生产目标物料的代码。从物料主数据选择的值。 |
| **物料名称(partName)** | 与物料代码关联的物料名称（基于物料主数据显示）。 |
| **计划数量(planQty)** | 本月计划生产的**目标数量**。作为工单发行的上限基准。 |
| **发行数量(orderQty)** | 该计划**至今已发行工单的累计数量**。每次发行时自动增加。 |
| **发行率(issueRate)** | 以条形图和%显示`发行数量÷计划数量`。100%表示已全部发行计划数量，条形变为绿色。 |
| **线别(lineCode)** | 生产进行的基本生产线。发行工单时作为默认值继承，可更改。 |
| **客户(customer)** | 与该计划关联的客户（用于接单对应等参考）。 |
| **优先级(priority)** | 计划处理优先级（1~10，默认5）。**数值越小越优先**处理，列表也按优先级升序排列。 |
| **状态(status)** | 计划的进行阶段。**DRAFT（草稿）**=可修改·删除，**CONFIRMED（确认）**=不可修改·删除·可发行工单，**CLOSED（关闭）**=结束。表头的问号(?)有状态转换说明。 |

> 上方过滤的**开始日期·结束日期**是计划月范围过滤。仅比较输入日期的**月份（前7位）**，因此同一月中输入任意日期均包含该月。

## ② 登记/修改面板输入字段

通过'添加计划'或修改图标打开的右侧面板输入项目。**修改模式下计划月·物料代码·物料类型不可更改**（锁定）。

### 基本信息
| 字段 | 作用 / 含义 |
|------|------|
| **计划月(planMonth)** | 选择生产目标月（`YYYY-MM`）。是计划编号生成的基准。修改时锁定。 |
| **物料类型(itemType)** | 选择成品/半成品。也用作物料搜索的过滤。修改时锁定。 |
| **物料代码(itemCode)** | 通过放大镜按钮从物料主数据中选择生产物料。不可直接输入（仅搜索选择）。必填项。 |
| **计划数量(planQty)** | 目标生产数量。必须**≥1**才能保存（≤0时保存按钮禁用）。 |
| **优先级(priority)** | 1~10之间的值（默认5）。数值越小越优先处理。 |

### 详细信息
| 字段 | 作用 / 含义 |
|------|------|
| **客户(customer)** | 从客户（客户）列表中选择（可选输入）。 |
| **线别(lineCode)** | 选择默认生产线（可选输入）。发行工单时作为默认值使用。 |
| **备注(remark)** | 自由备注（可选输入）。 |

## ③ 工单发行(IssueJobOrder)输入字段

在列表中点击**CONFIRMED**计划的**发行工单**按钮时打开的模态框。上方摘要显示计划编号·物料·计划数量·发行数量·**剩余数量**。

| 字段 | 作用 / 含义 |
|------|------|
| **发行数量(issueQty)** | 本次作为工单下发的数量。必须**≤剩余数量(计划数量−发行数量)**，超出时显示警告并阻止发行。必填项。 |
| **计划日期(planDate)** | 工单的具体生产预定日期（可选）。计划为月度单位，但工单包含具体日期。 |
| **线别(lineCode)** | 执行工单的线别。继承计划的线别作为默认值，可更改。 |
| **优先级(priority)** | 工单优先级（1~10）。继承计划值。 |
| **BOM半成品自动生成(autoCreateChildren)** | 勾选时将按该物料BOM下级的**半成品(SEMI_PRODUCT)**工单自动全层级生成。需要同时生产半成品时使用。 |
| **备注(remark)** | 工单备注（未输入时记录为`(计划编号)发行`）。 |

> 发行后将创建工单（状态WAITING），计划的**发行数量自动增加所发数量**。剩余数量全部发行后发行率达100%。

## ④ 自动编排(MPS)输入项目

通过上方**自动编排**按钮打开的模态框。基于接单（客户订单）一次性以DRAFT形式生成计划。

| 项目 | 作用 / 含义 |
|------|------|
| **目标月(month)** | 要生成计划的月份（默认=下月）。更改月份后交付期间自动设置为该月1日~月末。 |
| **交付开始日期·结束日期** | 作为目标的接单交付范围。 |
| **客户(customer)** | 可缩小到特定客户的接单（不选=全部）。 |
| **预览(preview)** | 以表格查询符合条件的接单。每行有复选框，可选择要编排的项目（显示订单编号·交付日期·物料·客户·订购量）。 |
| **执行编排** | 将所选接单项目生成为生产计划DRAFT。执行时该月的**既有DRAFT将删除后重新编排**，因此会弹出确认模态框。 |

---

## 使用顺序
1. **添加计划**（或自动编排/Excel上传）登记计划 → 状态**DRAFT**。
2. 内容确认后点击列表的**确认**按钮转为**CONFIRMED**（此后不可修改·删除）。
3. 在CONFIRMED计划中点击**发行工单**按钮，在剩余数量范围内发行工单。
4. 每次发行时发行数量·发行率更新。不再处理的计划可关闭(CLOSED)。
5. 误确认时通过**取消确认**恢复为DRAFT后修改。

## 输入规则 / 校验
- 物料代码为必填项，计划数量必须**≥1**才能保存。
- 仅**DRAFT状态**可修改·删除（CONFIRMED·CLOSED不可）。
- 工单仅在**CONFIRMED状态下**，且仅可发行**不超过剩余数量**的数量。
- 优先级范围为1~10。
- Excel上传需所有行通过校验（错误0条）才能激活上传按钮。

## 常见问题
- **问：** 计划已确认但数量填错了。
  **答：** 尚未发行工单时可通过**取消确认**恢复为DRAFT后修改，再重新确认。
- **问：** 发行按钮已禁用。
  **答：** 剩余数量（计划数量−发行数量）为0，或计划不是CONFIRMED状态时。
- **问：** 一个计划中能否分多次发行工单？
  **答：** 可以。只要剩余数量还有，可分多次发行（发行数量累计相加）。
- **问：** 执行自动编排后既有计划会消失吗？
  **答：** 仅同月的**DRAFT计划**会被重新编排替换。CONFIRMED·CLOSED计划不受影响。
