# 物料库存现状查询

## 画面目的
实时查询按仓库·物料持有的物料库存。以选项卡形式提供序列(LOT)单位的详细现状和按物料的分组合计，并可一目了然确认安全库存对比状态和有效期是否过期。

## 画面构成
- **两个选项卡**: **按序列详细**(个别LOT单位) / **按物料分组合计**(按物料汇总)
- **上方筛选**: 仓库选择下拉框 + 物料代码/物料名称搜索输入
- **DataGrid列表**: 根据所选选项卡显示不同的列构成

---

## ① 按序列详细选项卡列

| 列 | 作用 / 含义 |
|------|------|
| **物料代码(itemCode)** | 库存物料的代码。 |
| **物料名称(itemName)** | 库存物料的名称。 |
| **物料序列号(matUid)** | 个别LOT/序列号。同一物料若序列不同，则作为单独行管理。 |
| **仓库(warehouseName)** | 库存所在的仓库。 |
| **数量(qty)** | 当前仓库中保管的库存总数量。 |
| **安全库存(safetyStock)** | 该物料的安全库存基准数量(在物料主数据中设置)。 |
| **状态(stockLevel)** | 对比实际数量是否满足安全库存。**不足**(红色) /**注意**(黄色) /**正常**(绿色)。 |
| **生产日期(manufactureDate)** | 该LOT的生产日期。 |
| **经过天数(elapsedDays)** | 自生产日起经过的天数。 |
| **剩余有效期(remainingDays)** | 距保质期到期的剩余天数。**负数表示已过期**。 |
| **有效期状态(shelfLifeStatus)** | 有效期状态。**已过期**(红色) /**即将到期**(黄色) /**正常**(绿色)。 |

---

## ② 按物料分组合计选项卡列

| 列 | 作用 / 含义 |
|------|------|
| **物料代码(itemCode)** | 物料代码。 |
| **物料名称(itemName)** | 物料名称。 |
| **总库存数量(totalQty)** | 该物料所有仓库的库存总量合计。 |
| **安全库存(safetyStock)** | 物料的安全库存基准。 |
| **状态(stockLevel)** | 对比总库存是否满足安全库存。 |
| **序列数(serialCount)** | 该物料持有的LOT/序列个数。 |
| **仓库数(warehouseCount)** | 该物料分散保管的仓库个数。 |

---

## 使用顺序
1. 在上方筛选器中输入查询的**仓库**和**搜索词**(物料代码/物料名称)。
2. 在**按序列详细**选项卡中确认个别LOT单位的库存现状。
3. 在**按物料分组合计**选项卡中确认按物料汇总的现状。
4. 安全库存不足(红色)或有效期已过期(红色)的项可能需要立即采取措施。
5. 通过右上角**导出**可将当前查询结果下载为Excel文件。
