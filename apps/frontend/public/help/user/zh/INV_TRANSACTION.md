# 物料收发历史查询

## 画面目的
查询所有物料收发(入库·出库·移动·调整·废弃等)事务历史。可按日期·类型追踪所有库存变动明细。

## 画面构成
- **DataGrid列表**: 显示事务编号、类型、日期、仓库、物料、数量、状态等13个列
- **上方筛选**: 交易类型选择 + 日期范围 + LOT编号搜索

---

## ① DataGrid列表列

| 列 | 作用 / 含义 |
|------|------|
| **事务编号(transNo)** | 标识各库存变动明细的唯一编号。 |
| **类型(transType)** | 库存变动类型。包括**入库/出库/移动/调整/废弃**等，以色标区分。 |
| **处理日(transDate)** | 事务处理的日期时间。 |
| **出库仓库(fromWarehouse)** | 库存出去的仓库(出库/移动时)。 |
| **入库仓库(toWarehouse)** | 库存进入的仓库(入库/移动时)。 |
| **物料代码(itemCode)** | 变动物料的代码。 |
| **物料名称(itemName)** | 变动物料的名称。 |
| **LOT编号(matUid)** | 与该事务关联的LOT/序列号。 |
| **数量(qty)** | 变动数量。**正数(+) = 库存增加，负数(-) = 库存减少**。 |
| **状态(status)** | 事务状态。`DONE`(正常处理，绿色) / `CANCELED`(已取消，红色)。 |
| **原交易(original)** | 取消时的原事务编号(仅取消项显示)。 |
| **备注(remark)** | 事务的附加备注。 |

## 使用顺序
1. 在上方筛选中输入交易类型、查询期间、LOT编号。
2. 在查询结果中确认特定事务的详细信息。
3. 可使用列过滤·排序功能查找所需数据。
4. 通过**导出**可将结果下载为Excel。

## 主要交易类型
| 类型 | 含义 | 数量符号 |
|------|------|----------|
| RECEIVE | 物料入库 | + |
| MAT_OUT | 物料出库 | - |
| ADJUST_IN | 调整增加 | + |
| ADJUST_OUT | 调整减少 | - |
| TRANSFER | 仓库间移动 | 出库仓库 -, 入库仓库 + |
| LOT_SPLIT_IN/OUT | LOT分割 | 分割 -, 生成 + |
| SCRAP | 废弃 | - |
| MISC_IN | 其他入库 | + |
| PROD_CONSUME | 生产消耗 | - |
