# 物料分割

## 画面目的
将入库完成的物料LOT（序列号）**分成两个片段**的画面。当一个序列号中的数量需要按不同工单·仓库·用途分离时使用。分割后原序列号废弃(SPLIT)，**分割数量片段**和**余量片段**分别发放为新的物料序列号。

> 分割 = 1个序列号 → 2个序列号。原序列号不保留而被废弃，两个片段均获得新序列号和新标签。

## 分割流程
```
入库完成LOT（序列号1个，数量100）
  → 输入分割数量30
  → 原序列号废弃(SPLIT，库存0)
  → 新序列号A(30) + 新序列号B(70) 发放
  → 输出两个片段的物料标签
```

## 画面构成
- **上方统计卡片**：显示可分割的LOT总数和总数量。
- **主网格（可分割LOT列表）**：仅显示入库完成 + 库存 > 1 + 正常(NORMAL)状态的LOT。上方提供LOT编号·物料名称搜索。
- **分割按钮（行左侧）**：打开该LOT的分割输入模态框。
- **分割模态框**：输入分割数量后显示分成两个片段的预览。
- **标签预览模态框**：分割成功后输出2个新序列号的标签。

---

## ① 可分割LOT网格列

| 列 | 作用 / 含义 |
|------|------|
| **（操作）分割** | 打开该LOT分割模态框的按钮。列表中仅显示入库完成·库存>1·正常状态的LOT，因此显示的行均为可分割对象。 |
| **物料序列号(matUid)** | 物料LOT的唯一标识号（序列号）。印在标签条码上，是分割的原序列号。 |
| **物料代码(itemCode)** | 物料的MES内部物料代码。 |
| **物料名称(itemName)** | 物料名称。 |
| **当前数量(qty)** | 该序列号当前持有的库存数量（含单位显示）。分割数量必须小于此值。 |
| **供应处(vendor)** | 交付该物料的客户（供应商）名称。仅有代码时直接显示代码。 |

> 列表条件：**入库完成**（已完成入库处理的LOT）+ **当前数量≥2** + **预订数量=0** + **状态NORMAL**。有出库历史或保留(HOLD)状态的LOT不显示。

---

## ② 分割模态框（点击分割按钮时）

将所选LOT分为两个片段的输入画面。

| 输入项目 | 作用 / 含义 |
|------|------|
| **原始信息（上方摘要）** | 摘要显示所选物料序列号/物料代码/物料名称/当前数量（只读）。 |
| **分割数量 \*** | 从原始中分离出的数量。需**≥1，且小于当前数量**。剩余数量自动成为余量片段。 |
| **分割预览** | 输入分割数量后以`{分割数量} + {余量}`形式预览两个片段如何分配。 |
| **备注** | 自由输入分割原因或备注（可选，200字以内）。 |

保存后原序列号废弃，创建2个新序列号，然后打开**标签预览模态框**。

---

## ③ 标签预览模态框（分割成功后）

预览并输出分割后新发放的2个序列号标签。

| 项目 | 作用 / 含义 |
|------|------|
| **标签预览区域** | 列出新创建的两个片段序列号的标签。各标签印有新物料序列号、物料代码、数量等信息。 |
| **制造商显示** | 两个片段均继承原LOT的制造商进行显示·打印。 |
| **标签输出（打印按钮）** | 通过本地Print Agent发送到标签打印机。 |

---

## 使用顺序

1. 在上方搜索中按物料序列号(LOT编号)或物料名称查找要分割的LOT。
2. 点击该行的**分割**按钮。
3. 在模态框中输入**分割数量**（小于当前数量）。
4. 通过预览确认`分割数量 + 余量`是否符合预期。
5. 如有需要输入**备注**后点击**分割**。
6. 在标签预览中确认2个新序列号的标签后**打印标签**。
7. 将打印的标签贴到各物料捆包上（原标签废弃）。

---

## 输入规则 / 校验

- 分割数量必须**≥1，且小于当前数量**。等于或大于当前数量时分割按钮禁用。
- 以下LOT不可分割（不显示在列表中或保存时被阻止）：
  - **未入库完成**的LOT（需先完成入库处理）
  - **状态不是NORMAL**的LOT（HOLD/SPLIT/MERGED/DEPLETED等）
  - **有预订数量**的LOT（需先清理预订）
  - **有出库历史**的LOT（需先清理出库）
  - **物料主数据中设为不可分割(isSplittable = N)**的物料
- 分割不可撤销。原序列号废弃，替换为2个新序列号。

## 常见问题

- **问：** 分割后原序列号会怎样？
  **答：** 原序列号被废弃(SPLIT)，库存变为0，不再可用。分割数量片段和余量片段分别**发放为新序列号**。

- **问：** 能否分割成三个以上？
  **答：** 一次分割最多分成两个片段。如需更细的分割，可对分割结果的新序列号再次分割（可重复分割）。

- **问：** 分割后物料的入库·检验历史可追溯吗？
  **答：** 可以。新片段继承原LOT的到货编号·制造商·原始序列号(origin)信息，可向上追溯。

- **问：** 列表中看不到LOT。
  **答：** 可能原因：入库未完成、当前数量≤1、有预订/出库历史、状态不是NORMAL。请在[原材料LOT情况]中确认状态。

- **问：** 标签无法打印。
  **答：** 请确认PC上Print Agent是否正在运行。
