# 作业指导书管理

## 画面目的
按物料和工序**注册和管理作业指导(作业指导书)**。以文档形式管理现场作业人员需执行的作业顺序、注意事项、参考图片/文件。通过物料·工序·版本的组合标识文档，并可赋予版本以进行历史管理。

## 画面构成
- **主区域 — DataGrid列表**：显示已注册的作业指导书列表。提供列过滤·排序·导出功能。
- **右侧面板 (预览)**：点击行时在右侧显示作业指导书详细信息(标题·版本·附件·正文)。
- **右侧面板 (注册/修改)**：点击新增或修改按钮时在右侧打开表单面板。

---

## ① DataGrid列表列

| 列 | 作用 / 含义 |
|------|------|
| **物料代码(itemCode)** | 作业指导书所适用的目标物料代码。引用`ITEM_MASTERS.ITEM_CODE`。 |
| **物料名称(itemName)** | 目标物料名称。 |
| **工序代码(processCode)** | 作业指导书所属工序的代码。与物料代码·版本共同构成文档标识符(PK)。 |
| **文档标题(title)** | 作业指导书的标题。是现场区分文档的标准。 |
| **版本(revision)** | 文档的修订版本。新增时默认值为`A`，修订时按B、C……递增。 |
| **最后修改日(updatedAt)** | 文档最后修改的日期时间。 |
| **使用与否(useYn)** | `Y` = 使用中(激活)，`N` = 未使用(停用)。以绿色/灰色圆圈显示。 |

---

## ② 注册/修改表单字段

| 字段 | 作用 / 含义 |
|------|------|
| **物料代码(itemCode)** | 作业指导书适用的物料代码。注册后不可修改。 |
| **工序代码(processCode)** | 该物料内作业指导书所属的工序。注册后不可修改。 |
| **文档标题(title)** | 作业指导书的标题。必填项。 |
| **版本(revision)** | 文档版本。留空则保存为`A`，注册后不可修改。 |
| **附件(imageUrl)** | 作业指导书的附件图片/PDF/文档文件。可通过**拖放或文件选择**上传，也可**直接输入URL**。 |
| **内容(content)** | 作业指导书的正文，包括作业顺序、注意事项等。 |

---

## 使用顺序
1. 点击**新增**按钮注册新作业指导书(右侧面板)。
2. 输入物料代码·工序代码·文档标题，必要时上传附件并编写正文内容。
3. 在列表中点击行即可通过**预览**查看详细内容。
4. 在预览中点击**修改**按钮或网格的编辑图标更改内容。

## 输入规则 / 验证
- 物料代码·工序代码·文档标题为**必填项**。任意一项为空则保存按钮将停用。
- 同一`物料代码 + 工序代码 + 版本`组合不可重复注册。
- 附件支持图片、PDF、Office文档、TXT文件，**最大限制为10MB**。

## 常见问题
- **问：** 版本如何管理？**答：** 新增时默认为`A`。修改现有文档时可保持当前版本或以新版本(如B)编制。同一物料·工序可同时保有多个版本。
- **问：** 能否更改已注册的物料代码·工序代码？**答：** 注册后PK(物料代码·工序代码·版本)不可更改。需以新文档重新注册。
- **问：** 可以不上传文件仅输入URL吗？**答：** 可以，可直接输入外部图片/文档的URL。上传和URL任选其一即可。
