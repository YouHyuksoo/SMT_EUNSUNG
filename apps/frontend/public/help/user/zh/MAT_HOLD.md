# 物料库存保留管理

## 画面目的
按物料LOT(序列)单位管理**库存保留(停止使用)**或**解除保留**。当因质量问题或其他原因需暂停特定LOT的使用时使用。被保留的LOT不能用于出库·生产。

## 画面构成
- **上方 — 3个统计卡片**: 全部LOT / 保留中 / 正常
- **筛选区域**: 搜索词 + LOT状态(全部/正常/保留)选择
- **DataGrid列表**: 按LOT显示状态及保留/解除操作按钮
- **弹窗**: 保留或解除时输入原因

---

## ① DataGrid列表列

| 列 | 作用 / 含义 |
|------|------|
| **LOT编号(matUid)** | 保留目标的物料LOT/序列号。 |
| **物料代码(itemCode)** | LOT对应的物料代码。 |
| **物料名称(itemName)** | LOT对应的物料名称。 |
| **当前数量(qty)** | 该LOT的当前持有数量。 |
| **供应商(vendor)** | 供应该LOT的合作方(供应商)。 |
| **状态(status)** | LOT的当前状态。**HOLD**(红色) = 停止使用，**NORMAL**(绿色) = 正常。 |

---

## 使用顺序
1. **保留**: 在列表中点击要保留的LOT的🔒(锁定)图标。在弹窗中输入原因并确认。该LOT变为`HOLD`状态，出库/使用被阻止。
2. **解除保留**: 点击已保留LOT的🔓(解锁)图标。在弹窗中输入原因并确认。LOT恢复为`NORMAL`状态。
3. 可通过状态筛选集中查看**保留中**的LOT。
