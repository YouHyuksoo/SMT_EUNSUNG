# 实绩输入（Kiosk）

## 画面目的
在现场设备旁的**平板/PC**上，操作员直接输入生产实绩的集成画面。从工单选择到物料投入、作业指导书确认、生产数量/不良输入、自检，在一个画面中完成处理。（使用全屏无铬模式）

## 画面构成
```
┌──────────────────────────────────────────────────────────┐
│ 上方标题2行: ①设备ID·条码·设备日检  ②工单·操作员·生产实绩·操作员设备检查 │
├───────────────┬──────────────────────────┬───────────────┤
│ 左: BOM物料   │ 中: 作业指导书            │ 右: 良品条件  │
│   + 消耗性    │   下方3格:               │   + 作业历史  │
│   设备部件    │   自检 | 不良 | 实绩输入   │               │
└───────────────┴──────────────────────────┴───────────────┘
```

## ① 上方标题（选择·点检）
| 项目 | 作用 / 含义 |
|------|------|
| **设备ID/条码** | 选择操作的设备。以设备为基准关联工单·工序·物料。 |
| **设备日检** | 必须完成设备的日常点检才能进行作业（准备点检=互锁）。 |
| **工单** | 选择该设备上要执行的工单（生产对象·数量）。 |
| **操作员** | 选择登记实绩的操作员（可多人）。 |
| **操作员设备检查** | 操作员执行的设备点检（互锁对象）。 |
| **生产实绩（进度率）** | 显示相对于工单当前已登记的实绩/进度率。 |

> **互锁（准备点检）**：必须完成设备日检·操作员设备检查等必需点检后方可保存实绩。未完成则阻止进行。

## ② 左侧 — 物料投入
| 项目 | 作用 / 含义 |
|------|------|
| **BOM物料列表** | 所选工单（产品）的BOM基准投入物料列表。确认需求量与实际投入量。 |
| **物料扫描** | 扫描投入的物料序列号(matUid)进行投入处理。 |
| **消耗性设备部件** | 安装在设备上的消耗品（与产品BOM分开）。在下方区域扫描投入。 |

## ③ 中间 — 作业指导书 + 实绩/检查
| 项目 | 作用 / 含义 |
|------|------|
| **作业指导书** | 显示物料·工序别的作业指南（图片/PDF/文档）。点击可放大。 |
| **实绩输入** | 输入**良品数量(goodQty)**·**不良数量(defectQty)**·（必要时生产序列号）保存实绩。 |
| **不良** | 按**等级（致命/重/轻）**详细输入不良。输入的不良合计将随实绩的不良数量一起保存。 |
| **自检** | 操作员自检（参见下面流程）。 |

## ④ 右侧 — 良品条件 · 作业历史
| 项目 | 作用 / 含义 |
|------|------|
| **良品条件** | 认定为良品的标准说明。 |
| **作业历史** | 该作业中登记的历史实绩。 |

## 自检流程（首件·中件·末件）
| 时机 | 触发 | 含义 |
|------|------|------|
| **首件(FIRST)** | 首次保存实绩后**自动打开** | 生产初期品质确认 |
| **中件(MID)** | 点击面板按钮或**进度达60%时阻止** | 进行中品质确认 |
| **末件(LAST)** | 点击面板按钮 | 生产结束时确认 |

## 使用顺序
1. **选择设备** → 完成设备日检。
2. **选择工单·操作员** → 完成操作员设备检查（解除互锁）。
3. 在左侧**扫描投入物料/消耗品**。
4. 在中间确认作业指导书的同时作业。
5. **输入良品数量·不良数量后保存**（不良按等级详细输入）。
6. 按时机进行**自检**（首件/中件/末件）。

## 常见问题
- **问：** 实绩保存按钮无法点击。
  **答：** 可能准备点检（设备/操作员检查）未完成，或未选择工单·操作员。请确认互锁状态。
- **问：** 未进行中件检查，无法继续推进。
  **答：** 进度达60%时中件自检会阻止。完成自检即可解除。
- **问：** 直接输入不良数量与等级输入是否重复？
  **答：** 如有等级详细不良合计，则以该值优先（不重复统计）。
