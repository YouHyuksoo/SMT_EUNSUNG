# 消耗品出库

## 画面目的
扫描已入库消耗品的**单个UID(conUid)条码出库到现场**，或**取消出库(退回)**错误出库记录的画面。处理的出库/出库取消记录可以在下方历史表中按期间/类型查询。

> 消耗品入库时每个物品都分配了唯一UID(conUid)，实现**单个追踪**。出库也按UID单件处理，每次扫描扣减数量1（单个绑定出库）。

## 画面构成
- **上方（扫描面板）**：选择出库/出库取消模式并扫描UID条码的区域。
- **下方（历史表）**：出库/出库取消历史列表。使用上方工具栏按期间、搜索、类型过滤。
- 右上角**刷新**按钮重新加载列表。

---

## 出库/出库取消流程
同一UID根据状态流转：

```
入库完成(ACTIVE)  ──[出库]──▶  已出库(ISSUED)
       ▲                              │
       └──────[出库取消(退回)]─────────┘
```

- **出库**：仅可出库入库完成(ACTIVE)状态的UID。出库后状态变为**已出库(ISSUED)**，主库存数量减少1。
- **出库取消(退回)**：仅可取消已出库(ISSUED)状态的UID。取消后恢复为**入库完成(ACTIVE)**，库存数量恢复1。

## ① 扫描面板

| 项目 | 作用 / 含义 |
|------|------|
| **出库（模式按钮）** | 将扫描的UID进行出库处理的模式。默认模式。 |
| **出库退货（模式按钮）** | 取消扫描UID的出库（退回）的模式。选择时会显示"出库取消模式"提示。 |
| **UID输入框** | 扫描或直接输入消耗品UID条码。按Enter或确认按钮处理。处理后自动重新获取焦点，可连续扫描。 |
| **出库确认 / 取消确认（按钮）** | 根据当前模式切换标签。出库模式显示**出库确认**，出库退货模式显示**取消确认**。无输入时禁用。 |

## ② 出库历史表列

| 列 | 作用 / 含义 |
|------|------|
| **日期时间(createdAt)** | 出库/出库取消处理的时间。 |
| **消耗品代码(consumableCode)** | 已出库消耗品的主代码。 |
| **消耗品名称(consumableName)** | 已出库消耗品的名称。 |
| **UID(conUid)** | 已出库单个实例的唯一UID。标识哪个物理物品被领出。 |
| **类型(logType)** | **出库(OUT)**或**出库退货(OUT_RETURN)**。出库显示蓝色，出库退货显示紫色标签。 |
| **数量(qty)** | 处理数量。出库显示红色`-1`，出库取消(退货)显示绿色`+1`（用符号区分增减方向）。 |
| **线别(lineCode)** | 出库目标线别（如有记录）。 |
| **设备(equipCode)** | 出库目标设备（如有记录）。 |
| **备注(remark)** | 出库/取消时输入的备注或自动消息。 |

> 表上方工具栏：**开始日期·结束日期**（默认当天）、**搜索**（消耗品代码·消耗品名称）、**类型过滤**（全部类型/出库/出库退货）。右上角**导出**可将表格保存为文件。

## 使用顺序
1. 在上方扫描面板中选择**出库**（或退回错误出库时选择**出库退货**）模式。
2. 在UID输入框中**扫描消耗品UID条码**（或直接输入）。
3. 按Enter或点击**出库确认/取消确认**按钮。
4. 处理完成后输入框清空并重新获取焦点，可立即扫描下一个UID。
5. 在下方历史表中确认处理结果（按期间/类型过滤查询）。

## 输入规则 / 校验
- **出库**仅允许入库完成(ACTIVE)状态的UID。未入库(PENDING)、已出库(ISSUED)、已退回(RETURNED)状态将被拒绝。
- **出库取消（出库退货）**仅允许已出库(ISSUED)状态的UID。
- 扫描不存在的UID时显示"找不到UID"错误。
- 每次扫描处理数量1。需出库多个时请依次扫描各UID。

## 常见问题
- **问：** 扫描时出现"仅允许ACTIVE状态"错误。
  **答：** 该UID尚未入库确认或已出库。请在[消耗品入库](/consumables/receiving)中确认入库状态。
- **问：** 出库错误如何撤回？
  **答：** 在上方切换为**出库退货**模式，扫描同一UID后点击**取消确认**即可恢复为入库完成(ACTIVE)。
- **问：** 历史表中看不到刚刚出库的记录。
  **答：** 默认查询期间为当天。请检查日期范围和类型过滤后点击**刷新**。
- **问：** 能否一次性出库2个以上数量？
  **答：** 消耗品按UID单件追踪出库。多个物品请逐个扫描UID。
