# 物料IQC项目管理

## 画面目的
为每个物料（原材料）预先设定**进货检验(IQC)时检验什么**的画面。
选择物料后，登记适用于该物料的**检验项目**（例如：外观、尺寸、拉伸强度）及各项目的**规格**（上下限、判定基准）、**不良等级**、**抽样方式**。此处登记的内容在实际进货检验画面中直接展开为检验表。

> 检验项目本身（名称·判定方法·单位）在[检验项目池]中预先登记。本画面是将这些项目**关联到物料并填写规格**的工作。

## 画面构成
- **左侧 — 物料列表**：原材料(RAW_MATERIAL)物料列表。可按物料代码·物料名称搜索，已登记检验项目的物料右侧显示**数量标签**。
- **右上 — AQL基准（摘要）**：基于所选物料关联的AQL政策计算出的样本数·Ac/Re等**只读预览卡片**（基于物料主数据中关联的政策）。
- **右下 — 检验项目**：以表格添加·修改·删除该物料的检验项目并[保存]的区域。上方工具栏同时设置**默认样品数**·**是否破坏检验**。

## ① 物料列表（左侧）

| 列 | 作用 / 含义 |
|------|------|
| **物料代码 / 物料名称** | 要设置检验基准的原材料物料。点击后右侧显示该物料的检验项目。 |
| **数量标签** | 该物料已登记的**检验项目数**。为0时无标签（未设置），有数字表示已登记基准的物料。 |
| **搜索** | 按物料代码或物料名称过滤列表。 |

## ② AQL基准摘要卡片（右上，只读）
此卡片为**参考摘要**而非输入区。值根据[物料主数据]中物料关联的**AQL政策**和[AQL基准管理]的设置自动计算。

| 项目 | 作用 / 含义 |
|------|------|
| **LOT数量预览** | 假设输入入库LOT数量后，预览该数量下样本数·Ac/Re如何取值。 |
| **AQL政策** | 物料主数据中该物料关联的AQL政策代码。为空时AQL自动判定不适用。 |
| **默认样品数** | 该物料检验基准标题中设置的默认样品数（与下方工具栏值相同）。 |
| **检验水准** | AQL政策的检验水准（通常II）。与LOT大小共同决定样本大小。 |
| **样本数量** | 基于上述LOT数量计算出的样本个数。 |
| **Major Ac/Re** | 重缺陷的合格(Ac)·不合格(Re)判定个数。 |
| **Minor Ac/Re** | 轻缺陷的合格·不合格判定个数。 |
| **检验项目基准** | 该物料已登记的检验项目数量。 |
| **破坏/固定** | 检验项目中**破坏检验·全数·固定样本**等使用单独样本数的项目数。 |

## ③ 检验项目标题（右下工具栏）

| 项目 | 作用 / 含义 |
|------|------|
| **默认样品数** | 该物料检验时默认抽取的样本个数（未在个别项目中指定固定样本数时的默认值）。 |
| **破坏检验 / 非破坏** | 该物料检验是否破坏样本。设为`破坏`表示检验后样本不可使用。 |
| **模板加载/管理** | 将常用检验项目集合保存为模板，一次性应用于新物料。 |
| **添加项目** | 添加一行检验项目并立即进入编辑模式。 |
| **保存** | 保存更改的标题·全部检验项目。无更改时禁用。 |

## ④ 检验项目列

| 列 | 作用 / 含义 |
|------|------|
| **顺序(seq)** | 检验表中项目的显示顺序。行添加·删除时自动重新编号。 |
| **检验项目(inspItemCode)** | 从[检验项目池]已登记的项目中选择。选择后种类·单位自动跟随。 |
| **种类(judgeMethod)** | 项目的判定方法。**计量型(MEASURE)**=测量数值与上下限比较，**判定型(VISUAL)**=用眼睛确认良/不良。仅计量型时输入LSL/USL。 |
| **检验类型(inspectionType)** | 决定样本的方式。`AQL`=由AQL表自动计算，`DESTRUCTIVE`=破坏检验，`FULL`=全数检验。非AQL时需直接输入**样本数**。 |
| **样本数(sampleQty)** | 破坏/全数/固定类型中检验的**固定样本个数**。AQL类型时显示`自动`，不输入。 |
| **不良等级(defectGrade)** | 该项目不良的严重度。`CRITICAL`（致命）·`MAJOR`（重）·`MINOR`（轻）。AQL判定中按等级分别应用Ac/Re。 |
| **检验水准(inspectionLevel)** | 该项目适用的ISO 2859-1检验水准（II、S4等）。用于决定样本大小。 |
| **AQL** | 该项目适用的合格品质界限值（0.65/1.0/2.5等）。**数值越小越严格**。 |
| **下限(LSL)** | 计量型项目的**允许最小值**。测量值小于此值则为不良。 |
| **上限(USL)** | 计量型项目的**允许最大值**。测量值大于此值则为不良。 |
| **判定基准(judgeCriteria)** | 判定型项目等判断合格·不合格的基准文字（例如："应无划痕·异物"）。 |
| **单位(unit)** | 测量值的单位。从检验项目池继承，仅显示用。 |

> 示例：将`尺寸`项目设为计量型，设定LSL=9.8，USL=10.2 → 测量值超出9.8~10.2(mm)范围时判定为不良。

## 使用顺序
1. 在左侧**选择**要设置检验基准的物料。
2. 在工具栏中设置**默认样品数**和**是否破坏检验**。
3. 通过**[添加项目]**（或**[加载模板]**）添加检验项目，在各行中填写种类·检验类型·不良等级·规格。
4. 计量型项目输入**下限(LSL)·上限(USL)**，判定型项目输入**判定基准**。
5. 各行通过**[修改]→[确认]**编辑，最后**[保存]**。

## 输入规则 / 校验
- **检验项目(inspItemCode)为空的行不保存**（空行自动排除）。
- 非计量型项目不可输入LSL/USL（以判定基准管理）。
- 检验类型为`AQL`时样本数自动计算，不直接输入。
- 一次仅可编辑一行，编辑中其他行的修改·删除被锁定。

## 常见问题
- **问：** 检验项目选择列表中看不到所需项目。
  **答：** 需先在[检验项目池]中登记（使用`Y`）后方可在此画面中选择。
- **问：** AQL基准摘要卡片的值能否在此更改？
  **答：** 不可。该值是由[物料主数据]的AQL政策关联和[AQL基准管理]设置决定的只读参考信息。
- **问：** 每次重复输入相同项目很繁琐。
  **答：** 通过[模板加载/管理]保存项目组合后，可一次性应用于新物料。
