# 物料库存盘点管理

## 画面目的
管理比较实际仓库库存与系统库存的**库存盘点(Physical Inventory)**会话。开始盘点会话以便PDA扫描库存，会话完成后确认结果。

## 画面构成
- **上方 — 基准年月/仓库/搜索筛选**: 选择查询基准。
- **会话状态显示**: 如有进行中的盘点会话，则在上方显示。
- **DataGrid列表**: 按物料显示系统数量 / PDA盘点数量 / 差异。
- **盘点开始 / 结束按钮**: 启动或完成会话。

---

## ① DataGrid列表列

| 列 | 作用 / 含义 |
|------|------|
| **仓库(warehouseName)** | 库存所在的仓库。 |
| **物料代码(itemCode)** | 盘点目标物料代码。 |
| **物料名称(itemName)** | 盘点目标物料名称。 |
| **物料序列号(matUid)** | LOT/序列号。 |
| **系统数量(qty)** | 盘点开始时的系统库存数量。 |
| **盘点数量(countedQty)** | PDA扫描输入的盘点数量。尚未扫描时显示为`-`。 |
| **差异(diffQty)** | 盘点数量 - 系统数量。**正数(蓝色)** = 实际更多，**负数(红色)** = 实际更少，**0(绿色)** = 一致。 |
| **盘点时间(countedAt)** | 最后PDA扫描的时间。 |
| **最后盘点日(lastCountDate)** | 该物料最后被盘点的日期。 |

---

## ② 盘点会话管理

| 操作 | 说明 |
|------|------|
| **开始盘点** | 选择基准年月和仓库开始盘点会话。开始后**所有库存变动(入库·出库·调整)将被阻止**，PDA可进行盘点扫描。 |
| **结束盘点** | PDA扫描完成后结束会话。结束后解除库存变动阻止。 |
| **反映盘点** | 结束后在[物料库存盘点反映]画面中审查盘点数量并反映到库存。 |

> ⚠️ 开始盘点后，物料入库、出库、调整、LOT分割/合并等所有库存变动事务将被阻止。

## 使用顺序
1. 通过**开始盘点**按钮选择基准年月和仓库启动会话。
2. 在PDA上扫描该仓库的物料并输入盘点数量。
3. 刷新画面实时监控PDA扫描现状。
4. 所有扫描完成后，通过**结束盘点**按钮结束会话。
5. 在[物料库存盘点反映]画面中审查盘点结果并反映到库存。
