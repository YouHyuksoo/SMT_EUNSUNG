# 工单管理

## 画面目的
创建成品·半成品的**生产指令（工单）**，并管理生产进度状态的画面。
1条工单是指示"生产什么物料、何时（计划日期）、生产多少个（计划数量）"，创建后按**开始→完成**推进。必要时可按**保持（暂停）**中断或**取消**。

> 工单是生产实绩·检验·出货的起点。无工单则无法登记该物料的实绩。

## 画面构成
- **上方 — 标题·按钮**：工具查看、刷新、列表视图/树视图切换、创建工单。
- **操作栏**：选择行后显示该工单的编号·状态，仅激活当前状态下可用的操作按钮（开始/完成/保持/解除保持/取消）。工单输出按钮也在此处。
- **列表（网格）**：工单列表。上方有搜索·状态·设备·工序·计划日期过滤。
- **右侧滑动面板**：创建/修改输入表单（从右侧打开）。

## 状态流程
工单有以下5种状态，按既定顺序转换：

```
WAITING(等待) ──开始──▶ RUNNING(进行中) ──完成──▶ DONE(完成)
   │                        │
   │                        └──保持──┐
   └──保持───────────────────────────┤
   │                                 ▼
   │                            HOLD(保持) ──解除保持──▶ 恢复为之前状态
   │
   └──取消──▶ CANCELED(取消)   (HOLD状态也可取消)
```

| 状态 | 含义 |
|------|------|
| **等待(WAITING)** | 创建后的默认状态。尚未开始作业。可进行开始·保持·取消。 |
| **进行中(RUNNING)** | 已开始作业的状态。此时可登记生产实绩。可进行完成·保持。 |
| **保持(HOLD)** | 临时暂停状态。**实绩登记·出货全部阻止**。解除保持后恢复为之前状态（等待/进行中）。 |
| **完成(DONE)** | 作业结束的状态。实绩自动汇总。不再可修改·变更状态。 |
| **取消(CANCELED)** | 已取消的状态。如有1条以上实绩则不可取消。 |

## ① 列表列

| 列 | 作用 / 含义 |
|------|------|
| **操作（修改/删除）** | 每行的快捷操作按钮。铅笔=打开修改面板，垃圾桶=删除。（进行中工单不可删除） |
| **计划日期(planDate)** | 该工单的**生产预定日期**。列表默认按计划日期范围查询，未指定计划日期的记录始终一起显示。 |
| **优先级(priority)** | 作业处理顺序。**1为最高、10为最低**（默认5）。1~3突出显示（红色边框），7~10显示模糊，便于快速识别紧急指示。列表按优先级升序排列。 |
| **工单编号(orderNo)** | 标识工单的唯一编号（例如：`W260519-001`）。在树视图中通过缩进和箭头显示上·下级（成品-半成品）关系。 |
| **物料代码(partCode)** | 生产目标物料的代码。 |
| **物料名称(partName)** | 生产目标物料的名称。 |
| **物料类型(partType)** | 物料种类。**FG=成品**，**WIP=半成品**，以彩色标签区分。 |
| **线别(lineCode)** | 执行作业的生产线（可选值）。 |
| **工序(processCode)** | 该工单的**代表工序**。从物料路由的首工序自动继承，也可直接指定。 |
| **设备(equipCode)** | 作业设备代码。创建时可留空后分配。 |
| **客户PO编号(custPoNo)** | 该指示对应的客户采购单(PO)编号（可选值）。 |
| **计划数量(planQty)** | 计划生产的数量。 |
| **实绩数量(goodQty)** | 实际生产的**良品数量**。生产实绩汇总后显示。 |
| **进度率(progress)** | 计划数量与良品数量的比率(%)。以条形图显示。 |
| **状态(status)** | 当前工单状态（等待/进行中/保持/完成/取消）。以彩色标签显示。 |

## ② 创建/修改面板输入项目

| 项目 | 作用 / 含义 |
|------|------|
| **工单编号(orderNo)** | 不直接输入。**保存时自动生成**（`W+日期+序号`）。 |
| **物料名称(itemCode)** | 通过放大镜按钮打开物料搜索模态框选择。仅可选择**成品·半成品**。选择后自动查询路由信息。 |
| **计划数量(planQty)** | 生产数量。**必填**且必须≥1。 |
| **计划日期(planDate)** | 生产预定日期。**必填**。 |
| **优先级(priority)** | 1~10之间的值（默认5）。数值越小越优先处理。 |
| **客户PO编号(custPoNo)** | 对应客户PO编号（可选值，例如：`PO-2026-0001`）。 |
| **路由信息** | 非输入项，为**自动显示**。显示所选物料的路由代码·名称和工序顺序（箭头）。无路由时显示"未登记路由的物料"提示。 |
| **线别(lineCode)** | 选择作业线别（可选值）。 |
| **工序(processCode)** | 选择代表工序。未指定时自动应用路由首工序。更改工序后设备选择将重置。 |
| **设备(equipCode)** | 从所选工序登记的设备列表中选择。可不指定。该工序无设备时显示提示。 |
| **备注(remark)** | 管理用备注。 |
| **基于BOM的半成品工单自动生成(autoCreateChildren)** | **仅新建时**显示的复选框（默认开启）。开启后将展开成品BOM，**自动同时生成**其中的**半成品工单**。半成品计划数量根据BOM需求量计算。 |

## ③ 操作栏操作

| 按钮 | 可用状态 | 操作 |
|------|------|------|
| **工单输出** | 有选中行 | 输出工单(A4: 上部工单 + 下部物料请求)。 |
| **开始** | 等待(WAITING) | 转为进行中(RUNNING)并记录开始时间。 |
| **完成** | 进行中(RUNNING) | 转为完成(DONE)并自动汇总实绩。**有剩余数量也结束**。 |
| **保持** | 等待/进行中 | 转为保持(HOLD)。实绩登记·出货被阻止。 |
| **解除保持** | 保持(HOLD) | 恢复为保持前状态（等待/进行中）。 |
| **取消** | 等待/保持 | 转为取消(CANCELED)。**有实绩则不可取消**。 |
| **预发行** | （仅预发行模式时显示） | 按工单数量预先发行成品条码。 |

## 使用顺序
1. 点击**[创建工单]**打开右侧面板。
2. 输入**物料**（放大镜）、**计划数量**、**计划日期**（必填3项）。必要时指定线别·工序·设备·优先级·客户PO。
3. 如是成品且需同时制作半成品，保持**BOM自动生成**勾选后保存。
4. 在列表中点击选择工单行，点击**开始**按钮开工。
5. 生产进行中在[生产实绩]中登记实绩。
6. 作业结束后点击**完成**按钮结束（实绩自动汇总）。

## 输入规则 / 校验
- 物料·计划数量·计划日期均需填写才能保存（计划数量≥1）。
- 工单编号不直接输入，保存时自动生成。
- 状态只能通过操作按钮更改，不可跳越（例如不可从等待直接完成）。
- 已完成·已取消的工单不可修改。
- 进行中的工单不可删除。
- 取消仅在无任何实绩时方可进行。

## 常见问题
- **问：** 能否自行设定工单编号？
  **答：** 不可以。保存时按`W+YYMMDD+序号`格式自动生成。
- **问：** 开启BOM自动生成会创建什么？
  **答：** 展开成品BOM，**为其中的每个半成品创建工单**（在树视图中显示为下级）。半成品数量按BOM需求量计算。
- **问：** 保持和取消有何不同？
  **答：** 保持是**暂停**（稍后解除保持可恢复），取消是**终止处理**（不可撤销）。
- **问：** 点击完成时计划数量未全部完成怎么办？
  **答：** 即使有剩余数量也可完成。汇总完成时点的实绩。
- **问：** 列表中看不到工单。
  **答：** 请确认上方的计划日期过滤范围。默认值为今天日期。扩大计划日期范围可显示更多记录。
