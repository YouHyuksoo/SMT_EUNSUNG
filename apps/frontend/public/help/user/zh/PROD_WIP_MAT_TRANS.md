# 工序物料收发

## 画面目的
按设备(工序)维度查询**原材料工序在制品(WIP)库存**的交易记录。追踪原材料投入工序(WIP_IN)、生产消耗(PROD_CONSUME)及其取消记录，支持按设备、日期筛选。

## 画面构成
- **DataGrid列表**: 8列 — 时间、类型、设备、品目、LOT、数量、参照、备注
- **顶部筛选**: 设备选择 + 交易类型选择 + 日期范围 + 搜索词(品目代码/名称/LOT/设备名称)

---

## ① DataGrid列表列

| 列 | 作用 / 含义 |
|------|------|
| **时间(createdAt)** | 交易处理时间。 |
| **类型(transType)** | 工序收发类型。**WIP_IN**(蓝色) / **WIP_IN_CANCEL**(红色) / **PROD_CONSUME**(橙色) / **PROD_CONSUME_CANCEL**(红色)。 |
| **设备(equipName)** | 交易发生的设备名称和代码。 |
| **品目(itemCode)** | 交易对象原材料品目代码和名称。 |
| **LOT(matUid)** | 交易的LOT/序列号。 |
| **数量(qty)** | 变动数量。**正数(蓝色)** = 库存增加，**负数(红色)** = 库存减少。 |
| **参照(refType)** | 原始交易的参照类型和ID(如工单号)。 |
| **备注(remark)** | 交易附加说明。 |

## 交易类型说明
| 类型 | 含义 | 数量 |
|------|------|------|
| WIP_IN | 原材料从物料仓库投入设备(工序) | + (正数) |
| WIP_IN_CANCEL | 取消工序入库 — 原材料退回仓库 | - (负数) |
| PROD_CONSUME | 设备中原材料实际生产消耗 | - (负数) |
| PROD_CONSUME_CANCEL | 取消生产消耗 — 消耗的原材料还原至工序库存 | + (正数) |

## 使用顺序
1. 在顶部筛选中选择**设备**和**交易类型**，根据需要设置**日期范围**和**搜索词**。
2. 在列表中查看各交易详情。
3. 使用列筛选和排序功能查找特定数据。
4. 通过**导出**功能将结果下载为Excel。
