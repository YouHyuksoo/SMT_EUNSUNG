# PO管理

## 画面目的
向供应商(SUPPLIER)采购原材料等**采购单(PO)**的登记·查询·修改·删除画面。PO需确认(CONFIRMED)后才能进入后续到货·进货检验阶段，所有物料入库完成后关闭(CLOSED)。

## 画面构成
- **左侧（PO列表网格）**：以表格形式查询已登记的PO。上方可按PO编号·客户·物料综合搜索，或按采购日期区间·状态过滤。
- **右侧（登记/修改面板）**：点击**登记**按钮或行的✏️时从右侧滑动打开。同时输入PO标题信息和物料列表。

---

## ① PO列表列

| 列 | 作用 / 含义 |
|------|------|
| **PO No.(poNo)** | 标识采购的唯一编号。登记时自动生成（`PO-YYYYMMDD-NNN`格式），一旦创建不可更改。 |
| **客户(partnerName)** | 采购目标供应商名称。从客户主数据的SUPPLIER类型中选择。 |
| **采购日期(orderDate)** | PO发行的日期。默认值为今天（登记时）。 |
| **交付日期(dueDate)** | 交货请求期限。到货管理中判断交付遵守率的依据。 |
| **物料数(itemCount)** | 该PO包含的采购物料行数。 |
| **总金额(totalAmount)** | 按物料（采购数量×单价）合计。未输入单价时以0计算。 |
| **状态(status)** | PO的当前进行状态（参见下面状态流程）。 |

### 状态流程

```
DRAFT(临时登记)
    ↓ 确认
CONFIRMED(采购确认)
    ↓ 部分入库
PARTIAL(部分入库)
    ↓ 全部入库
RECEIVED(入库完成)
    ↓ 关闭
CLOSED(关闭)
```

> **仅DRAFT状态可修改·删除。** CONFIRMED后状态在到货过程中自动变更。

---

## ② PO标题输入字段

| 列 | 作用 / 含义 |
|------|------|
| **PO No.(poNo)** | 登记时自动生成（`PO-YYYYMMDD-NNN`）。也可直接修改，但不可重复。修改时不可变更（锁定）。 |
| **客户(partnerName)** | 选择采购的供应商。仅SUPPLIER类型客户显示在选择列表中。 |
| **采购日期(orderDate)** | PO发行日。默认值为今天。 |
| **交付日期(dueDate)** | 交货请求日。可留空，但建议为到货日程管理而输入。 |
| **备注(remark)** | PO整体的备注（特记事项、参考合同编号等）。 |

---

## ③ 采购物料输入字段

点击添加物料按钮时打开物料搜索模态框。可同时选择多个物料（支持多选）。已添加的物料代码不可重复添加。

| 列 | 作用 / 含义 |
|------|------|
| **物料代码(itemCode)** | 采购物料的MES代码。在物料搜索模态框中选择后自动填入。 |
| **物料名称(itemName)** | 所选物料的名称（自动显示）。 |
| **行号(lineNo)** | 该PO内物料行的序号。默认按添加顺序自动分配。 |
| **发布号(revNo)** | 同一物料的采购批次。在同一PO中按不同交付期分割同一物料时区分。默认值为1。 |
| **采购数量(orderQty)** | 该行采购的数量。仅可输入**≥1的整数**。 |
| **备注(remark)** | 该物料行的备注。 |

> 单价(unitPrice)在当前画面中不直接显示输入UI。如有需要请咨询运营人员。总金额在有单价信息时自动计算。

## 搜索·过滤

| 过滤项 | 操作 |
|------|------|
| **综合搜索** | 输入PO编号或客户名称，不区分大小写进行部分匹配搜索。 |
| **采购日期区间** | 按采购日期设置查询期间。默认值为今天（当日）。 |
| **状态** | 仅显示所选状态的PO。留空则查询全部。 |

---

## 使用顺序

1. 点击右上角**登记**按钮打开PO登记面板。
2. 选择**客户**（SUPPLIER类型）。
3. 输入**采购日期**和**交付日期**。
4. 点击**添加物料**按钮搜索·选择要采购的物料（可多选）。
5. 输入各物料的**采购数量**（≥1的整数必填）。
6. 点击**登记**按钮保存。PO编号自动生成，状态为DRAFT。
7. 采购内容确认后，在网格中打开该PO将状态转为**确认(CONFIRMED)**。
8. 入库处理在**到货管理**画面进行。

---

## 输入规则 / 校验

- **PO No.**：必填。不可重复。
- **客户**：必填。
- **采购物料**：至少1个必填。
- **采购数量**：必填，≥1的整数。输入小数·0·负数时阻止保存。
- **可删除条件**：需为DRAFT状态且无到货历史。

---

## 常见问题

- **问：** 想自行指定PO编号。
  **答：** 可修改自动生成的编号直接输入。但重复时保存被阻止。

- **问：** 已确认的PO想再次修改。
  **答：** CONFIRMED后的状态无法在画面直接修改。请向运营人员咨询恢复为DRAFT的流程。

- **问：** 删除PO时失败。
  **答：** 需为DRAFT状态且无到货历史才能删除。已CONFIRMED或已有部分到货时删除被阻止。

- **问：** 总金额显示为0。
  **答：** 未输入单价时总金额以0计算。单价登记需要运营人员设置。

- **问：** 同一PO中能否添加两次相同物料？
  **答：** 同一物料代码仅可添加一次。如需不同交付期·批次，请区分发布号或另行创建PO。
