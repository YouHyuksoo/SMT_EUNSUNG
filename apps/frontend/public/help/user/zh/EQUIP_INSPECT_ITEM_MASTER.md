# 检查项目主数据

## 画面目的
注册和管理设备检查所用的**检查项目**作为标准信息。每个项目具有检查类型（日常/定期/PM/作业人员）、判定区分（判定型/测量型）、周期、判定标准（上限/下限）等属性。已注册的项目可在[按设备检查项目]画面中连接到单个设备使用。

## 画面构成
- **DataGrid列表**：显示已注册的检查项目列表，包含图片、项目代码、设备类型、检查项目名称、检查类型、判定区分、判定标准、周期及使用与否。
- **右侧面板（注册/编辑）**：添加或编辑项目时，右侧会打开滑动面板。
- **顶部筛选**：搜索关键词 + 设备类型 + 检查类型选择

---

## ① DataGrid列表列

| 列 | 作用 / 说明 |
|------|------|
| **图片 (imageUrl)** | 与检查项目相关的参考图片。以缩略图显示，点击可放大。 |
| **项目代码 (itemCode)** | 标识检查项目的唯一代码。注册后不可修改。 |
| **设备类型 (equipType)** | 该检查项目适用的设备类型（公共代码 `EQUIP_TYPE`）。 |
| **检查项目名称 (itemName)** | 检查项目的名称。 |
| **检查类型 (inspectType)** | 按检查主体/时机的分类。**DAILY**（日常检查，蓝色）/ **PERIODIC**（定期检查，橙色）/ **PM**（预防性维护，紫色）/ **WORKER**（作业人员检查，绿色）。 |
| **判定区分 (itemType)** | 检查结果的判定方式。**VISUAL**（判定型，灰色）= 目视OK/NG / **MEASURE**（测量型，青色）= 数值测量。 |
| **判定标准 (criteria)** | 检查结果的合格标准。测量型显示为**下限(LSL) ~ 上限(USL)**值及单位。 |
| **周期 (cycle)** | 检查执行周期。DAILY / WEEKLY / MONTHLY / QUARTERLY / SEMI_ANNUAL / ANNUAL。 |
| **使用 (useYn)** | `Y`（绿色）= 使用中，`N`（红色）= 未使用。 |

## ② 注册/编辑面板字段

| 字段 | 作用 / 说明 |
|------|------|
| **项目代码 (itemCode)** | 检查项目的唯一标识代码。注册后不可修改。 |
| **设备类型 (equipType)** | 检查项目适用的设备类型。 |
| **检查项目名称 (itemName)** | 检查项目的名称。 |
| **检查类型 (inspectType)** | 选择检查类型（DAILY/PERIODIC/PM/WORKER）。 |
| **判定区分 (itemType)** | **VISUAL**（判定型）：目视判定OK/NG / **MEASURE**（测量型）：通过测量数值确认是否在标准范围内。 |
| **周期 (cycle)** | 选择检查执行周期。 |
| **使用 (useYn)** | 使用与否（Y/N）。 |
| **判定标准 (criteria)** | VISUAL：输入OK/NG判定标准文本。 / MEASURE：选择单位 + 输入**下限(LSL)** + **上限(USL)**数值。 |
| **图片 (imageUrl)** | 上传检查参考图片（JPEG/PNG/GIF/WebP）。可通过点击或拖放上传。 |
| **备注 (remark)** | 附加管理备注。 |

## 使用顺序
1. 点击**添加**按钮注册新的检查项目（右侧面板）。
2. 输入检查类型、判定区分、周期等属性。测量型时设置上限/下限标准值。
3. 必要时上传检查参考图片。
4. 在列表中点击编辑图标修改现有项目。
5. 已注册的项目可在[按设备检查项目]画面中连接到单个设备使用。

## 输入规则 / 验证
- 项目代码和检查项目名称是**必填项**。
- 测量型（MEASURE）必须同时输入下限（LSL）和上限（USL）。
- 仅可上传图片文件（JPEG/PNG/GIF/WebP），最大限制为5MB。

## 常见问题解答
- **问：** 检查类型之间有什么区别？**答：** **DAILY**=每天作业前检查，**PERIODIC**=定期（月/季度/半年/年），**PM**=预防性维护设备检修时，**WORKER**=作业人员自行进行的检查。
- **问：** 判定型（VISUAL）和测量型（MEASURE）有什么区别？**答：** 判定型是OK/NG等定性判断，测量型是测量具体数值以确认是否在标准范围（下限~上限）内。
- **问：** 注册后可以修改项目代码吗？**答：** 不可以，因为它是主键。必须删除后重新注册。
