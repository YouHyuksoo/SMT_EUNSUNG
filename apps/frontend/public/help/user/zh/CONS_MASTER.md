# 消耗品主数据

## 画面目的
注册和管理生产所用的**消耗品（模具·夹具·工具等）的基础信息**。此处注册的消耗品将成为入库出库·库存·寿命（次数）管理、Kiosk生产实绩的消耗品投入、使用处（产品·设备）映射的基础。

> 消耗品是越用越磨损或寿命耗尽的物料。在此画面中统一定义**是什么（代码·名称·分类）、能用多久（预期寿命·警告阈值）、放在哪里各放多少（安全库存·保管位置）、用在哪里（使用处映射）**。

## 画面构成
- **左侧（列表）**：消耗品网格。上方可**按消耗品名称/代码搜索**、**按分类过滤**、**刷新**、**注册消耗品**。
- **中间（滑动面板）**：点击行的✏️（修改）或"注册消耗品"时打开的注册/修改表单。
- **右侧（使用映射面板）**：在列表中选择行时，可随时注册·修改·删除该消耗品的**产品/设备使用映射**。

## 消耗品的两种用途
同一消耗品主数据在现场有两种用途：
- **设备安装用（模具·夹具·刀片等）**：安装在设备上累积次数（使用次数），达到预期寿命时更换。属于Kiosk的"消耗性设备部件"管理对象。
- **产品BOM投入用**：制造产品时一起投入的消耗性部件。包含在产品BOM中，按生产数量扣减。

> 哪些产品·设备使用哪些消耗品，通过右侧**使用映射**定义。

## ① 基本信息

| 列 | 作用 / 含义 |
|------|------|
| **消耗品代码(consumableCode)** | 标识消耗品的**唯一代码**。例如：`CM-AP-110`。入库出库·库存·使用映射通过此代码关联，注册后不更改（修改模式下锁定）。 |
| **消耗品名称(consumableName)** | 现场识别的名称。例如：`110端子压接模具`。 |
| **分类(category)** | 消耗品种类。从模具(MOLD)·夹具(JIG)·工具(TOOL)中选择（公共代码CONSUMABLE_CATEGORY）。是列表过滤·统计的依据。 |
| **图片(imageUrl)** | 消耗品照片。**仅保存（注册）后**可上传，也显示在列表缩略图中。 |

## ② 寿命 / 管理

| 列 | 作用 / 含义 |
|------|------|
| **预期寿命(expectedLife)** | 该消耗品无需更换可使用的**累计次数**。例如：`100000`。累计次数达到此值时状态变为**需更换(REPLACE)**。 |
| **警告阈值(warningCount)** | 提前提示更换临近的次数。例如：`80000`。累计次数达到此值时状态变为**警告(WARNING)**，提示准备更换。通常设为小于预期寿命的值。 |
| **安全库存(safetyStock)** | 判断库存不足的基准数量。持有库存低于此值视为不足。 |

> 累计次数（当前使用次数）在Kiosk生产·更换等处自动累计·重置。本画面仅定义其**基准值（预期寿命·警告阈值）**。

## ③ 客户 / 位置

| 列 | 作用 / 含义 |
|------|------|
| **保管位置(location)** | 消耗品存放的默认保管场所。例如：`模具室-A1`。 |
| **客户(vendor)** | 供应消耗品的客户/制造商。例如：`JST`。 |
| **单价(unitPrice)** | 消耗品的单个单价。供入库单价·资产评估参考。 |

## 使用处映射（右侧面板）
注册所选消耗品用于**哪些产品（型号）·哪些设备**。是Kiosk生产实绩中按工单（产品·设备）查找合适消耗品的依据。

| 项目 | 作用 / 含义 |
|------|------|
| **产品/型号(productItemCode)** | 使用该消耗品的产品/型号。仅可选择成品·半成品物料。 |
| **设备(equipCode)** | 生产该产品时使用该消耗品的设备。 |
| **单位用量(usagePerUnit)** | **每单位生产消耗的次数（使用次数）**。生产数量×此值累积消耗品次数。默认值1。 |
| **使用与否(useYn)** | 仅`Y`时为有效映射。`N`时保留映射但不使用（点击列表中的`Y`标签可切换）。 |
| **备注(remark)** | 映射管理用备注。 |

> 使用映射按**产品 + 设备 + 消耗品**组合为1条。同一消耗品可映射到多个产品·设备。

## 使用顺序
1. 通过上方**注册消耗品**（或行✏️）打开滑动表单。
2. 输入**基本信息**（代码·名称·分类）后保存。
3. 保存后重新打开表单上传**图片**（注册后不可直接上传）。
4. 填写**寿命/管理**（预期寿命·警告阈值·安全库存）、**客户/位置**（保管位置·客户·单价）。
5. 在列表中选择消耗品行，在右侧**使用映射**中注册产品·设备·单位用量。

## 输入规则 / 校验
- **消耗品代码**和**消耗品名称**为必填项。为空时保存按钮禁用。
- 消耗品代码不可重复（已存在时保存被拒绝）。
- 图片**仅保存后**可上传（jpg/png/gif/webp，5MB以下）。
- 使用映射需**同时选择产品与设备**才能保存。

## 常见问题
- **问：** 代码输入错误，但修改模式下无法更改。
  **答：** 消耗品代码是入库出库·库存·映射关联的键，不可更改。请删除后重新注册。
- **问：** 注册时图片上传框显示灰色。
  **答：** 图片需在保存后上传。先保存基本信息，再重新打开表单上传。
- **问：** 预期寿命和警告阈值有何区别？
  **答：** 达到警告阈值时变为**警告（准备更换）**，达到预期寿命时变为**需更换**。顺序为提前提醒→实际更换时间点。
- **问：** 列表中有看不到的消耗品。
  **答：** 列表仅显示使用中(`useYn=Y`)的消耗品。请同时确认分类过滤和搜索词。
